Projects

To create a new gPROMS project select New from the Project menu. This will bring up a tree in the left-hand pane containing a number of folders:

  • Variable Types

  • Stream Types

  • Connection Types

  • Models

  • Tasks

  • Processes

  • Experiments

    • ... performed

    • ... to be designed

  • Experiment Designs

  • Parameter Estimations

  • Optimisations

  • Saved Variable Sets

  • Miscellaneous Files

New project tree entries

The Project can be renamed from its default of Project_1.gPJ by:

  • Selecting Save As from the Project menu

  • Entering a new File Name

  • Clicking Save
