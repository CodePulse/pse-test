Overview

gPROMS ModelBuilder is at the centre of:

  • all model development and maintenance activities;
  • the archiving of models and results;
  • the execution of all model-based activities.

This guide gives a description of the features and functionality of gPROMS ModelBuilder - including some of the advanced tools that are available:

  • An introduction to the gPROMS ModelBuilder environment looks at Projects and the general properties of all Entities.
  • Constructing flowsheet Models explains how to graphically develop and configure hierarchical flowsheets from (library) Models.
  • Executing simulations explains how to configure solvers and result management, and how to execute simulations.
  • In Viewing results it is shown which options exist to view results of Simulations, including plots, Stream Tables and customised Model Reports.
  • Building Models is supported by various tools and utilities.

The gPROMS ModelBuilder Guide is the first of a comprehensive set of manuals that cover all aspects of gPROMS ModelBuilder :

  • Custom Modeling Guide
  • Model Validation Guide
  • Optimisation Guide
  • Physical Properties Guide
  • Foreign Objects and Foreign Processes
  • System Programmer Guide
  • gO:RUN User Guide
  • gO:MATLAB User Guide
  • gPROMS Server Guide (available upon request).

Guides in italics are currently available as PDF documents in the documentation sub-folder of the installation and can be accessed from the Windows Start Menu.
