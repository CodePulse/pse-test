Additional information display

The Topology editor provides additional information via a pop-up message approach. The following information is available:

  • Connection information: placing the mouse pointer over a connection displays the corresponding gPROMS language for the connection equation.

  • Port information: placing the mouse pointer over a Port displays the Port name, type and direction.
