Allowable entity names

All entities in gPROMS (Models, Tasks, Parameters, Variables etc.) are subject to the same restrictions on the names they can be given.

  • Names may only contain letters, numbers and the underscore character "_"

  • All names must begin with a letter

  • Upper and lower case letters may be used but all names are case insensitive: that is, VariableOne, variableone and VARIABLEONE will all be considered the same name.
