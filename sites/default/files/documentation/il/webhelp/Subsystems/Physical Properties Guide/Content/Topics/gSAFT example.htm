gSAFT example

gSAFT is distributed with an example for a highly non-ideal mixture consisting of HF and water. The figure below depicts a T-X-Y plot at a given pressure where measurements (circles) are compared against predictions (solid lines).

HF-water mixture: comparison of SAFT-VR SW prediction and measurements

The example project HF_water.gPJ contains a gPROMS Model that allows to reproduce the trajectories of the predicted behaviour. The gPROMS Model for this prediction is very simple and it is intended to demonstrate the basic aspects of choosing an EOS, here the SAFT-VR SW EOS, by selecting the database, and of specifying the components hydrogen_fluoride and water as the mixture.

PARAMETER

  physProps         AS FOREIGN_OBJECT
  components        AS ORDERED_SET 
  pressure          AS REAL          default 101325

VARIABLE
  z                 AS ARRAY(components)           of Fraction

  dewTemperature    AS                                Temperature 
  bubbleTemperature AS                                Temperature

SET
  physProps   := "gSAFT::VR_SW_databank.dtb<hydrogen_fluoride,water>";
  components  := physProps.components;

EQUATION
  $z("hydrogen_fluoride") =  1;
  sigma(z)                =  1;

  dewTemperature          = physProps.DewTemperature   (pressure, z);  
  bubbleTemperature       = physProps.BubbleTemperature(pressure, z);

INITIAL
  z("hydrogen_fluoride") = 0.0;

z varies between 0 and 1 and allows plotting the dew and bubble temperature which renders the diagram below.
