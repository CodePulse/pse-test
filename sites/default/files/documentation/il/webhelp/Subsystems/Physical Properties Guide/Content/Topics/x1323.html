Result vector for binary diffusion coefficients

Multiflash offers two method for binary diffusion coefficients: vapour and liquid.

Binary diffusion coefficient methods and their arguments
Property Name Inputs Description Type
LiquidDiffusionCoefficients T,P,n Binary liquid diffusion coefficients Vector
VapourDiffusionCoefficients T,P,n Binary vapour diffusion coefficients Vector
Each of them returns a result vector of size NoComp*NoComp. The result vector is organised in rows per component where the order follows the order in the multiflash command file. The coefficient for the interaction of component i and j can be accessed by the index expression 
j + (i - 1) * NoComp
A complete example looks like this:
PARAMETER
 NDc AS INTEGER
 NoComp AS INTEGER
 PhysProp AS FOREIGN_OBJECT "Multiflash"

 VARIABLE
 T AS Temperature
 P AS Pressure
 z AS DISTRIBUTION (NoComp) OF NoType
 DC AS DISTRIBUTION(NDc) OF NoType
 X AS ARRAY(NoComp, NoComp) OF NoType

 SET
 NoComp := PhysProp.NumberOfComponents ;
 NDc = NoComp*NoComp;

 EQUATION
 DC = PhysProp.LIQUIDDIFFUSIONCOEFFICIENTS(T, P, z) ;
 FOR i := 1 TO NoComp DO 
 FOR j := 1 TO NoComp DO 
 X(i, j) = DC(j + (i - 1) * NoComp); 
 END 
 END
