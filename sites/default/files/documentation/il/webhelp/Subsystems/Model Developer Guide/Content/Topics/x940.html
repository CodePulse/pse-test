The VARIABLE section

The VARIABLE section is used to declare the Variables of a Model. All quantities that are calculated in Model Equations must be declared as Model Variables. For instance, in the example process, M, h, and are variables of the Buffer Tank Model:

VARIABLE 
 HoldUp AS Mass 
 FlowIn, FlowOut AS MassFlowrate 
 Height AS Length

Like Parameters, Variables are always Real continuous numbers. All Variables must be given a type, however, Variable Types are user-defined (see also: Declaring Variable Types).
