Documentation

The documentation for gPROMS FormulatedProducts is in HTML5 format which enables features such as web-style searching. The documentation contains several sections covering gPROMS platform features but most importantly the model libraries themselves. The various sections of the documentation are briefly outlined here.

The documentation contains the following broad sections:

  • Software Environment
    • Description of the interface of gPROMS FormulatedProducts (common with all gPROMS products) explaining aspects such as viewing results
  • Model Library
    • Documentation for each unit operation model within the gPROMS FormulatedProducts library
    • Key equations for each unit can be accessed via the PSE website by logging into MyPSE (restricted access, please contact support.formulatedproducts@psenterprise.com for more information)
  • Workflows
    • A description of the parameter estimation workflow as part of the overall model validation cycle
  • Custom modelling
    • Important information for customising gPROMS FormulatedProducts such as connection type contents and physical property methods
    • Documentation for general gPROMS modelling including basics of gPROMS language, concepts such as arrays and running stochastic simulations
  • Model validation
    • The model validation cycle in gPROMS
  • Optimisation
    • How to use gPROMS' optimisation capability
  • Global System Analysis
    • Overview of a major new feature in gPROMS to perform uncertainty and sensitivity analysis
  • Physical Properties
    • Information on physical properties modelling using packages such as Multiflash
  • Foreign Objects and Foreign Processes
    • How to use FO and FP such as the Excel interface for gPROMS
  • System Programming
    • How to develop and use bespoke foreign objects and foreign processes
