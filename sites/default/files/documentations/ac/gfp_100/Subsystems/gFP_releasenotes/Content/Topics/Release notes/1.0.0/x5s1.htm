Custom modelling templates

To customise gPROMS FormulatedProducts the custom modelling templates are required. In this folder there are two sections, both are described below.

Kinetic and property model templates

This folder contains an open version of gFORMULATE Custom model library. To use first close the library version by navigating to the project tree and finding the project, right-click and close.

Within this project there are templates for the various phenomena used in gFORMULATE. Copy the file of interest into the project file you wish to use it in and change the corresponding phenomena option in the desired flowsheet unit operations to "Custom".

User interface templates

This folder contains copies of all the library unit operation interfaces. These are open and so can be edited. This is for advanced users who wish to add top level variables to the interface either for reporting or for user input.
