Interacting with the new physical properties capability

A major new feature of gPROMS FormulatedProducts is the incorporation of flexible physical properties modelling. A main part of this is the interaction with a properties database which contains information of species, mixtures, equipment and more.

To interact with this capability there first needs to be a database to use. A minimal database is provided in the installer and can be interacted with via the Data Import and Properties Manager which can be downloaded from activeCollab.

Once the database contains the required entries, a material system file can then be created using the Properties Manager. This material system file is then entered into the gFP global specifications model on the flowsheet (replacing legacy global specification model).
