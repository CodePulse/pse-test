Backwards compatibility

This section outlines the backwards compatibility of gPROMS FormulatedProducts 1.0 with the legacy libraries of gCOAS, gSOLIDS and gCRYSTAL.

Specifications

Limited backward compatibility in that we can currently only guarantee backward compatibility with respect to physical properties entered in via the global specifications objects in gCRYSTAL, gSOLIDS and gCOAS.

We are aiming for a future release to provide backward compatibility with respect to specifications entered in other flowsheet models, which are mainly the unit operation models.

Calculation results

PSE has performed extensive testing to ensure that results obtained in gPROMS FormulatedProducts correspond to those obtained using gCRYSTAL 4.2, gSOLIDS 4.1 and gCOAS 1.3 when using the same specifications.
