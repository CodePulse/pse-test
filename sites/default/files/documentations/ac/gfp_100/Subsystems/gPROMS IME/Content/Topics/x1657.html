Image annotations

When an image annotation is added to the Flowsheet, it can be moved and resized just as a text annotation. However, when the image file is selected, the annotation is automatically resized to the size of the image. Therefore, it is best to select the image file first. To do this, double click on the image annotation to bring up the file-selector dialog. You can then select the desired image using the standard file browser. Supported formats are:

  • SVG

  • GIF

  • JPEG

  • PNG

  • BMP
  • WBMP.

Once the image has been selected, you can move or resize it by left clicking on the image annotation to select it and then:

  • Holding down the left mouse button anywhere within the control points and dragging the mouse to move the image annotation.

  • Left clicking on one of the control points and dragging it to resize the image annotation. (The mouse pointer will change when it is over a control point to indicate which way the point can be moved: vertically only, horizontally only or in any direction.)

    • If you hold down the Shift key while resizing, the aspect ratio of the image will be set to 1 (i.e. a square).
