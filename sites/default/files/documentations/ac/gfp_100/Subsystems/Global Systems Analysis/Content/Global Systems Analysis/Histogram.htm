Histogram

A histogram shows how the values are distributed for each factor or response. This is particularly useful for uncertainty propagation as it is easy to observe the dispersion in responses as a result of the uncertainty and range of factor values. Similarly, it can be used to visualise the distribution of a factor to verify if the samples cover the space as intended.

To create a histogram within a GSA Case, reproduce the steps below:

  1. Right-click either on the top level grain of the Case or on the 'System Analysis Group' folder.
  2. Select 'New result element...'.
  3. Select Histogram in the drop down menu, provide a name and click on OK.

On the Attributes panel, at the right-hand side, it is possible to include a title, select the factor or response whose distribution to display, the display axis title and whether it should be updated on variable change.

Histogram settings

A histogram has a number of customisable settings, listed below.

  1. Type

    1. Simple: Identifies the number of samples in each bin

  2. Number of bins
  3. Bar colour

It should be noted that the histogram provides a one-dimensional representation of the data and any multi-dimensional features are lost.
