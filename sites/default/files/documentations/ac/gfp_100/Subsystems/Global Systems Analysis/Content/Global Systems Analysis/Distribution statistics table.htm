Notation

The minimum and maximum represent the smallest and largest value of the factor or response, respectively. In contrast, the median represents the value of the mid-data point when sorted either from small to large or from large to small. A large difference between mean and median values is a common indicator of skewness in the data. Quartiles also inform how the data is distributed by dividing it into percentile groups.

This metric should be interpreted with care as it is not suitable for comparison, for example, when the value of the mean is either close to zero or very large.

To create a distribution statistics table within a GSA Case, reproduce the steps below.

  1. Right-click either on the top level grain of the Case or on the 'System Analysis Group' folder.

  2. Select 'New result element...'.

  3. Select 'Distribution Statistics Table' in the drop down menu, provide a name and click on OK.

  4. In the Attributes panel on the right-hand side, click on the ellipsis button to select which variables to add to the table.

    Selection of variables to add to the distribution statistics table

  5. Select the metrics to display in the 'Table configuration' section of the Attributes panel.

    Selection of metrics to be displayed in the distribution statistics table
