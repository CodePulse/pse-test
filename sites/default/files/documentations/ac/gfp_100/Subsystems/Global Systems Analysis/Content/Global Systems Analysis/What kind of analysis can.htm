Types of analysis

GSA can perform three types of analysis:

  • Scenario or parametric
  • Uncertainty
  • Sensitivity

The purpose of each type of analysis differs widely:

  • Scenario or parametric sensitivity analysis is a series of simulations where the values of one or more factors are varied over a grid to investigate the resulting changes in one or more responses. It allows an assessment of the impact of different factors on each response.
  • Uncertainty analysis propagates the uncertainty in the factors to the responses. The values returned represent the actual uncertainty associated with the responses.
  • Sensitivity analysis provides metrics which indicate how factors and their uncertainty influence responses. However, the values obtained do not represent actual sensitivities. This is discussed in more detail in 'Principles underlying sensitivity analysis'.

Scenario or parametric sensitivity analysis can be used when a better understanding of a model or system is required, for example:

  • What range of profit can be expected for different cases?
  • How much does a response vary as a result of a change in a single or multiple factors?
  • What range of temperature can be observed in different scenarios?

Uncertainty analysis can be used when there is a requirement to determine how the uncertainty, in the factors, influences responses, for example:

  • How does uncertainty in pricing affect profitability?
  • How do the uncertainties in operation affect the safety of the plant?
  • What is the effect of physiological variability on drug uptake in humans?
  • How does uncertainty in an unmeasured factor influence the results of a scenario testing exercise?
  • How does the uncertainty in parameters affect predictions and KPIs?
  • How much do errors in decision variables influence safety constraints?
  • How much does the uncertainty in kinetics affect responses?

Sensitivity analysis can be used when there is a requirement to know which factors are the most influential on a particular response, for example:

  • What factors influence responses the most, in order to focus research or investments?
  • What phenomena to consider or exclude in a model?
  • What parameters to focus on in a parameter estimation exercise?
  • What decision to make in an optimisation problem?
