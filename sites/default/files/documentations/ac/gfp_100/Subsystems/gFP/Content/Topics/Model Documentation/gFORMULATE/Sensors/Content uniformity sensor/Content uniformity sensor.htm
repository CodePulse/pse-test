Content uniformity sensor

This model describes evaluated the Relative Standard Deviation (RSD) for Solid Oral Dosage Forms (SODFs), using a number of literature models. The key inputs for these models are the particle size distribution of the API/drug substance, as well as the dosage level of the SODF.

The following models for describing the content uniformity of tablet and capsules dosage forms are included in Content_uniformity_sensor:

  • Johnson (1972)
    • Single-sized API particles
    • Less than 1% API/drug
    • 1 – 10% API/drug
    • Above 10% API/drug
  • Rohrs et al. (2006)
  • Hilden et al. (2012)
