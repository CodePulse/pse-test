Measurements tab

The measurements tab allows the visual comparison of model predictions with the measured values. The comparison can be viewed using the “View” dropdown selector in three ways, as follows:

  1. Measurements Plot – 2-D plot of the measured and predicted values as a function of time, as shown below.
  2. Parity plot – 2-D plot of the measured values versus the predicted values.
  3. Table – Measured values, predicted values and the standard deviation as a function of time.
