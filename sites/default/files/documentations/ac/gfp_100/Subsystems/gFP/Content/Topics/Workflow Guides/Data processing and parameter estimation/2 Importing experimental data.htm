Importing experimental data

Concept > Performed experiment entity.  gPROMS FormulatedProducts has experiment entities used to represent experiments with their controls and measurements. Each experiment entity must be linked to a Process and are used in parameter estimation.

 

The Performed experiment entity is used to specify the full details of an experiment.  This includes both the operation, the (measured) inputs, of the experiment as well as the measured outputs of the experiments.

To create a performed experiment entity, using an existing project containing a process relating to the performed experiment, right click on the project name in the project tree and select “New entity”.  Create a “Performed experiment” entity, remembering to give it a title.

A new performed entity will appear, the tabs and required inputs are described below.
