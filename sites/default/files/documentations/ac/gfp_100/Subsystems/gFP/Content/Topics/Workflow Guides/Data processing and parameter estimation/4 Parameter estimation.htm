Parameter estimation

Concept > Parameter estimation entity. gPROMS FormulatedProducts contains parameter estimation entities that can estimate multiple parameters using measurements from multiple experiments.  Each measurement is associated with a Sensor which is given a variance (through a variance model). Measurements in different experiments using the same sensor or same type of sensors may further be grouped into Sensor Groups.

 

Once you have set up all the required performed experiment entities that you wish to include in a parameter estimation, you must create a new “Parameter estimation” entity.  The complete specification of a Parameter Estimation problem requires some additional information which is not provided in the Process or Performed experiment entities, viz.

  • Information on the unknown parameters to be estimated.
  • The performed experiments to be used.
  • Sensor specifications (if they have not already been defined in the Performed experiment entity). 

 

Similarly to creating a “Performed experiment” entity, right-click on the project in the project tree and select “Parameter estimation”, naming the item before clicking “Ok”.  The new entity will appear, see for example figure below.
