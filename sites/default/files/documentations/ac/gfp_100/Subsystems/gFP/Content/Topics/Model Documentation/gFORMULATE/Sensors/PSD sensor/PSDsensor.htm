PSD sensor

The PSD sensor model monitors the particle size distribution for a selected distributed phase either within active phases or nested in a composite phase. The model report includes various characterisations of a particle size distribution for the selected phase including:

  • volume densities
  • cumulative volume densities
  • particle size means
  • quantiles
  • moments of the distribution
