Shortcut dissolver

The shortcut dissolver model performs non-kinetic dissolution or precipitation of solid phases to reach or approach the saturation point.

The user must specify either an outlet temperature or the heat duty and the model calculates the outlet stream to be at solid-liquid equilibrium or undersaturation. Crystallisation kinetics are not taken into account (for kinetics-based modelling of crystallisation, please use the MSMPR crystallizer or the PF crystallizer models).
