Measurements tab

The measured data of an experiment is entered in the table found on the “Measured data” tab of the Performed experiment entity.  The data table accepts the following information: the name of the measured variable, a name for the sensor used (optional), a description of the measurement variance model for this sensor (optional) and the measured values at discrete points in time.

To select the calculated variable in the flowsheet that corresponds to the measured variable in the experiment, press “Select measured variables…” and find the required attribute (you may also use the variable path name).  The time the measurement is taken at must also be supplied, the units for the time is seconds.  See for example figure below.

If the sensor name and the variance model are not specified in the Performed experiment entity, they will need to be specified in the Parameter Estimation entity.
