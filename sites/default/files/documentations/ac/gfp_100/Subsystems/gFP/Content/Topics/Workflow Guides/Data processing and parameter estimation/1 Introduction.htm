Introduction

This user guide provides guidance on how to process experimental or plant data within gPROMS FormulatedProducts and use the data to estimate parameters of the model or flowsheet in gPROMS FormulatedProducts.  A prerequisite to this is a flowsheet configured to your experiment or process. For guidance on how to build and simulate a flowsheet please see the gPROMS FormulatedProducts user guide “Workflow for Model Configuration and Simulation”.

In addition to parameter estimates, the parameter estimation process in gPROMS FormulatedProducts will generate information on:

  • The quality of the fit, i.e. how well do experimental observations and model predictions match.
  • The confidence intervals of the parameters.  This is important to assess the reliability of the model when using it to predict behaviour under very different conditions than those employed to generate the experimental data used in the parameter estimation process.
  • The cross-correlation between the parameters.  This information indicates whether the data used for parameter estimation contains sufficient information to discriminate between say growth and agglomeration (two size enlargement mechanisms).
