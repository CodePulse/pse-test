Uncertainty analysis

An uncertainty analysis can be used to determine what effect the uncertainty in the factors has on the uncertainty in the responses. This type of analysis can be accomplished using the Monte Carlo method.

The Monte Carlo method performs multiple model evaluations (simulations) with deterministic and/or probabilistic factors. The results of these evaluations are then used to determine the uncertainty in the responses. This method is appropriate for any type of model, regardless of complexity.

The selection of ranges and variability/probability distributions for the factors under consideration must be performed carefully as it can have a significant impact on the ranges and distributions of the responses (see 'Including correlated factors/multivariate distributions' ).
