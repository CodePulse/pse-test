Note on random sampling seed

Generating a set of truly random data points is extremely difficult. In gPROMS, all random generators are pseudo-random and produce a set of points that appears random. The starting point of this random data set is determined by a seed, for which all generated data sets are always identical. The seed values must be positive integer values in the range from 0 to 232-1.

Although not relevant for most GSA applications, each seed has a small bias. This means that certain phenomena may be under-represented. If it is suspected that this might be the case, the same analysis with different seeds can be executed and checked to determine if the results differ significantly.

The pseudo-random nature of the data sets has some significant advantages:

  • It allows reproduction of the results. This is important when, for example, a response is added to further investigate a behaviour observed when the activity was first executed.
  • It facilitates the comparison of analyses performed with different settings to investigate the effect of those settings on the results.
