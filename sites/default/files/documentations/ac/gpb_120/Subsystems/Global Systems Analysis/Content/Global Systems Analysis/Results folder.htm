GSA results file

The GSA results file (with extension .nc) contains the results of successful and failed samples. It is a NetCDF-4 file (version 4.3.3.1) using HDF5 as the underlying file format. This type of file can be read by programmes such as HDFView.

Additional graphical tools can be used to sort and plot the data. For example, colour-coding different sections within the factor space.
