2D scatter plot

A 2D scatter plot is a two-dimensional plot which can display the sample values for factors, responses or a combination of both. It is possible to use a 2D scatter plot to visualise how factor and/or response values are distributed over the sampling space and detect patterns and/or correlations in the data between factors, responses, and factor and responses.

To create a 2D scatter plot within a GSA Case, reproduce the steps below.

  1. Right-click either on the top level grain of the Case or on the 'System Analysis Group' folder.
  2. Select 'New result element...'.
  3. Select 'Scatter 2d' in the drop down menu, provide a name and click on OK.
  4. In the Attributes panel, at the right-hand side, select the variables to plot on each axis, a title and the symbol colour to be used.
