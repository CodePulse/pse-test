Pressure (Tube)

The Pressure (Tube) tab is used to specify tube pressure drop data.

Name Default Unit Description
Pressure drop Selector
On
Calculates the pressure drop based on selected correlation.
Off
Ignores the tube pressure drop.

Pressure drop correlation

 Selector
Specified Fanning factor
Selects specification of the Fanning friction factor.
Smooth pipe
Selects the use of a smooth pipe pressure drop correlation.
Linear mass flow coefficient
Determines the pressure drop from a linear mass flow coefficient.
Haaland
Uses the Haaland equation to calculate the pressure drop.
Custom
Selects user-supplied pressure drop equations listed in the pressure_drop_channel_1D_custom_gML model.

Include effect of gravity

 Selector
No
Ignores gravity effect due to tube inclination in the momentum equation.
Yes
The effect of the inclination angle of the tube is included in the momentum equation.
Fanning friction factor - Assigns the Fanning friction factor.

Angle of inclination

rad

 Assigns the tube inclination angle.
Linear mass flow coefficient kg s-1 Pa-1 Value of the linear mass flow coefficient.
