Stream duplicator

The Stream duplicator model can be used to create repeated outlets of a given stream inlet, i.e., outlets that have the same conditions (flowrate, temperature, pressure, etc.) as the inlet stream. The number of identical outlets is defined by the number of connections made to this unit in the flowsheet topology section.

This unit does not correspond to any physical operation, but can be used to aid in, for example, setting up a flowsheet which models a superstructure of a process in which various alternative implementations or a process are all included on the flowsheet.

This model requires no specifications and has no specification dialog.
