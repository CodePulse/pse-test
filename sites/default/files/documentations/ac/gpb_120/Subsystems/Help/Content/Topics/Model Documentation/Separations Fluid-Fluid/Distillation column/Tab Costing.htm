Costing

The Costing tab is used to specify data relative to the cost calculation of the column.

Name Default Unit Description
Costing Selector
Off
In this mode no costing is carried out.
On
In this mode costing is carried out using the pre-defined cost correlations.
Custom
In this mode the unit cost is determined by a custom cost correlation provided by the user.
CE index - Chemical engineering plant cost index (CEPCI) (Vatavuk, 2002)1Vatavuk, W.M., 2002. Chemical Engineering Journal..
Column material Selector
CS
Uses carbon steel as the column material for the cost calculation.
Low alloy steel
Uses low-alloy steel as the column material for the cost calculation.
Monel400
Uses Monel-400 as the column material for the cost calculation.
SS304
Uses stainless steel 304 as the column material for the cost calculation.
SS316
Uses stainless steel 316 as the column material for the cost calculation.
Titanium
Uses titanium as the column material for the cost calculation.
Tray material Selector
CS
Uses carbon steel as the tray material for the cost calculation.
Low alloy steel
Uses low-alloy steel as the tray material for the cost calculation.
Monel
Uses monel as the tray material for the cost calculation.
SS303
Uses stainless steel 303 as the tray material for the cost calculation.
SS316
Uses stainless steel 316 as the tray material for the cost calculation.
Maximum allowable stress psi Maximum allowable stress supported by the column shell.
Steel density lb in-3 Density of the steel from which the column is constructed.
