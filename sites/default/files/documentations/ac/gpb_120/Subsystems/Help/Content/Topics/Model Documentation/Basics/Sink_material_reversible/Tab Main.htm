Main

The Main tab is used to specify the process material back flow stream data. The Main tab is also used to specify the pressure when a pressure-driven or a pressure-driven (flow-driven initialisation) simulation is carried out.

With the exception of the variables Pressure and Flowrate, under a normal flow direction all the inlet stream properties (e.g. temperature, enthalpy, compositions, etc.) are not equal to the Main tab specified values. In these cases, the inlet conditions are equal to the conditions of the normal flow stream. Under back flow conditions all the inlet stream properties are equal to the Main tab specified values.

Name Default Unit Description
Pressure bar Pressure at the sink.
Temperature K Temperature of the back flow stream.
Vapour fraction kg/kg Vapour fraction of the back flow stream.
Enthalpy kJ/kg Specific enthalpy of the back flow stream.
Degree of subcooling K

The difference between the saturated liquid temperature (bubble temperature) and the actual temperature of the outlet stream contents of the vessel.
Degree of superheating K

The difference between the actual temperature of the outlet stream contents of the vessel and the saturated vapour temperature (dew temperature).
Mass fraction kg/kg Mass fraction of the back flow stream.
Molar fraction mol/mol Molar fraction of the back flow stream.
