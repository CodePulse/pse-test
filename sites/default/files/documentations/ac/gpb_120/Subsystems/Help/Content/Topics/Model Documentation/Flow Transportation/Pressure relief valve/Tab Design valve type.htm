Design (valve type)

The Design (valve type) tab is used to specify data regarding the type of valve used.

Name Default Unit Description

Valve type

 
Conventional spring-operated
Selects a conventional spring-operated valve.
Balanced bellows
Selects a balanced bellows valve.
Pilot-operated
Selects a pilot-operated valve.
Inherent characteristic Selector
Spring or balanced bellows curve
A spring characteristic function is used to determine the flow coefficient fraction.
Characteristic curve hysteresis Selector
Off
The same curve is used for opening and closing.
On
Valve behaves differently when opening and closing.
Full lift pressure Selector
Increasing fraction of set pressure
Minimum pressure at which the valve is completely open, specified as the increment of the set pressure.
Specified here
Minimum pressure at which the valve is completely open is specified.
Primary lift pressure Selector
Increasing fraction of set pressure
Pressure at which primary lift occurs, specified as the increment of the set pressure.
Specified here
Pressure at which primary lift occurs is specified.
Closing pressure Selector
Decreasing fraction of set pressure
Pressure at which the valve starts to close once completely open, specified as the decrement of the set pressure.
Specified here
Pressure at which the valve starts to close is specified.
Reseating pressure Selector
Specify blowdown
Reseating pressure is specified by the blowdown (difference between the set pressure and the reseating pressure).
Specified here
Reseating pressure is specified.

Blowdown fraction

Pilot-operated Selector
Modulated action
Valve has proportional opening and closing.
Pop action
Valve opens and closes instantly.
Primary lift - Valve lift corresponding to the end of the linear phase.
Full lift pressure fraction - Fraction of the set pressure, corresponding to the full lift pressure, i.e. the minimum pressure at which the valve is completely open.

Primary lift pressure fraction - Fraction of the set pressure, corresponding to the primary lift pressure, i.e. the pressure at which primary lift occurs.
Primary lift pressure fraction bar Pressure at which primary lift occurs.
Closing pressure fraction - Fraction of the set pressure, corresponding to the closing pressure, i.e. the pressure at which the valve starts to close once completely open.
Closing pressure bar Pressure at which the valve starts to close once completely open.
Reseating pressure bar Reseating pressure is specified.
Blowdown fraction - Blowdown is specified as a fraction of the set pressure.
