Stoichiometric matrix

The stoichiometric matrix determines the amount of products that are produced or the reactants that are consumed for each biochemical process considered in the model.

Each type of ASM model has an associated stochiometric matrix with its own components, stochiometric units and biological processes definition. An example for the ASM1 model is provided in the corresponding section : 'ASM1'
