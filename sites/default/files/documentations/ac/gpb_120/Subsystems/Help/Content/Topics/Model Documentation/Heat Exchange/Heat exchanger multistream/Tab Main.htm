Main

The Main tab is used to specify data relating to the thermal operation of the heat exchanger.

Name Default Unit Description
Outlet specification types Selector
All specifications of the same type
All output specifications are of the same type (e.g. all are temperatures).
Different types for each specification
Each specification may be of a different type (e.g. for some outputs temperatures are specified and for other the vapour fraction is specified).

Outlet thermal specification

Selector
Temperature
Specify the temperature of the outlet stream.
Vapour fraction
Specify the vapour mass fraction of the outlet stream.
Enthalpy
Specify the mass specific enthalpy of the outlet stream directly.
Saturated liquid (Vapour fraction = 0)
When selecting this option, the outlet stream will consist of a saturated liquid (i.e. the outlet stream temperature is the bubble temperature of the mixture).
Saturated vapour (Vapour fraction = 1)
When selecting this option, the outlet stream will consist of a saturated vapour (i.e. the outlet stream temperature is the dew temperature of the mixture).
Subcooled liquid
When selecting this option, the outlet stream will consist of a subcooled liquid. The delta temperature of subcooling is specified.
Superheated vapour
When selecting this option, the outlet stream will consist of a superheated vapour. The delta temperature of superheating is specified.
Stream to be determined from energy balance Selector One stream outlet temperature is determined from the global energy balance over all the streams.
Specification type Selector For each outlet stream, select which type of specification (Temperature, Vapour fraction, Enthalpy, etc) will be used for that particular outlet stream.
Outlet temperatures K Temperatures for all the outlet streams except one
Outlet vapour fraction - Vapour fractions for all the outlet streams except one.
Enthalpy kJ/kg Mass specific enthalpies for all the outlet streams except one.
Outlet degree of subcooling K The number of degrees the outlet temperature is below the bubblepoint.
Outlet degree of superheating K The number of degrees the outlet temperature is above the dewpoint.
