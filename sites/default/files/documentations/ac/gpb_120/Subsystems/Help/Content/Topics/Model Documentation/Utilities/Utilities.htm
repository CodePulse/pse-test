Utilities

Two model libraries are supplied within gPROMS Model Library for Utilities Systems (gML Utilities). gML Utilities contains the models to represent supply and demand of utilities on site, conversion and distribution of utilities by site operational management, and to evaluate costs, constraints and other key performance indicators. These models are visible on the Palette. gML Utilities interface contains the variables and connections used in these models: these can be used to create customised models which can be connected to the library models.

Many of the library models contain an 'Interface' tab. Once a model of a utility system is built, configured and tested, it can also be deployed within the gPROMS Utilities Planner and gPROMS Utilities Advisor applications. The interface tab is used to allow easy customisation of these applications. The options in this tab are described in the 'The Interface tab'. These options relate to gPROMS Utilities Planner and gPROMS Utilities Advisor only and do not affect performance of the models.
