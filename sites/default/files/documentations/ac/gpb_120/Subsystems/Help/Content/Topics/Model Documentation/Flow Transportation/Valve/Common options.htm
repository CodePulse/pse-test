Common options

The menus at the top of the Specification dialog are used to specify the calculation mode.

Name Default Unit Description
Mode Selector
Mass and energy balances only
Turns off the design and performance modes. Typically in this mode the outlet pressure or pressure drop is assigned, but a simple linear flow-pressure relationship that does not take into account choking, can also be used.
Design
Turns on design mode. This mode should be used to determine the valve flow coefficient for a specified flowrate and pressure drop.
Performance
Turns on performance mode. In this mode the valve predicts the outlet pressure for a given flowrate and stem position or the flowrate for a given pressure drop. This mode is necessarily required in pressure-driven systems to express the flowrate (pressure drop) as a function of the pressure drop (flowrate).
