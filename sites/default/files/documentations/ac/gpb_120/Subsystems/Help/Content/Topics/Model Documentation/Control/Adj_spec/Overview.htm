Adj spec

The Adj spec model can be used to trade off specifications between different models. This is done by imposing a specification ("target") on a variable of a particular model and at the same time removing a specification on a variable ("adjusted variable") of another model. This variable can then be varied in order to meet the target specification.

This principle can be used for various reverse calculations, imposing performance specifications and to represent controllers in steady-state problems. For example, this model may be used to determine the flowrate (adjusted variable) of a given inlet mixer stream that generates a particular composition (target variable) in the outlet mixer stream. Both the adjusted and target variables are specified by the user in the Adj_spec dialog.

Depending on which specifications are traded, the resulting problem can be more or less difficult to converge. To aid in the convergence of problems that become more difficult, the Adj spec model permits the specification of an initial guess for the adjusted variable. The flowsheet will first be converged while imposing the initial guess on the adjusted variable, after which the problem is changed such that the target specification is imposed.
