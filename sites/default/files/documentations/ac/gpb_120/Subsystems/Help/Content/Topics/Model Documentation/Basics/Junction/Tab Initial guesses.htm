Initial guesses

The Initial guesses tab is used to provide initial guesses for some variables in order to improve the speed and robustness of initialisation.

Name Default Unit Description
Specify Selector
All streams except one (last stream)
All streams except the last stream must be specified. The number of entry fields that will be shown is equal to the number of streams minus one.
All streams (leave one stream specification blank)
All streams except one must be specified (the stream left blank is not necessarily the last one). The number of entry fields that will be shown is equal to the number of streams. One entry field needs to be left blank. The entry field that is left blank may correspond to a different component for each stream.
N.B. Entering values in all fields will result in an overspecified problem.

Specification type

Selector
Split fraction
Supplies initial guesses for all the split fractions of the outlet streams except one.
Flowrate
Supplies initial guesses for all the flowrates of the outlet streams except one.
Initial flowrate Selector
Equal to inlet flowrate
Supplies a guess for the initial flowrate equal to the inlet flowrate.
Specified
Supplies a specified guess for the initial flowrate.
Outlet pressure specification Selector
Minimum of inlet pressures
The outlet stream pressure is set equal to the pressure of the inlet stream with the lowest pressure.
Specified pressure
The outlet stream pressure is set by the user.
Pressure of a particular inlet stream
The outlet stream pressure is set equal to the pressure of an inlet stream specified by the user.
Flowrate basis Selector
Mass
Mass basis is selected for flowrate specification.
Molar
Molar basis is selected for flowrate specification.
Split fractions - The split fraction is a value between 0 and 1 which determines the proportion of the total inlet flow that will initially be split into each outlet.
Molar flowrate kmol/h Sets the Initial total molar flowrate of the outlet streams.
Outlet pressure bar Initial value of the outlet stream pressure.
Outlet pressure equal to pressure of stream - Index of the inlet stream used to define the initial outlet pressure.
