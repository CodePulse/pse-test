Dynamics (Wall)

The Dynamics (Wall) tab is used to specify dynamic mode and wall initial conditions data. This tab can be turned off and on.

Name Default Unit Description
Dynamics Selector
Off
Simulates the wall in steady-state mode. If this option is selected the wall will always be in steady-state.
On
The wall dynamic energy equation is included in the model and therefore a set of initial conditions are required.

Initial conditions

Selector
Steady-state
The model is initially at steady-state.
Dynamic
The initial values for all the state variables of the unit must be specified.

Initial temperature

 K

Initial wall temperature.
