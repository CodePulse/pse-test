Initial guesses

The Initial guesses tab is used to provide initial guesses for some variables in order to improve the speed and robustness of initialisation.

Name Default Unit Description
Liquid level specification Selector
Equivalent to feed phase fraction
The liquid level fraction inside the vessel is equal to the inlet feed phase vapour fraction.
Specified here
The liquid level is specified internally (Operation tab).

Liquid level fraction specification basis

 Selector
Liquid level fraction
Supplies an initial guess for the fraction of the liquid level relative to the height of the vessel.
Liquid volume fraction
Supplies an initial guess for the fraction of the liquid volume relative to the total volume in the vessel.
Liquid phase fraction on mass basis
Supplies an initial guess for the liquid phase fraction on mass basis in the vessel.
Liquid level fraction m/m Fraction of liquid level relative to the vessel height.
Liquid volume fraction m3/m3 Fraction of liquid volume relative to the vessel volume.
Liquid phase fraction on mass basis kg/kg Fraction of liquid mass relative to the total mass inside the vessel.
