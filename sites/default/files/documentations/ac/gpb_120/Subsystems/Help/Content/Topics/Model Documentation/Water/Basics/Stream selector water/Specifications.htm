Specifications

The Stream selector water specifications dialog comprises the following variables:

Name Default Unit Description
Stream selection Selector
Specify stream number (usable for optimisation)
The option to select the outlet stream to be equal to a particular inlet stream. During (Mixed Integer) optimisation problems, the inlet stream number to which the outlet stream is equal can be varied.
Specify SOS-1 array (usable for optimisation)
The option to select the outlet stream to be equal to a particular inlet stream by specifying a SOS-1 array of zeros. During (Mixed Integer) optimisation problems, the inlet stream number to which the outlet stream is equal can be varied.
Specify stream number (fixed)
The option to select the outlet stream to be equal to a fixed inlet stream. This is equivalent to the option to specify the stream number in a way that is suitable for optimisation, but does not permit the stream number to be optimised. However, if the stream number will not be optimised then this options may be marginally faster or more robust.
Index of selected inlet - Sets the index of the selected inlet
Stream selection array - Sets the flow ratio values
