Timing

The Timing tab is used to specify the time duration for each step.

Name Default Unit Description
Steps with termination condition Selector
None
All steps have a fixed time length.
Specify
Specify the steps which have a termination condition. These are steps which do not have a fixed time duration but instead are terminated when a certain input signal to the Schedule block becomes larger than 0. For example, using models from the gMLSignal library a comparison between two pressures can be made and when one is higher than the other the input signal to the Schedule block will become larger than 0, terminating a pressurization step in an adsorption bed.

Steps with termination at a fixed time

Selector
None
All steps either have a fixed time length or are terminated when a certain condition is met.
Specify
Specify the steps which are terminated at a certain time. This time is relative to the start time of the cycle.

Time duration per step

 s

Time duration for the steps that have a fixed time duration.

Steps terminated by external condition

 -

Select the steps that are terminated using an eternal condition.
Termination condition per step. - Select the connections that carry the signals that are used as termination conditions for each step that is terminated by a condition. The step will terminate if the condition is greater than zero.
Steps terminated at a specific time - Select the steps that are terminated at a specific time. This time is relative to the start time of the cycle.

Final time per step

 s

Final time for steps that are terminated at a specific time. This time is relative to the start time of the cycle.
