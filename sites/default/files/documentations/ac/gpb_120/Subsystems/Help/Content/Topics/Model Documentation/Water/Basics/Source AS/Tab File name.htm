Influent file name

Name Default Unit Description
Influent file name -
Defines the file from which the influent water characteristics are read.

This file must be placed in the 'Miscellaneous files' folder in the project tree.

The format of the file should be the same as the image below. Please note that this example is only valid for the default ASM1 model combined with the default water chemistry model. For others models, new columns should be created where relevant and the order of the columns should correspond to the output of the source model. We recommend copy-paste the default 'BSM1_dry' file into a data-management software to create new ones.

The units used in the file are the same as the default ones in the specifications dialog. The first column is the time in 'days'.
