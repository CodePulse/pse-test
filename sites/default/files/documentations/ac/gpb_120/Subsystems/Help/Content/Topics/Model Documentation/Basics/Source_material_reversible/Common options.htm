Common options

The menus at the top of the specification dialog are used to select options related to how the stream conditions are specified.

Name Default Unit Description

Thermal specification

Selector
Temperature
Vapour fraction
Enthalpy
Saturated liquid (Vapour fraction = 0)
When selecting this option, the outlet stream will consist of a saturated liquid (i.e. the outlet stream temperature is the bubble temperature of the mixture).
Saturated vapour (Vapour fraction = 1)
When selecting this option, the outlet stream will consist of a saturated vapour (i.e. the outlet stream temperature is the dew temperature of the mixture).
Subcooled liquid
When selecting this option, the outlet stream will consist of a subcooled liquid (i.e. outlet temperature is calculated using the provided degree of subcooling relative to the saturated liquid).
Superheated vapour
When selecting this option, the outlet stream will consist of a superheated vapour (i.e. outlet temperature is calculated using the provided degree of superheating relative to the saturated vapour).
Component specification Selector
Component fractions and overall flowrate
Selects specification of the component fractions and total flowrate.
Component flowrates
Selects specification of the component flowrates.
Composition specification Selector
Mass
Mass basis is selected for composition specification.
Molar
Molar basis is selected for composition specification.
Flowrate specification Selector
Mass
Mass basis is selected for flowrate specification.
Molar
Molar basis is selected for flowrate specification.
Volumetric (standard)
Selects volume basis at standard temperature and pressure conditions.
Volumetric
Selects volume basis.
