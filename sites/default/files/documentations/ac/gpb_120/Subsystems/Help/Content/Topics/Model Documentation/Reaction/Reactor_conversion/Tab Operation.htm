Operation

The Operation tab is used to assign thermal specifications.

Name Default Unit Description
Thermal specification Selector
Adiabatic operation
Selects no heat exchange with surroundings (sets heat duty to 0).
Heat duty
Selects specification of a transferred heat duty.
Temperature difference
Selects specification of the temperature difference between the outlet stream and the inlet stream.
Temperature
Vapour fraction
Enthalpy
Saturated liquid (Vapour fraction = 0)
When selecting this option, the outlet stream will consist of a saturated liquid (i.e. the outlet stream temperature is the bubble temperature of the mixture).
Saturated vapour (Vapour fraction = 1)
When selecting this option, the outlet stream will consist of a saturated vapour (i.e. the outlet stream temperature is the dew temperature of the mixture).
Subcooled liquid
When selecting this option, the outlet stream will consist of a subcooled liquid (i.e. outlet temperature is calculated using the provided degree of subcooling relative to the saturated liquid).
Superheated vapour
When selecting this option, the outlet stream will consist of a superheated vapour (i.e. outlet temperature is calculated using the provided degree of superheating relative to the saturated vapour).
Duty specification Selector
Specified here
Selects internal specification of a transferred heat duty (tab Operation).
From energy connection
The heat duty is obtained externally using a model connected to the gMLEnergy port.
