Chemical equilibrium calculation in Multiflash

The Multiash chemical equilibrium module in general allows performing simultaneous phase and chemical equilibrium calculations. gPROMS provides an interface to a single gas phase calculation at given temperature, pressure and input composition, ChemEquilibrium1P. The Multiflash ChemEquilibrium1P method calculates the composition at chemical equilibrium by performing a Gibbs energy minimisation algorithm .

Equilibrium flash physical property functions and their arguments
Property Name Inputs Description Type
ChemEquilibrium1P T,P,n Composition at chemical equilibrium Vector

The result vector is of size NoComp.

Notes:

  • Since the method is restricted to single phase gas calculations only, the Multiflash configuration file used in the model has to be limited to include just a single gas phase description. If a liquid phase is defined, the calculation will fail, even though the liquid phase is not present at the conditions the calculation is performed.

  • Analytical derivatives are currently provided and gPROMS will calculate them via perturbation.

  • The module requires a separate license.
