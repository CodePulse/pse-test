Public Model Attribute Chooser

The intelligent selection dialog for choosing Public Model Attributes (PMA) is now also available for the GUI editor for schedules and tasks where it allows the specification of the REASSIGN, REINITIAL, REPLACE and SWITCH elementary tasks. An example is shown below for a REASSIGN elementary task with several PMAs.

Value table specification

When Select is pressed, a dialog appears which displays all the units on the flowsheet in a collapsible tree view and permits the choice of the desired attributes. Note that the active choice is displayed with check marks and that multiple entries can be chosen at the same time.

PMA chooser tree view

This graphical chooser presents the attributes that the model developer has denoted as public model attributes and which can be meaningfully be chosen in a given context. For the REASSIGN, REINITIAL and REPLACE elementary tasks, all PMAs that are gPROMS variables are displayed and it is not possible to narrow the choice down further because it cannot be determined reliably if a PMA indeed has been used as an assigned variable before or not.

For the SWITCH elementary task, only those PMAs are displayed which correspond to a gPROMS selector.

The PMA chooser is a very convenient alternative to using the intelligent keyboard input which continues to be accessible.
