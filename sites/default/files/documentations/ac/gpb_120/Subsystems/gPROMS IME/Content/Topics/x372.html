Project properties

Each Project records information relating to the user(s) who created and last modified it, as well as the dates and times of creation and last modification. This information is read-only and is recorded and maintained automatically by gPROMS ProcessBuilder under the Project's Properties.

Project Properties can be viewed by right-clicking on the Project's name in gPROMS ProcessBuilder's navigation tree and selecting Properties in the context-sensitive menu that appears.

Project properties dialog

The Properties dialog has a second tab that controls the list of Projects that can be cross-referenced.
