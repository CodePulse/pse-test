2D plots

The data can be viewed by selecting the Graph -> View Data... menu-item. The data is displayed in a table with the values of the free-domain in the first column and the values of lines plotted against that domain in subsequent columns. If the plot contains lines from variables in different Processes, or lines plotted against different free-domains then multiple tables are displayed.

Windows

The data can be exported to a file by selecting Graph -> Export Data.... This displays a Windows file dialog for you to specify the file name and type.

Linux

The data can be exported to a file by selecting either the Graph -> Export Data -> Tab Delimited Table... or Graph -> Export Data -> Space Delimited Table... menu-item. This displays a standard Motif file dialog for you to specify a file name.
