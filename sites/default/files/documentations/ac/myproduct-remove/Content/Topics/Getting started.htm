Getting started with gPROMS Utilities Briefer

gPROMS Utilities Briefer is an advanced simulation and optimisation tool for model-based support of utility management and operating decisions. Built on the gPROMS Platform, the software allows users to quickly construct flowsheets representing large-scale utility management systems. Models for supply and demand, generation, conversion, and distribution of utilities are selected from a model library palette and positioned and connected as required using 'drag and drop'. The process flowsheet can then be used for simulating "what-if" scenarios, and for optimising equipment usage decisions such as: which boilers to operate? Whether to use steam or electric driven pumps? Which utilities to convert? How to route steam around the site?

The fundamental layout of the user interface and a detailed description of all its features are given in the gPROMS Utilities Briefer Guide. For a description of the available models and their purpose and functionality see Model Library. The models and solvers for simulation and optimisation are flexible and can be customised by the user or on request.

A series of sequential workflow guides are available for configuring a runtime solution. To get started directly with developing flowsheet models, see Developing models with gPROMS Utilities Briefer. To optimise flowsheet models, see Configuring and running optimisations. To configure and use the Excel-base Runtime interface, see Exporting models for remote execution and Using the Excel interface to optimise multiple scenarios.

These workflow guides are described below.

Workflow guide name Description
Developing models with gPROMS Utilities Briefer Workflow guide describing how to add units to a flowsheet using the gPROMS Utilities Briefer model library, and how to simulate "what-if" scenarios.
Configuring and running optimisations Workflow guide describing how to optimise flowsheets in gPROMS Utilities Briefer, and how to view, interpret and simulate the results.
Exporting models for remote execution Workflow guide describing how to prepare and export a model for remote execution in order to allow optimisation of variable availabilities, demands and contract prices.
