Specifying partial derivative outputs

As stated before, gFOMD must only return the partial derivatives of the method with respect to the one input specified by gPROMS in DInputName. This is because gPROMS calls for the partial derivatives separately with respect to each input: returning all of the partial derivatives with respect to all of the inputs at the same time in the DerivOutputs argument will cause errors.
