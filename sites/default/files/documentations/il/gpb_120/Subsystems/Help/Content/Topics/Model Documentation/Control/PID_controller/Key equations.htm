Non-interactive and interactive formulations

In the ideal PID algorithm, the integral (I) and derivative (D) act on the input signal and their outputs are added to obtain the response. This is referred to as the non-interactive formulation. In many implementations the integral (I) and derivative (D) are realized in sequence. This is referred to as the interactive formulation of the PID algorithm. Both formulations have a different response for the same tuning parameters. In the configuration dialog a selection can be made between the two, with the non-interactive algorithm being the default option. Terms acting on error of process variable

In the ideal PID control algorithm all terms (P,I,D) act on the error signal. To limit the magnitude of PID responses which setpoints are changed, in the configuration dialog terms can be chosen to act on the process variable (PV) instead of on the the error signal.

This is important to obtain a good controller response when either the set point or the process variable changes such that the set point can be achieved again.
