Transport properties

The Transport properties tab is used to specify the transport properties of material.

Name Default Unit Description
Liquid heat transfer coefficient Selector
Fixed value
Uses user-specified constant liquid heat transfer coefficient.
User supplied correlation
Uses user-defined correlation for heat transfer coefficient.
Vapour heat transfer coefficient Selector
Fixed value
Uses user-specified constant vapour heat transfer coefficient.
User supplied correlation
Uses user-defined correlation for heat transfer coefficient.
Diffusion Selector
Fixed value
Uses user-specified binary diffusion coefficient.
From physical property package
Uses diffusion coefficients obtained from physical property package.
User supplied correlation
Uses user-defined correlation for diffusion coefficients.
Viscosity Selector
Fixed value
Uses user-specified viscosity.
From physical property package
Uses viscosity obtained from physical property package.
User supplied correlation
Uses user-defined correlation for viscosity.
Surface tension Selector
Fixed value
Uses user-specified surface tension.
From physical property package
Uses surface tension obtained from physical property package.
User supplied correlation
Uses user-defined correlation for surface tension.
Liquid heat transfer coefficient W m-2 K-1 Liquid phase heat transfer coefficient of material.
Vapour heat transfer coefficient W m-2 K-1 Vapour phase heat transfer coefficient of material.
Liquid binary diffusion coefficients m2/s Diffusion coefficients between components in liquid phase.
Vapour binary diffusion coefficients m2/s Diffusion coefficients between components in vapour phase.
Liquid viscosity Pa s Liquid phase viscosity.
Liquid viscosity (Reference conditions) Pa s Liquid phase viscosity at the reference temperature and pressure. This is used to correct binary diffusion coefficients which are calculated at the reference temperature and reference pressure and then corrected using temperature and viscosity at each point in the column.
Vapour viscosity Pa s Vapour phase viscosity.
Surface tension N/m Surface tension.
