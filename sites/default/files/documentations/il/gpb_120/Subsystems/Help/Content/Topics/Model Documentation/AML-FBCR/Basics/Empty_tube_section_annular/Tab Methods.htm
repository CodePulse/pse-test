Methods

The Methods tab is used to specify modelling and correlation options.

Name Default Unit Description

Bulk reactions

Selector
Off
Disable bulk reactions.
On
Enable bulk reactions (custom kinetic models for bulk reactions need to be provided).
Tube type Selector
Rough
Select a rough tube to include the effect of relative roughness in the calculation of the friction factor.
Smooth
This option should be selected for tubes with low relative roughness.

Heat transfer correlation

Selector
Dittus Boelter
Fluid-wall heat transfer correlation applicable to turbulent flow conditions, with 0.6<Pr<160 and Re>10000.
Gnielinski
Fluid-wall heat transfer correlation applicable to turbulent flow conditions, with 0.5<Pr<2000 and 3000<Re<5x106.

Friction factor correlation

Selector
Blasius
Friction factor correlation for smooth tubes.
Hermann
Friction factor correlation for smooth tubes.
Prandtl and von Karman
Friction factor correlation for smooth and rough tubes.
Colebrook and White
Friction factor correlation for rough tubes.
Churchill
Friction factor correlation for rough tubes.
Haaland
Friction factor correlation for rough tubes.
User specified value
The friction factor is provided by the user.

Heat transfer mode

Selector
Heating
Selects a heating mode for the Dittus Boelter calculation.
Cooling
Selects a cooling mode for the Dittus Boelter calculation.
Heat transfer adjustable factor -

Correction factor for the fluid-wall heat transfer coefficient.
Friction factor adjustable factor -

Correction factor for the friction factor.
Friction factor - User specified friction factor.
Wall roughness m Roughness of the inside tube wall.
