Numerics

The Numerics tab is used to specify model robustness parameters.

Name Default Unit Description
Film dimension Selector
0D
The stagnant films are represented using a 0-D model. This can speed up calculations at the cost of accuracy.
1D
The stagnant films are represented using a 1-D model. This can increase accuracy, in particular if reactions are present.
Initialisation procedure Selector
Automatic (Robust)
Automatic (robust) is a unit initialisation procedure that is optimised for robustness.
Automatic (Fast)
Automatic (fast) is a unit initialisation procedure that is optimised for speed.
Automatic (Series of steps)
Automatic (Series of steps) is a unit initialisation procedure which aims to strike a balance between speed and robustness. The number of steps can be specified. Increasing the number of steps increases robustness at the cost of speed of initialisation.
Manual
Allows selection of one of the existing initialisation procedures.
Off
Does not use an initialisation procedure for this unit. This option can be used when the unit is initialised from a Saved Variable Set (SVS).
Number of elements in axial domain - Number of discretisation points in axial domain.
Non condensable components -
Select one or more components which are assumed to be non-condensable. These components will be assumed not to be present in the liquid phase. Selecting these components can improve solution speed.
Non evaporating components - Select one or more components which are assumed to be non-evaporating. These components will be assumed not to be present in the vapour phase. Selecting these components can improve solution speed.
Step sizes grid transformation - This parameter affects how the homotopic path that is followed during robust initialisation and series of steps initialisation is formed. Specify 0 for linear ramping when contacting phases during initialisation or increase the value to enforce a more gradual ramp at the start.
