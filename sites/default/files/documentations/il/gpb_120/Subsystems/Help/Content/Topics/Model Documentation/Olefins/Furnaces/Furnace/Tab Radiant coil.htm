Radiant coil

The Radiant coil tab is used to select the coil type and specify the coil geometry. The geometry of each coil section can be specified. A coil section with the suffix "a" indicates an adiabatic section. The list of available coils, as well as their geometries can be found in 'Standard coil geometries'.
