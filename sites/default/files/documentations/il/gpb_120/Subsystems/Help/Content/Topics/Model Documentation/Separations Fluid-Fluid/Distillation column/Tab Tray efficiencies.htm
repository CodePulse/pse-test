Tray efficiencies

The Tray efficiencies tab is used to specify data relative to the efficiency of the trays of the column.

Name Default Unit Description
Specification Selector
Equilibrium
Assumes thermodynamic equilibrium at each stage.
Murphree tray efficiencies
Models non-equilibrium separation using Murphree efficiencies (see Key equations).
Efficiency specification method Selector
Tray efficiency per section
The same tray efficiency is applied to the whole section
Tray efficiency
A different tray efficiency is applied to each tray
Efficiency per component per section
The same efficiency per component is applied to all the trays in the section
Efficiency per component per tray
A different efficiency is applied for each component on each tray
Number of sections -

Number of sections in the column. The minimum value is one.
Section start and end tray numbers - Index of the first and last tray of each section. All the trays in the column should be included. For example, in a 8 trays distillation column, stage 1 is the condenser and stage 10 is the reboiler. If there is only one section defined, it should be from tray 1 to tray 8.
Tray efficiency per section - Specifies one single efficiency for all components in all the tray in the section.
Tray efficiency - Specifies the same efficiency for all the components in a single tray.
Efficiency per section per component - Specifies one efficiency per component for all the trays in a section.
Efficiency per tray per component - Specifies a different efficiency for each component and each tray.
