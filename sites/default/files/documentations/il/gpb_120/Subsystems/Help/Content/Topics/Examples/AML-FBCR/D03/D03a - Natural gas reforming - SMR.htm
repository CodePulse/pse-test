D03a - Natural gas reforming - SMR

In this example, a steam methane reformer is modelled considering the reformer tubes and the furnace. The reformer tubes are heated inside the furnace with the very end of the tube outside the furnace considered to be adiabatic. The AML:FBCR is applied to model in detail the phenomena inside the catalyst bed inside the reformer tubes. A simplified model is used to calculate the heat from the furnace side.

The flowsheet models included represent the furnace side in two different approaches. In SMR_reactor_in_fixed_heater the wall temperature profile is provided and in SMR_reactor_in_fired_heater a fired heater model using a simplified radiative heat flux calculation is used to model the furnace.

The S_C_ratio model is used to allow the user to specify inlet S/C ratio at the flowsheet level. Some gML Signal models are used to specify temperature and pressure specifications in only one of the two process material sources.

The processes (D03a_SMR_reactor_in_fixed_heater_Init and D03a_SMR_reactor_in_fired_heater_Init) can be used to initialise the models from poor initial guesses - the existing Saved Variable Sets will be updated if the option to auto-update is selected upon starting a simulation. The remainder processes use the saved variable sets to initialise the simulation.

D03a_SMR_reactor_in_fixed_heater_Init process for initialisation of model "D03a_SMR_reactor_in_fixed_heater" and to generate a Saved Variable Set
Saved Variable Sets Description
SMR_in_fixed_heater Saved Variable Set used for all processes and generated from process D03a_SMR_reactor_in_fixed_heater_Init
