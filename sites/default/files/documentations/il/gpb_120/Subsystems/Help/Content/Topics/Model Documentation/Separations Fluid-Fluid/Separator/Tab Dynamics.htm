Dynamics

The Dynamics tab is used to insert dynamic mode and initial conditions data.

Name Default Unit Description
Dynamics Selector
Off
Simulates the unit in steady-state mode. If this option is selected, the unit will always be in steady-state mode.
On
Simulates the unit in dynamic mode. If this option is selected, changes in time for mass, momentum and energy holdups will be automatically calculated.

Mode

 Selector
Flow-driven
Simulation mode where the flowrates are specified at the sources or in splitter units.
This mode can be used to perform steady-state material and energy balances. It is not necessary to define flow-pressure relationships for flow-transportation equipment.
Pressure-driven (flow-driven initialisation)
Simulation mode where pressures at sources and sinks are specified and where flowrates are calculated, based on these prossures and the flow-pressure relationships in the unit operations. To facilitate initialisation of a steady-state process in pressure-driven mode, a flow-driven simulation is performed prior to the pressure driven one and the results from this simulation are used as an initial guess for the pressure driven simulation.
This mode requires the definition of flow-pressure relationships for all flow-transportation equipment (e.g. valves, pipes, compressors, pumps and turbines should be set to Performance Mode). It also requires the specification of initial guesses for the flowrates of all sources and for split fractions of all splitters. These will be used for the flow-driven initialisation.
Pressure-driven (flow-driven initialisation, apply results to pressure driven case)
Simulation mode where the pressures at sources and sinks are specified and where flowrates are calculated, based on these pressures and on the flow-pressure relationships in the unit operations. To facilitate initialisation of a steady-state process in pressure-driven mode, a flow-driven simulation is performed prior to the pressure driven one and the results from this simulation are used as an initial guess for the pressure driven simulation. The pressures at the sink, for the pressure-driven mode, are set to the pressures determined during the flow-driven initialisation mode.
This mode requires the definition of flow-pressure relationships for all flow-transportation equipment (e.g. valves, pipes, compressors, pumps and turbines should be set to Performance Mode). It also requires initial guesses for the flowrate of all sources and splitters for the flow-driven initialisation.
Pressure-driven
Simulation mode where the pressures at sources and sinks are specified and where flowrates are calculated, based on these pressures and on the flow-pressure relationships in the unit operations. This mode can be used when no initialisation in flow-driven mode is required (e.g. when the flowsheet is not initialised at steady-state or when the initial state is restored from a snapshot created using a previous simulation).
This mode requires the definition of flow-pressure relationships for all flow-transportation equipment (e.g. valves, pipes, compressors, pumps and turbines should be set to Performance Mode).

Initial conditions specifications

 Selector
Steady-state
The model is initially at steady-state.
Dynamic
The initial values for all the state variables of the unit must be specified.

Initial holdup

Selector
Composition (w) and intensive variable (P, T, h)
Determines the initial material holdup in the vessel from the composition and an intensive variable (pressure, temperature or enthalpy).
Overall holdup (M_T) and composition (w)
Determines the initial material holdup in the vessel from the overall holdup and the composition.
Component holdup (M)
Specifies the initial holdup of each component directly.

Holdup specification

 Selector
Mass
Selects mass basis for the specification of holdup or composition.
Molar
Selects molar basis for the specification of holdup or composition.

Initial thermal specification Selector

Vapour fraction
Enthalpy
Saturated liquid (Vapour fraction = 0)
When selecting this option, the vessel will initially consist of a saturated liquid (i.e. the vessel temperature is the bubble temperature of the mixture).
Saturated vapour (Vapour fraction = 1)
When selecting this option, the vessel will initially consist of a saturated vapour (i.e. the vessel temperature is the dew temperature of the mixture).
Subcooled liquid
When selecting this option, the vessel will initially consist of a subcooled liquid (i.e. outlet temperature is calculated using the provided degree of subcooling relative to the saturated liquid).
Superheated vapour
When selecting this option, the vessel will initially consist of a superheated vapour (i.e. outlet temperature is calculated using the provided degree of superheating relative to the saturated vapour).
Liquid volume fraction
When selecting this option, the vessel will initially hold a specific volume fraction of liquid.
Liquid level fraction
When selecting this option, the vessel will initially hold a specific level fraction of liquid.
