Reporting

The Reporting tab is used to select data to be reported once the simulation has converged.

Name Default Unit Description
TMT Selector
Maximum value
Reports the maximum value of tube metal temperature (TMT) in the radiant coil.
Value at specified location
Reports tube metal temperature at a given position relative to the radiant coil outlet.
Components for calculating conversion -

The components for which the axial profile of conversion is reported. This specification is independent of the components selected for conversion specification ('Operation').
TMT measurement (relative to coil outlet) m

The position relative to coil outlet at which the tube metal temperature is measured. N.B. The reported value will be at the closest discretisation point in the axial domain.
