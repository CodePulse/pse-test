Configuration

The temperature and heat transfer coefficient parameters are specified in the Configuration tab.

Name Default Unit Description
Heat transfer coefficient specification Selector
Specified
The cooling section heat transfer coefficient is specified in this tab.
From bus port
The cooling section heat transfer coefficient is provided by connection with the setPort.
Temperature profile Selector
Constant
Selects a constant axial temperature profile for the cooling fluid.
Linear
Selects a linear axial temperature profile for the cooling fluid.
Custom
Selects a customised temperature profile.
Temperature specification Selector
Specified
The temperature of the cooling fluid is specified in this tab.
From bus port
The temperature of the cooling fluid is provided by connection with the setPort.
Heat transfer coefficient Wm-2K-1 Cooling section heat transfer coefficient.
Temperature K Temperature of the cooling fluid.
Temperature at inlet (z=0) K Temperature of the cooling fluid at the inlet.
Temperature at outlet (z=1) K Temperature of the cooling fluid at the outlet.
