Key equations

The reactions can take place either on the tube or on shell side of the reactor. In this description it will be assumed these take place on the tube side. Except for where the reaction terms are added to the mass balance, the description would be similar for the case where they would occur on the shell side.

Furthermore, in this description it is assumed that both fluid flows, through the reactor tubes and through the shell, flow co-currently. The equations for the counter-current option are similar, with the respective boundary conditions changed.

Tube-side mass and energy balances
