Deaerator

The Deaerator model can be used to represent a vessel for feedwater storage and purification at saturation. The model has inlet streams for water and steam, and an outlet feedwater stream. The Deaerator is configured by specifying the pressure and vent ratio. The model icon, as it appears on the Palette, is shown below.
