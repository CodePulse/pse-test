Numerics

The Numerics tab is used to specify model robustness parameters.

Name Default Unit Description
Low flow protection Selector
On
Turns on the option to handle mass and energy balances differently when flowrates are close to zero. This will affect the way in which compositions are calculated (e.g. if two streams with a zero flowrate are mixed, the composition of the resulting stream will be equal to the composition of one of the inlet streams).
Off
The model uses the calculated flowrates in mass and energy balances in the normal way, even if the flowrates are close to zero. This option may result in shorter simulation times but might not be robust for very small or zero flowrates in certain circumstances.
Initialisation procedure Selector
Automatic (Fast)
Automatic (Fast) is a unit initialisation procedure that is optimised for speed. This initialisation procedure may occasionally fail for some problems. If this happens, use the Automatic (Robust) option.
Automatic (Robust)
Automatic (Robust) is a unit initialisation procedure that is optimised for robustness. This initialisation procedure should work in all cases but is generally slower than the Automatic (Fast) option.
Manual
Allows selection of any one of the existing initialisation procedures. This option is recommended for advanced users only.
Off
Do not use an initialisation procedure for this unit. This option can be used when the unit is initialised from a Saved Variable Set (SVS).
Typical flowrate kg/s Specifies the typical flowrate for this simulation. Under normal circumstances, this flowrate parameter does not need to be specified but for some simulations, with either very small or very large flowrates in (part of) the flowsheet, setting this parameter may improve the accuracy of the results.
Liquid volume fraction threshold m3/m3 Minimum liquid volume fraction below which the system will be considered all vapour.
Low inlet flow threshold kg/s The low flow threshold is the point below which the contribution of the flows to the mass and energy balances will be handled differently in order to improve robustness.
Typical reaction rate Units depend on the composition basis Specifies the typical reaction rate for this particular simulation. In general, the typical reaction rate parameter does not need to be specified but for some simulations with either very small or large reaction rate in (part of) the flowsheet setting this parameter might improve the accuracy of the results.
Typical pre-exponential factor Units depend on the composition basis Specifies the typical pre-exponential factor for this particular simulation. In general, the typical pre-exponential factor parameter does not need to be specified but for some simulations with either very small or large pre-exponential factor in (part of) the flowsheet setting this parameter might improve the accuracy of the results.
Typical equilibrium constant J/mol Specifies the typical equilibrium constant for this particular simulation. In general, the typical equilibrium constant parameter does not need to be specified but for some simulations with either very small or large equilibrium constant in (part of) the flowsheet setting this parameter might improve the accuracy of the results.
