Methods

The Methods tab is used to specify modelling and correlation options.

Name Default Unit Description

Bulk reactions

Selector
Off
Disable bulk reactions.
On
Enable bulk reactions (custom kinetic models for bulk reactions need to be provided).

Partial enthalpy calculation

Selector
Ideal mixture
Calls the vapour enthalpy method for pure components.
Real mixture
Obtains an estimate of partial enthalpy from vapour enthalpy calls for mixture with perturbed composition.
The partial enthalpy calculation method is part of the Fluid properties model.

Additional user specified parameters

Selector

Used to create extra variables in the bed properties model, which can be assigned from the model interface. Choose from:

Off
On

Flow resistance

Pressure drop coefficient A

-

Adjustable factor (a multiplier) of the laminar flow term in the correlation for pressure drop.

Pressure drop coefficient B

-

Adjustable factor (a multiplier) of the turbulent flow term in the correlation for pressure drop.

Heat and mass transfer between pellet and bulk fluid

Solid-fluid transfer correlation

Selector

Correlation to be used for prediction of the mass and heat transfer coefficients in the fluid-solid interface. Choose from:

Hougen
Gnielinski
Constant
User defined
Mass transfer coefficient adjustable factor - Adjustable factor (a multiplier) for the mass transfer coefficient correlation. This factor can be tuned to generate model results that are compatible with the experimental data.
Heat transfer coefficient adjustable factor - Adjustable factor (a multiplier) for the heat transfer coefficient correlation. This factor can be tuned to generate model results that are compatible with the experimental data.
Mass transfer coefficient m s-1 Specify the mass transfer coefficient if the "Constant" option is selected for the solid-fluid transfer.
Heat transfer coefficient W m-2K-1 Specify the heat transfer coefficient if the "Constant" option is selected for the solid-fluid transfer.
Number of parameters [Integer] Number of user specified parameters per series.
Series [Ordered Set] List of series the user wants to define. The series “pressure drop” and “heat transfer” are provided as an example.
User specified parameters - Here the user can specify values for the created variables. These variables will be accessible in the bed properties model.
