Reactor conversion mass balance

The Reactor conversion mass balance model is used to model a reactor for which the conversion for each of the reactions is known. For each reaction, the conversion is defined with respect to a base component. The reactions may be assumed to occur either in parallel or in series. If the conversion of a reaction can depend on the inlet composition or conditions in the reactor, the use of a more detailed reactor model is required to accurately predict the reactor outlet composition.

Energy balances are not taken into account for this model. Hence, this model might be useful when a simplified process flowsheet, modelled exclusively by mass balances, is required.
