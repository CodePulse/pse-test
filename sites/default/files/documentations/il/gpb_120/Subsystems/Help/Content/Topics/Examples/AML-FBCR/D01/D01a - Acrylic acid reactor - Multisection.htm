D01a - Acrylic acid reactor - Multisection

In this example, the acrylic acid production process is modelled considering a multisection reactor with four different sections: two inert inlet and outlet sections and two sections with different catalyst packing.

The flowsheet models included represent the same reactor with different levels of accuracy regarding the catalyst bed side and the utility side. The catalyst bed is modelled using 1D and 2D beds (D01a_Reactor_1D_* and D01a_Reactor_*) and using lumped or distributed pellet models (D01a_Reactor_* and D01a_Reactor_with_Pellets_*). The utility side is modelled as an adiabatic reactor, with a simple fixed coolant model or even with an axially distributed 1D cooling jacket model.

The processes (D01a_Reactor_in_Fixed_Coolant_Initialisation and D01a_Reactor_with_Pellets_with_Jacket_Initialisation) can be used to initialise the models from poor initial guesses - the existing Saved Variable Sets will be updated if the option to auto-update is selected upon starting a simulation. The remainder processes use the saved variable sets to initialise the simulation.

The tasks provided are used when linking the AML:FBCR in gPROMS with an external CFD package.
