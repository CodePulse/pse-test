Design (Column geometry)

The Design (Column geometry) tab is used to define the calculation mode with respect to hydraulics and tray design, including geometry specifications. This tab can be turned off and on.

Name Default Unit Description
Design Selector
Off
In this mode the geometrical characteristics of the column and trays are not taken in consideration.
On
In this mode the geometrical specifications are either calculated from flooding correlations or provided by the user.
Internals Selector
Trays
Selects a range of available correlations applicable to trayed column.
Packing
Selects a range of available correlations applicable to packed column.
Packing type Selector
Structured
For crimped layers of wire mesh or corrugated sheets stacked in the column.
Random
For discrete pieces of packing of a specific geometrical shape randomly packed into the column shell.

Column diameter

 Selector
Calculated from flooding factor
Calculates the minimum column diameter necessary to avoid flooding.
Specified
The column diameter is specified directly by the user.

Calculate flooding

 Selector
Off
Turns off the flooding calculation.
On
Turns on the flooding calculation.

Column height

 Selector
Calculated from plate geometry
The column height is calculated using the assigned plate spacing and the number of column trays.
Specified
The column height, used to determine the required plate spacing, is specified.

Column diameter m Column diameter specification if the column diameter selector is set to Specified.
Allowed flooding factor - Maximum allowed flooding factor specification if the column diameter selector is set to Calculated and the flooding calculation is turned on.
Plate efficiency % Plate efficiency used for the calculation of the total column height.
Plate spacing m Interval between two consecutive trays.
Column height m Total height of the column.
Diameter of packing element mm The packing diameter is specified and used to determine the HETP of random packings. Larger packing diameters lead to an increase in the HETP values and reduced efficiency.
Surface area of packing element m2/m3 The surface area of the structured packing element is used to determine the column height of packed columns. Larger surface areas lead to lower HETP values and consequently to reduced column heights.
