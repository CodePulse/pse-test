Electrolyte system equilibriums

The electrolyte system model, also called the pH model, solves the electro-neutrality equation and determines the electrolyte distribution within the water.

This is done by computing a series of equilibrium relationships between the lumped components and the various electrolytes considered in the system. The input to the model is an array containing the concentrations of the lumped components. The same model can also be used to work at constant pH. In this case, spectator ions (named Z+ and Z-) will be determined and tracked throughout the system.

The gML Water library also allows users to define their own water chemistry model from a pre-defined databank. A simple Calcium / Carbonate / Phosphorus model is provided as a template. All the compounds currently available within the databank can be visualized within a case by running the process DatabankElectrolytes within the gML Water Auxiliary pH library.
