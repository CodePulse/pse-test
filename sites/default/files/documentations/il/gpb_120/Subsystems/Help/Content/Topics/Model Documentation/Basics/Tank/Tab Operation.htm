Operation

The Operation tab is used to specify Tank operation data. This tab is not available if the design mode is turned off.

Name Default Unit Description
Thermal specification Selector
Adiabatic operation
No heat exchange with surroundings (sets heat duty to 0).
Specified here
Uses the heat duty specified in this dialog.
From energy connection
The heat duty is obtained externally using a model connected to the gMLEnergy port.
Perfect control (no dynamics)
The heat duty is implicitly defined by assigning a thermal specification.

Temperature difference
Selects specification of the temperature difference between the outlet stream and the inlet stream.
Saturated liquid (Vapour fraction = 0)
When selecting this option, the vessel will consist of a saturated liquid (i.e. the vessel temperature is the bubble temperature of the mixture).
Saturated vapour (Vapour fraction = 1)
When selecting this option, the vessel will consist of a saturated vapour (i.e. the vessel temperature is the dew temperature of the mixture).
Subcooled liquid
When selecting this option, the vessel will consist of a subcooled liquid (i.e. outlet temperature is calculated using the provided degree of subcooling relative to the saturated liquid).
Superheated vapour
When selecting this option, the vessel will consist of a superheated vapour (i.e. outlet temperature is calculated using the provided degree of superheating relative to the saturated vapour).
