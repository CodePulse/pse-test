Adjust (Initial guesses)

The Adjust (Initial guesses) tab is used to specify adjusted variable types/values.

Name Default Unit Description

Adjusted variable type

 Selector
Scalar
The specified variable is a scalar.
Array
The specified variable is an array.
Set
The specified variable is a set.
Limit adjusted variable Selector
No
The variable to be adjusted is not limited.
Yes
The variable to be adjusted is limited between an upper and a lower bound. By providing these limits an additional option can be selected for the "Initialisation procedure" which ramps the adjusted variable.

Adjusted signal variables

 Selector Selects the variable from a drop-down menu with options given by the model connected to the bus port.

 

Adjusted variable initial guess

The units of the variables depend on the type of variable chosen (see 'Units of measurement in gMLBus ports').
