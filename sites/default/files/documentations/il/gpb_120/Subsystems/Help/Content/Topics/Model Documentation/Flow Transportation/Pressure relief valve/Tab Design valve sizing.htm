Design (valve sizing)

The Design (valve sizing) tab is used to specify valve sizing data.

Name Default Unit Description
Phase Selector
Liquid
Liquid fluid specification.
Vapour

Sizing variable Selector
Effective discharge area
The effective discharge area is determined for a given effective discharge coefficient.
Effective discharge coefficient
The effective discharge coefficient is determined for a given effective discharge area.
Rupture disk Selector
Off
No rupture disk is used.
On
The valve is sized assuming a rupture disk exists.
