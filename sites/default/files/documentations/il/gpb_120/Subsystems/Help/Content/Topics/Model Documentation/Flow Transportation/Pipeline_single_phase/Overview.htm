Pipeline single phase

The Pipeline single phase model describes the flow of fluid through a pipe. In this 1-D distributed model it is assumed that the fluid properties vary along the pipe. The relationship between the flowrate and the pressure drop is determined using friction factor correlations. For turbulent flow, a constant friction factor can be specified or the friction factor can be calculated from the Colebrook, Haaland or smooth tube correlations.
