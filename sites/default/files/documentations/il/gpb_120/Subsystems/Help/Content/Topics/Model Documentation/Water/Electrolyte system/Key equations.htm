Key equations

The electrolyte system model has two sets of components: The so-called compounds, which consists of all the real ionic species found in the solution; and the "ingredients", which are made up by compounds sharing a common constituent. For example, CaCO3 (aq), HCO3 (aq), or CO32-(aq) are compounds, and StCa (total soluble calcium) and StiC (total soluble inorganic carbon) are ingredients.

The compounds considered in the model can be chosen from a list by manually editing "Custom_sets_gML" and "Stoichiometry_Custom_gML", found for example in the "gML Water Chemistry Model Ca-iC-iP" models library.
