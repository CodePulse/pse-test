Description

This example illustrates the application of the AML:FBCR to acrylic acid production process. Different reactor configurations are considered for this application, specifically:

  1. Multisection reactor

    In this case the acrylic acid reactor model represents a multisection reactor with four different sections: two inert sections, inlet and outlet, and two catalyst sections with different catalyst packings.

    This example contains the following project files:

    • AML-FBCR D01 - Kinetics and Properties
    • AML-FBCR D01a - Acrylic acid reactor - Multisection

  2. Multitubular reactor

    In this case the acrylic acid reactor multisection model is applied in a multitubular configuration with three cooling compartment sections.

    This example contains the following project files:

    • AML-FBCR D01 - Kinetics and Properties
    • AML-FBCR D01b - Acrylic acid reactor - Multitubular

  3. Multisection annular reactor

    In this case the acrylic acid reactor multisection model is configured using axial annular sections in co-current and counter-current modes.

    This example contains the following project files:

    • AML-FBCR D01 - Kinetics and Properties
    • AML-FBCR D01c - Acrylic acid reactor - Multisection annular

Note the "Kinetics and Properties" project needs to be loaded in all examples as it contains information on the kinetics and the physical properties of the system.
