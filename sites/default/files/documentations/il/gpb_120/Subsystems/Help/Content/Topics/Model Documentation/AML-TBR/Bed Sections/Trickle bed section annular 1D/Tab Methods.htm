Methods

The Methods tab is used to specify modelling and correlations options.

Name Default Unit Description

Ergun

Custom

Specchia and Baldi, 1977
Liquid holdup is predicted from a predefined correlation.
Satterfield, 1969

Custom
Liquid holdup is predicted from a user defined correlation.
External wetting efficiency Selector
Off
Particles are completely wet.
On
Particle external surface wetting can be incomplete. Internal particle pores are completely filled with liquid.
On - neglecting the liquid stagnant zone
Use the particle external wetting model without liquid stagnant pockets.
