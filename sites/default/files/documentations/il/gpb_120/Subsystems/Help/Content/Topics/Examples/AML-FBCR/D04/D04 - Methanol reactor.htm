D04 - Methanol reactor

In this example, the methanol production process is modelled considering a radial inward flow reactor with internal cooling.

Two different reactor models are configured to either neglect the pressure drop in the inlet shaft (uniform inlet mode) or to explicitly account for it.

The processes (D04_Radial_Flow_Methanol_Reactor_Initialisation and D04_Radial_Flow_Methanol_Reactor_uniform_inlet_Initialisation) can be used to initialise the models from poor initial guesses - the existing Saved Variable Sets will be updated if the option to auto-update is selected upon starting a simulation. The remainder processes use the saved variable sets to initialise the simulation.

This project also includes the models with the kinetics for the reactor model configuration considered: 1D distributed pellet model in radial bed model. It also includes the physical property package configuration file for the main process stream.
