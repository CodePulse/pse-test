Main

The Main tab is used to specify how the inlet stream is split over each outlet stream.

Name Default Unit Description
Stream specification Selector
All streams except one (last stream)
All streams except the last stream must be specified. The number of entry fields that will be shown is equal to the number of streams minus one.
All streams (leave one stream specification blank)
All streams except one must be specified (the stream left blank is not necessarily the last one). The number of entry fields that will be shown is equal to the number of streams. One entry field needs to be left blank. The entry field that is left blank may correspond to a different component for each stream.
N.B. Entering values in all fields will result in an overspecified problem.

Outlet specification

 Selector
Split fractions
Specify the split fraction for each stream and each component. Using this option may be more appropriate if the inlet flowrate is subject to change.
Component flowrates
Directly specify the component flowrate for all but one outlet stream.
Total flowrates and composition
Directly specify the composition and flowrate of all but one stream. Note that if the stream specification option is selected where one stream is left blank, then the flowrate and the complete composition vector for that stream needs to be left blank.
Component flowrate basis Selector
Mass
Mass basis is selected for flowrate specification.
Molar
Molar basis is selected for flowrate specification.
Split fractions - The split fraction is a value between 0 and 1 which determines the proportion of the total inlet flow that will initially be split into each outlet.
Component outlet mass flowrates kg/s Component mass flowrates in each outlet.
Component outlet molar flowrates kg/s Component molar flowrates in each outlet.
Total outlet mass flowrate kg/s Mass flowrates for each outlet.
Outlet mass fraction kg/kg Mass fractions for each outlet.
Total outlet molar flowrate kmol/h Molar flowrates for each outlet.
Outlet molar fraction mol/mol Molar fractions for each outlet.
