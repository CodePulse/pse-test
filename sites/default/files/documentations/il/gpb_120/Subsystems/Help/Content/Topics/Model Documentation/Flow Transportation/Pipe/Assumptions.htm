Assumptions

The Pipe model uses the following assumptions:

A1

Infinitely fast flow dynamics.

A2

Constant cross sectional area.

A3

The flow in the pipe is adiabatic.

A4

The pipe is always full of fluid.

A5

The physical properties can be averaged over the pipe length.

A6

No phase change along the pipe length.
