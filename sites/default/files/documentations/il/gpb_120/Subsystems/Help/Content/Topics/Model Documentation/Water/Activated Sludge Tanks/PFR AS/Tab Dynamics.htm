Dynamics

The Dynamics tab defines options related to dynamic simulation.

Name Default Unit Description
Initial conditions specifications Selector
Steady-state
The model is initially at steady-state.
Startup
All components concentrations are initially equal to the inlet.
Gas flow mode Selector
Flow-driven
Simulation mode, where the gas flowrates are specified at the sources or in the splitter units.
It is not necessary to define flow-pressure relationships for flow-transportation equipment.
Pressure-driven (Flow-driven initialisation)
Simulation mode, where the user specifies pressures at the gas sources and flowrates are calculated based on these pressures on the flow-pressure relationship in the unit operations.
This mode requires the definition of flow-pressure relationships for all flow-transportation equipment (i.e. valves, pipes, compressors, etc. should be set to Performance Mode).
A flow-driven simulation is performed during initialisation to facilitate the initialisation of the pressure-driven mode.
Pressure-driven (no initialisation)
Simulation mode, where the user specifies pressures at the gas sources and flowrates are calculated based on these pressures on the flow-pressure relationship in the unit operations.
This mode should be used with no initialisation procedure (for instance to hot-start from a previous simulation).
