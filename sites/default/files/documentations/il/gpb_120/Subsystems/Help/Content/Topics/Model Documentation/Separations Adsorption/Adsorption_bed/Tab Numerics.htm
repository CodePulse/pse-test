Numerics

The Numerics tab is used to specify parameters related to how the equations of the model are solved.

Name Default Unit Description
Discretisation scheme Selector
Finite volume
Use a finite volume scheme to discretise the bed in the axial flow direction.
Finite difference
Use a finite difference scheme to discretise the bed in the axial flow direction.
Flow mode Selector
Reversible
Solve the mass and energy balance equations in the bed in a way that is able to handle flow-reversal.
Unidirectional
Solve the mass and energy balance equations in the bed in a way that is optimal for problem with unidirectional flow (i.e. problems where the flow does not reverse). This option is faster and more robust for cases where flow-reversal does not occur.
Physical properties Selector
Standard
Physical properties of the fluid in the bed are predicted using the standard physical properties package.
Custom
Custom effective physical properties correlations. Use the custom model in gML Templates to code the custom correlation using the gPROMS ProcessBuilder Custom Modelling feature.

Number of discretisation points per layer

 
For each layer, enter the number of discretisation points here.
Minimal superficial gas velocity threshold for correlations   The minimal value of the gas velocity used for correlations that have gas velocity as an input.
Dispersion/convection ratio for central difference scheme - The ratio between the convective and dispersive flux at which the discretisation method is switched from an upwinding scheme to a central difference scheme. Changing this parameter may affect numerical stability and computational speed of the simulation.
Small velocity to switch to central difference scheme m/s The velocity at which the discretisation scheme for the bed is switched from an upwinding scheme to a central difference scheme. Changing this parameter may affect numerical stability and computational speed of the simulation.
