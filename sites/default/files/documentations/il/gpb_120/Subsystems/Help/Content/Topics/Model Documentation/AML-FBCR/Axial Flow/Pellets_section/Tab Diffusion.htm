Diffusion

The diffusion model and the diffusion parameters are specified in the Diffusion tab.

Name Default Unit Description
Phase Selector
Gas
Select this option if the fluid is a gas.
Liquid
Select this option if the fluid is a liquid.
Diffusivity calculation Selector
From physical property package
Binary diffusion coefficients are determined by the physical property package.
From Fuller method [Gas]
Binary diffusion coefficients are calculated using the Fuller method.
Reference value [Liquid]
Binary diffusion coefficients are predicted from a single reference overall diffusion coefficient. The overall coefficient is specified by the user, at reference conditions.
Explicit specification
Binary diffusion coefficients are explicitly specified by the user, at reference conditions.

Gas phase
Bulk component - Most abundant component in the system. Used to get a single Schmidt (Sc) number, in mass balances and in diffusion equations.
Reference pressure bar Reference pressure used in diffusion coefficient dependency.
Reference temperature K Reference temperature used in diffusion coefficient dependency.
Reference mass fraction kg/kg Representative composition for calculation of the mean diffusion coefficient to obtain a single Schmidt number, i.e., a single particle-fluid boundary layer thickness.
Diffusion volumes - Input for the Fuller method for gas phase diffusivity calculation.
Binary diffusion coefficient m2/s Binary diffusion coefficient at reference conditions.
Liquid phase
Bulk component - Most abundant component in the system.
Reference pressure bar Reference pressure for diffusivity calculations.
Reference temperature K Reference temperature for diffusivity calculations.
Reference mass fraction kg/kg Reference mass fraction used in the diffusion coefficient call and to define the mixture binary diffusion coefficient.
Reference viscosity Pa s Reference liquid viscosity for diffusivity calculations.
Temperature power dependence - Exponent of temperature term for diffusivity calculations.
Viscosity power dependence - Exponent of viscosity term for diffusivity calculations.
Binary diffusion coefficient m2/s Binary diffusion coefficient at reference conditions.
