Internals

The Internals tab is used to select packing type and correlations related to mass transfer.

Name Default Unit Description
Packing type Selector
Mellapak 250Y
Selects packing related parameters specific for Mellapak 250Y.
Ceramic Berl Saddle
Selects packing related parameters specific for Ceramic Berl Saddle.
User defined
User has to specify packing related parameters.
Mass transfer interfacial area Selector
Random packing - Onda interfacial area correlation
Calculates interfacial area from Onda correlation.
Random or structured packing - Billet interfacial area correlation
Calculates interfacial area from Billet correlation.
Structured packing - Rocha interfacial area correlation
Calculates interfacial area from Rocha correlation.
User defined interfacial area correlation
Calculates interfacial area from user defined correlation in custom model.
Mass transfer coefficient correlation Selector
Random packing - Onda mass transfer correlation
Calculates mass transfer coefficients from Onda correlation.
Random or structured packing - Billet mass transfer correlation
Calculates mass transfer coefficients from Billet correlation.
Structured packing - Rocha mass transfer correlation
Calculates mass transfer coefficients from Rocha correlation.
User defined mass transfer correlation
Calculates mass transfer coefficients from user defined correlation in custom model.
Holdup correlation Selector
Random or structured packing - Billet holdup correlation
Calculates liquid holdup from Billet correlation.
Structured packing - Rocha holdup correlation
Calculates liquid holdup from Rocha correlation.
User defined holdup correlation
Calculates liquid holdup from user-defined correlation in custom model.
Pressure drop correlation Selector
No pressure drop
Assumes no pressure drop across column.
Random or structured packing - Flow coefficient pressure drop correlation
Calculates pressure drop from flow coefficient correlation.
Structured packing - Rocha pressure drop correlation
Calculates pressure drop from Rocha correlation.
Structured packing - Billet and Schultes pressure drop correlation
Calculates pressure drop from Billet and Schultes correlation.
User defined pressure drop correlation
Calculates pressure drop from user-defined correlation in custom model.
Mass transfer model Selector
Maxwell-Stefan
Uses Maxwell-Stefan mass transfer model.
Fick
Uses Fick mass transfer model.
