Initial guesses

The Initial guesses tab is used to specify data necessary for model initialisation. These are necessary only when the LMTD method is used for the prediction of the outlet streams conditions. Typically, the provided initial guess is the same as the specification used in the heat exchanger design.

Name Default Unit Description
Initial guess used for Selector
Unit initialisation only
When this option is selected, the heat exchanger uses the provided initial guess only for unit initialisation. After the unit is initialised, the heat exchanger starts predicting the outlet stream conditions, based on the specified heat transfer coefficient and heat transfer area, and it will continue in this mode during flowsheet initialisation.
Flowsheet initialisation (flow-driven mode)
When this option is selected, the heat exchanger uses the provided initial guess not only for unit initialisation, but also during flowsheet initialisation (during flow-driven mode). After the flowsheet is initialised, the heat exchanger switches to predicting the output streams conditions, based on the specified heat transfer coefficient and heat transfer area.

Start from no heat exchange
When this option is selected, the heat exchanger is initialised starting from the point where no heat is exchanged.
Hot stream specification
When this option is selected, a thermal specification for the hot outlet stream must be specified. This specification is used to determine the heat exchanged in the unit.
Cold stream specification
When this option is selected, a thermal specification for the cold outlet stream must be specified. This specification is used to determine the heat exchanged in the unit.

Saturated liquid (Vapour fraction = 0)
When selecting this option, the outlet stream will initially consist of a saturated liquid (i.e. the outlet stream temperature is the bubble temperature of the mixture).
Saturated vapour (Vapour fraction = 1)
When selecting this option, the outlet stream will initially consist of a saturated vapour (i.e. the outlet stream temperature is the dew temperature of the mixture).
Subcooled liquid
When selecting this option, the outlet stream will initially consist of a subcooled liquid (i.e. outlet temperature is calculated using the provided degree of subcooling relative to the saturated liquid).
Superheated vapour
When selecting this option, the outlet stream will initially consist of a superheated vapour (i.e. outlet temperature is calculated using the provided degree of superheating relative to the saturated vapour).

In co-current, it corresponds to the difference between the hot stream outlet temperature and the cold stream outlet temperature. In counter-current, it corresponds to the difference between the hot stream outlet temperature and the cold stream inlet temperature.

In co-current, it corresponds to the difference between the hot stream inlet temperature and the cold stream inlet temperature. In counter-current, it corresponds to the difference between the hot stream inlet temperature and the cold stream outlet temperature.
