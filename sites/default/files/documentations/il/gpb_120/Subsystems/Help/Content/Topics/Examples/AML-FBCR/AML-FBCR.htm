AML:FBCR

The gPROMS® Advanced Model Library for Fixed-Bed Catalytic Reactors (AML:FBCR) provides accurate predictive modelling for any reactor comprising fixed catalyst beds. The library is provided with a set of working examples which demonstrate the application of the tool to real life scenarios.

The library includes the following examples:

  • AML-FBCR D01 Acrylic acid reactor - multisection and multitubular

    Illustrates different (multisection, multitubular and annular) configurations of fixed-bed reactors for the production of acrylic acid.

  • AML-FBCR D02 Styrene reactor - radial flow

    Shows an application of different radial flow fixed-bed reactor configurations in adiabatic mode for styrene production.

  • AML-FBCR D03 Natural gas reforming - SMR and ATR

    Shows two applications of fixed-bed reactors in the natural gas reforming process for steam methane reforming (SMR) and autothermal reforming (ATR).

  • AML-FBCR D04 Methanol reactor - radial flow with cooling

    Radial flow configurations of fixed-bed reactors with internal cooling are used in the production of methanol.

  • AML-FBCR D05 VAM reactor - boiling water cooling

    Fixed-bed reactors are modelled together with a detailed model of the coolant side using boiling water, in both co- and counter-current configurations, for vinyl acetate monomer production.

This document provides background information about each example and the models used in each case.
