Steps

The Steps tab is used to define the steps in the schedule. For each step the variables in the connected units are set to a specified set of values.

Name Default Unit Description
Steps -
Add a number of steps by defining a label for each step.

Control setting

Selector

Provide a value for each variable in the connected units for each step. This is the value that will be applied to the unit at each step. Note that the units of measurement for the variables are the internal units for each variable type and might not correspond to what is selected in the dialog of each connected unit (see 'Units of measurement in gMLBus ports').
