Column

The Column tab is used to specify the number of stages, the position of the feed(s) and of the eventual side-draws.

Name Default Unit Description
Reboiler type Selector
Kettle
A kettle reboiler is used.
Thermosyphon
A thermosyphon reboiler is used.

Condenser type

 Selector
Total
All vapour from the top stage of the column is totally condensed. There is one liquid product stream only at the top of the column and no vapour product stream.
Partial
Vapour from the top stage of the column is partially condensed, so there can be two distillates: one vapour and one liquid.
Number of stages -
Number of stages in the column.

Feed stage(s)

 -
Position of the feed(s).

Feed introduction

 Selector
Vapour on stage
The feed is introduced on the specified stage.
Vapour above stage
The vapour part of the feed is introduced on the stage above the specified one, while the liquid part is introduced on the specified stage.
Liquid side draw position - Position of the liquid side-draw(s).

Vapour side draw position

 -
Position of the vapour side-draw(s).
