Dynamic mass balance

To be able to calculate a dynamic mass balance over time, the total amount of material that has flowed out of each Source material model (or through each Stream analyser) and into each Sink material model (or through each Stream analyser) as well as the material holdups in each unit equipment have to be calculated. For the Source material and Sink material the relations used to determine the mass and energy accumulation can be found in 'Key equations' and 'Key equations'.
