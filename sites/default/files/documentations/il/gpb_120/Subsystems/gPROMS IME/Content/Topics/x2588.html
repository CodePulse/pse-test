Project and entity Compare

gPROMS ProcessBuilder provides a comparison tool that can be used to compare any two selected Entities or Projects. The tool can be accessed via gPROMS ProcessBuilder's Tools menu, provided two comparable entries (e.g. Entities or Projects or Entity groups) have been selected in gPROMS ProcessBuilder's main navigation tree.
