Export to previous versions

The current gPROMS ProcessBuilder uses a different file format for storing its projects to that used by previous versions. In the interests of forward compatibility, projects may be exported to the previous format using the the Export ... tool in the Tools menu. Note that not all newer gPROMS concepts are recognised by those earlier versions of gPROMS ProcessBuilder and that the exported project will load but may require manual work to make it fully functional.
