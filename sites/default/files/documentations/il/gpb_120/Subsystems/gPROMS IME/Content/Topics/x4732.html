Adding a surface to a plot

Adding a surface to a 3D Plot is achieved in the same way as adding a line to a 2D Plot. Only one 3D surface can be plotted at a time. If the plot already contains a surface, then the Surface -> Add... menu-item is renamed Surface -> Change...; otherwise, the behaviour is the same.
