Library contents

The contents of each library are tabulated below. Also indicated in the table is the model class (Node, Connector, Hybrid or Control) and nature of the models equations (algebraic: steady-state or differential-algebraic: dynamic) for each model.

Note that all the models make use of the same set of Variable and Connection Types. These are stored in the PML Basics library. Hence, all the other PML library projects cross-reference the Basics library.

Please note that each of the libraries comes with its own set of manuals which are available as PDF documents from the Process Systems Enterprise group in Windows start menu and in the doc sub-directory of the gPROMS installation.
