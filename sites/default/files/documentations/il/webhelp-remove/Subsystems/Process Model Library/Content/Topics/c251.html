Introduction

The gPROMS Process Model Library (PML) is a collection of commonly used process equipment models. The library is designed to enable the user to quickly construct process flowsheet models that are suitable for use in steady state and dynamic simulation, parameter estimation and optimisation studies. The PML is open source and therefore also serves as a good starting point and reference of best modeling practice for the user's custom model development. The models have fully integrated support for a variety of rigorous physical property packages such as Multiflash and other Cape-Open compliant packages.

The library models employ an accurate representation of pressure-flow dynamics within the flowsheet and where appropriate flow reversal phenomena are modelled. Rigorous modelling of liquid and vapour mass and energy hold-up, phase appearance and disappearance and equilibrium and kinetic reactions are all handled, if appropriate for the given piece of equipment.

The gPROMS Process Model Library (PML) consists of six complementary libraries:

  1. PML Basics Library

  2. PML Control Library

  3. PML Flow Transportation

  4. PML Heat Exchange Library

  5. PML Reaction Library

  6. PML Separation Library

Each library is stored as a separate gPROMS project library file and has its own documentation. Both the library project files and the documentation can be found in the gPROMS installation. You can open the libraries and some example flowsheets from the gPROMS tool bar:

Once you left click on the Load Libraries button, the following dialog will appear.

Here, you can select which version of the Process Model Library to load. Left click on the name of a Library in the Library tab to see a description of it in the Information, tab on the right. Once you have decided which version to load, check the box next to it and press OK.

Note that other libraries may appear in the list and you may be able to load more than one library at the same time: simply check the box of each library you want to load.
