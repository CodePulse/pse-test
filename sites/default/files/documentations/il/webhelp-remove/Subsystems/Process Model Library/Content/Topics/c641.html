PML Model specifications

For each PML Model it is possible to make all necessary specifications quickly and easily from its Specification dialog. The Model's Specification dialog can be opened by double clicking on the Model icon on the topology tab of a flowsheet model (see building flowsheet models using the PML).

For many models more than one physically correct set of specifications is valid. To facilitate this, different specification modes are provided. For example, the PID Controller model (in the PML Control Library) can be specified in P, PI, PID mode and all models that consider hold-up support multiple initial condition modes.

All specifications made from Specification dialogs are represented in the gPROMS language of associated Process entities; and advanced users may want to edit the specifications directly in the gPROMS language. To do this once you have created a Process Entity for a particular flowsheet model from the Entity menu choose Include Unit dialog specifications which will un-check this option.
