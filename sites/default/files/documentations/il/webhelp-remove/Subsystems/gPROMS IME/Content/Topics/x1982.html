Interacting with executing simulations

The execution of an activity is initiated by the user pressing the OK button on the execution control dialog. At this point the gPROMS ModelBuilder creates the Case. It also requests and obtains the license necessary for execution; this license is associated with the Case until it is released either automatically at the end of the execution or manually at a later stage if the user has disabled this automatic license release mechanism. The fact that the Case is holding ("tying down") a license is indicated by a "clock" symbol attached to the Case's name in the gPROMS ModelBuilder's navigation tree. The symbol automatically disappears as soon as the license is released.
