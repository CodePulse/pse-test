To execute a simulation

Select the Process1Alternatively a model may be selected to run the simulation - this will execute a process with the same name; if none exists, it will be created. to be executed on the project tree and any of the following procedures can be used to start the simulation:

  • navigate to the Activities drop down menu and select Simulate (keyboard shortcut - F5).

  • press the green simulate arrow on the toolbar pane running across the top of the main gPROMS ModelBuilder window.

  • Right click on the process and select Simulate from the sub-menu.
