Exporting images

Graphical images of plots can be exported from gRMS for inclusion in documents and presentations.

Windows

Select the Graph -> Export Image... menu-item. This displays a standard Windows file dialog for you to specify a file name and image type from the following:

  • Enhanced Metafile (emf)

  • Aldus Placeable Windows Metafile (wmf)

  • Windows Bitmap (bmp)

  • Standard PNG (png)

  • Interlaced PNG (png)

  • JPEG (jpg)

Linux

Select the Graph -> Print... menu-item and use the Printer Dialog to select output to a file.
