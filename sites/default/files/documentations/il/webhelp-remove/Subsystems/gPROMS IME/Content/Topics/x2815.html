Additional files

If the gPROMS model requires additional files such as a .gSTORE file for initialisation or data files for foreign objects then they can be specified using the 'Files' page of the export wizard.

Export CAPE-OPEN Unit Operation Wizard: Additional files page

  • The contents of the list is restricted to those files present in the project containing the gPROMS model being exported. By default all files are marked to be included except those with .gCO, .mfl and .ipp extensions.
