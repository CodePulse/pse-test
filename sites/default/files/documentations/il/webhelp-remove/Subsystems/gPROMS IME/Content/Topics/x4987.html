Common templates

Plot templates saved in the 'oc' directory of the gPROMS installation directory (as identified by the value of the GPROMSHOME environment variable) are known as common templates. These appear in the Graph -> Open Common Template menu for easy access. If you have common templates called plot2d.gpt and plot3d.gpt then gRMS uses these when you select the Graph -> New 2D Plot and Graph -> New 3D Plot menu-items. Usually you would create these templates from plots with no lines or surfaces.
