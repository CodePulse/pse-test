Site custom constraints

The Site custom constraints model can be used to represent customised site constraints such as spinning reserve. There are no direct connections to this model: type the names of each custom constraint to be added and introduce the equations in the 'gPROMS language' tab. Custom model library management must be enabled in order to configure the gPROMS language. The model icon, as it appears on the Palette, is shown below.
