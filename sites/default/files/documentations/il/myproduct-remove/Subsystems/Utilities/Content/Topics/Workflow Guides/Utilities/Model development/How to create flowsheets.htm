Creating flowsheets

To begin creating a flowsheet, open gML Utilities and start a new Project. Add a new Model entity by right-clicking on the Project and selecting 'New entity...'.

Opening gML Utilities

Create a flowsheet by dragging units from the Palette onto the 'Topology' tab of your Model entity. Most units require connections to provide information on inlet streams from upstream units, and can also pass information to downstream units. Required connections will be displayed in the “Description” panel. Connect two units by clicking sequentially on the two ports you want to connect.

Model specifications can be introduced by double-clicking on the units in your flowsheet and changing the default values. Specifications can by turned on or off by checking & unchecking the box. This is not usually necessary as the models are set up to specify the required number of variables by default.

Load the example Project to see how the flowsheet can be put together.
