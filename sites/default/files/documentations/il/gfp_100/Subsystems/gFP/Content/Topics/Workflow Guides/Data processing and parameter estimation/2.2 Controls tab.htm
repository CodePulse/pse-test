Controls tab

An experiment control is a variable that is adjusted from one experiment to another (e.g. drying gas temperature) and/or during an experiment (e.g. the flowrate of fluidising gas in a fluidised bed dryer).

Experiment controls thus enable the description of the recipes (batch and continuous) and operating points (continuous, steady-state) used to conduct an experiment.  The user can specify the variation in a variable value using one of three different mechanisms:

  • Time-invariant controls
  • Provide a single variable value that will apply throughout the duration of the experiment. 
  • Appropriate for steady-state and dynamic experiments.
  • Piecewise constant controls
  • Provide a value per control interval1 A control interval corresponds to a phase of an experiment with a specified time duration..  This value will apply throughout the duration of the corresponding control interval.
  • Appropriate for dynamic experiments only.
  • Piecewise linear controls
  • Provide a start value and an end value per control interval.  The variable value will be varied linearly over time from the start value to the end value
  • Appropriate for dynamic experiments only.
Note

Any variables that are assigned values in a Process but are not specified as controls in the experiment will retain their assigned values.
