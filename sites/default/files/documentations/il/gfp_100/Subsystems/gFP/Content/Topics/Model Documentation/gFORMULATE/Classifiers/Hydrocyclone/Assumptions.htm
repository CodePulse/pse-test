Assumptions

The model is developed under the following assumptions:

  • This is an instantaneous, steady state model (i.e. the residence time of material in this unit is assumed to be zero).
  • All material entering the unit is mixed instantaneously before undergoing classification.
  • The distribution of solid particles between the top and bottom stream as a function of size is calculated using one of several possible empirical correlations described in Key equations.
