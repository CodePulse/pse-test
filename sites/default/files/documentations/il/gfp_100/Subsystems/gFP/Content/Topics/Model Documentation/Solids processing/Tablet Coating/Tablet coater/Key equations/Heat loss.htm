Heat loss

There is heat loss due to the temperature difference between the unit and ambient environment.

A few options are provided to specify heat loss rate.

  • Calculated - HLF (Heat loss factor)
  • Calculated - UAΔT
  • UAΔT - dynamic wall temperature
  • Explicitly specify the heat loss rate
  • Perfectly insulated
