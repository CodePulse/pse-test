Classification functions

The classification function is the ratio of mass flow rate between the oversize stream and the inlet stream (for a given particle size). A few options are available for the classification function as described below.

Note: In the equations below, parameters to be specified or estimated are printed in blue.

Solid particles are distributed between the top and bottom stream according to the particle size distribution, hydrodynamic properties and operating conditions in the hydrocyclone. The efficiency of solid partition can be modelled by the classification function as described below, using any of the three kernels discussed by Frachon and Cilliers (1999) 1Frachon, M. and Cilliers, J.J., (1999) A general model for hydrocyclone partition curves. Chem. Eng. J., 73, 53-59..
