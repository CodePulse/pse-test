Analysis of results

Upon completion of a parameter estimation run, a “Results” folder will be created within the parameter estimation case file, as shown below:

 

Project tree view of a completed parameter estimation case file.

 

This “Results” folder contains the following files (<case name> = “small_scale_FBD” in the case shown above).

  • gradient.txt – provides objective function gradients for all the estimated parameters
  • <case name>.out – raw text file with all parameter estimation results, used to generate the parameter estimation report
  • <case name>.params – provides the optimal final value and the specified lower and upper bounds for all the estimated parameters
  • <case name>.stat-mr – a macro-enabled MS Excel export file, which allows the automatic creation of an MS Excel workbook containing all the parameter estimation results
  • <case name> - full formatted parameter estimation report

 

On the top of the parameter estimation report, the final optimisation status and the final value of the objective function (“ln(likelihood function)”) will be given. The objective function employed for parameter estimation in gPROMS FormulatedProducts will be described in more detail in 'Appendix A – Maximum likelihood objective function '. The key sections of a typical parameter estimation report and the “Measurements” tab will be briefly described in the coming sections.
