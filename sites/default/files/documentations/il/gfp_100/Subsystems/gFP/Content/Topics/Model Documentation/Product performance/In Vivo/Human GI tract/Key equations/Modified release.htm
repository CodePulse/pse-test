Modified release

As composite phases disintegrate, the materials contained within them emerge. These phases are referred to as nested phases. The nested phases are born at a rate that is governed by the mass fractions of each nested phase within the composite. The contents of the composite phase are carried from upstream processes. Additionally, the size distribution, composition, porosity, and shape factors of these nested phases are captured based on their values when the composite phase was formed. As the composite phase disintegrates, it is assumed that the nested phases emerge with properties based on these variables.

One empirical rate expression is provided to describe the rate of disintegration of the composite phase, and a template is also provided so that the user can specify an alternate custom rate expression.
