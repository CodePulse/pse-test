Prediction of tablet porosity

In the tablet press, one or more phases are compacted into a tablet. Compacted phases can be primary particle phases and/or granulate phase. Granulate phases are composed of granule primary phases. The subscripts c, p, up, g, and gp represent compacted phases, all primary phases, ungranulated primary phases, granulate phases and primary phases within granules, respectively.

It is assumed that each primary phase within the tablet forms its own porosity under the compaction pressure. The porosity is only related to the compaction pressure and the phase properties, which means the porosity is not affected by other primary phases.
