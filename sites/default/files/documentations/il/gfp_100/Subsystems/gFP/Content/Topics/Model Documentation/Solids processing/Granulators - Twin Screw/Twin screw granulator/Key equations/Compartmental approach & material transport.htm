Compartmental approach and material transport

A compartmental approach is used to evaluate material transport along the granulator, where each axial compartment is assigned a residence time based on the type of screw element it contains. The residence time for each screw element type is specified by the user. The outlet flow rate of each compartment is given by Equation (1).

It is assumed that material only flows in one direction along the granulator. The inlet flow rate of a compartment is equal to the outlet flow rate of the previous compartment, except for the first compartment, where the inlet flow rate is defined by the granulator inlet port. The mass flow rates for the continuous liquid phase are evaluated using the same approach. This model assumes that axial transport does not depend on the phase or the particle size.
