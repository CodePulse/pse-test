Getting started

The simplest application of GSA is a scenario or parametric analysis. It is the most straightforward to set up and easier to interpret. It is recommended that new users start with a parametric or scenario analysis.

In contrast, uncertainty analysis is more complex because it requires correct specification of the probability distributions and factor spaces to obtain desired results.

Sensitivity analysis, in addition to the aforementioned points, requires an understanding of how the specification of the factor space and distribution impact the outcome. Furthermore, it requires an understanding of exactly how the sensitivities are defined, since these are different from mathematical sensitivities (partial derivatives) and also different for each of the two techniques considered in GSA. This documentation covers all types of analysis and contains guides to set up and make the most of each type.
