Case 6: Model uncertainty

Reaction mechanisms are not always known beforehand. Consequently, there may be more than one feasible expression for the reaction rate and it may be difficult, or even impossible to determine which is the most appropriate. GSA can be used to study the effect of such model uncertainties on KPIs.

Experimental data was gathered to estimate the activation energy and pre-exponential coefficient for both assumptions (see 'Parameter Estimation in gPROMS' and the parameter estimation entity 'Par_set_2' in the GSA example Project for details). Note that when executing a parameter estimation for the second assumption, 'Par_set_2', the value for model parameter ReacID must be set to 2.

Create a GSA entity named 'Case6_ModelUncertainty'. The settings for the General tab are shown in the figure below. In this example, uncertainty analysis is performed with the quasi-random Sobol sample generation method.

Case 6: General tab

Specify the parameters of interest as factors in the Factors tab. Either use the 'Select...' button or enter the full variable path into the 'Variable in model' column, pressing Control+Space to use the autocompletion feature.

Select each factor and click on the 'Edit...' button to specify all other factor settings. Note that, since only variables are eligible as factors, all parameters of interest must be moved from the PARAMETER section to the VARIABLE section of the model before the GSA entity is set up.

In this example, since experimental data is used to estimate the parameters for both assumptions, specify 'Normal distribution' for both activation energy and pre-exponential coefficients. In contrast, parameter ReacID can only assume values 1 and 2, to represent the first and second assumptions, respectively. Therefore, specify its distribution as 'Enumerated set' with the two scenarios shown in the figure below.

Case 6: Editing factors

As there are no sampling restrictions in this example, leave section 'Sampling restrictions' blank.

Specify the number of uncertainty scenarios in section 'Number of samples'. For uncertainty analysis, the total number of realisations of the base activity to be calculated is given by:

Notation

  Nd Number of deterministic cases
  N Number of uncertainty scenarios (user defined)

Case 6: Factors tab

Specify the responses in the Responses tab. Similar to factors, either use the Select button or enter the full variable path directly into the 'Variable in model' column, pressing Control+Space to use autocompletion.

In this example, the KPIs selected as responses are yield, molar concentration of methanol, and temperature.

Case 6: Responses tab

The 'GSA language' tab is automatically updated once tabs General, Factors and Responses are set up. Launch the execution by right-clicking on the GSA entity and selecting 'Analyse system' or pressing icon in the toolbar. 

Case 6: Executing GSA

The 'Analyse System' dialog will be displayed. Change the Case name and select the desired number of workers for parallel computation. Note that this number should not exceed the total number of logical cores on the machine.

Once execution completes successfully, right-click on the 'System Analysis Group' folder and select 'New result element...' to create result elements. Alternatively, copy and paste the result elements from other Cases. Note that, although it is possible to create a 'Factor Sensitivity Table' in the context of uncertainty analysis, this result element is not applicable and will not be populated with results within a Case.

Case 6: Creating a result element

The 'Scatter 2d' result element can be used to illustrate the relationship between any two factors, responses or a factor and a response. Select which variables to show on the X and Y axes via the Attributes, shown in the figure below.

By plotting yield in the Y-axis against inlet molar fraction and ReacID in the X-axis, it is possible to observe the differences between the two assumptions. According to the first assumption, the reaction is of second order and the reaction rate depends on the concentration of both reactants. Since the summation of all inlet molar fractions is one, when the inlet molar fraction of methanol varies from 0.01 to 0.5, the molar fractions of the other reactants also change. However, according to the second assumption, the reaction is of first order and depends only on the concentration of methanol.
