gPROMS FormulatedProducts libraries

gPROMS FormulatedProducts consists of four libraries covering shared models such as sensors and controllers, active ingredient manufacture, solids processing and product manufacture and product performance. All of these libraries are interoperable with one another and, as such, maybe used on the same flowsheet. Within each of these libraries is a number of sub-modules containing the model libraries. These sub-modules are described in more detail in 'Model libraries'.

gFORMULATE 1.0 Libraries

The gFORMULATE libraries contain the building blocks of the model libraries. They also contain the shared models that may be used at many points in the library such as sensors, controllers and analysers. There are also some basic unit operation models such as a tank, agglomerator and fan.

The most commonly used models in this library are the sources and sinks as these define where material enters and leaves the process.

All the other libraries depend on the gFORMULATE libraries and as such they should always be open.

gFORMULATE 1.0 - Synthesis & Fluid Separation

gFORMULATE Synthesis & Fluid Separation Libraries contain the reactor models (stirred tank and plug flow) as well as flash drum models for two-phase and three-phase flash calculations.

These libraries are new to gPROMS FormulatedProducts.

gFORMULATE 1.0 - gCRYSTAL

gFORMULATE 1.0 gCRYSTAL modules contain models for crystallization processes including MSMPR crystallizer, wet mills and a plug flow crystallizer model. This library contains the ported models from the legacy gCRYSTAL libraries as well as the new models described previously.

gFORMULATE 1.0 - gSOLIDS

Models for solids processing are found in the gSOLIDS library. Different routes of drug product manufacture can be described using these models. There are models for roller compaction, continuous blenders for direct compression, spray drying, wet granulation and tabletting.

gFORMULATE 1.0 - gCOAS

gCOAS contains libraries for pharmacokinetics, in-vivo and in-vitro product performance as well as property tools to perform analyses such as speciation analysis. There are also supplementary in-vivo models that can be found within the gCOAS library folder (they will not open directly from the "Open/close libraries" dialog).
