gCOAS In Vivo

The gCOAS In Vivo library contains a model of the gastrointestinal tract that predicts oral absorption performance as a function of product formulation and drug substance properties, patient physiology and ingestion conditions. In addition to a number of interface and report enhancements, two new options have been added to gPROMS FormulatedProducts. Users may now utilise a dissolution profile for dissolution of primary particles rather than the default population balance approach. The dissolution profile may be input as a Weibull function or interpolated directly from raw data. When a modified release tablet dose is administered, the release of nested material (e.g. primary particles) from the composite material (e.g. granules) can be modelled via a modified release kinetic model.
