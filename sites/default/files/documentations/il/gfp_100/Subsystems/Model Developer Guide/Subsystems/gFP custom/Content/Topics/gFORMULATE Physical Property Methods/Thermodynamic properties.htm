The gFORMULATE properties package system supports external properties packages (for example, Multiflash). The available methods are shown below.

Fluid property approaches
Method name Input Type of output Dimension of output Description
FluidFOPerPhase VapourPhaseID(phase) or LiquidPhaseID(phase) Foreign object - Fluid property foreign object of a fluid phase
WhichFluidApproachPerPhase VapourPhaseID(phase) or LiquidPhaseID(phase) Ordered set - Fluid property approach that is used by a fluid phase, which returns "Multiflash" if Multiflash is used, or "gFORMULATE" if gFP property approach is used.
FluidPackageSpeciesPerPhase VapourPhaseID(phase) or LiquidPhaseID(phase) Ordered set - Fluid package species of a fluid phase, which are the species considered by the property approach in the fluid phase.

 

If you use Multiflash package, please refer to 'The MultiflashTM physical property interface' for Multiflash properties methods.
