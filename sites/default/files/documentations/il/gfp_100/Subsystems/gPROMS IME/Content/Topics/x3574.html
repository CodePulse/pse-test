Navigation shortcuts - Find shortcuts

Find shortcuts
Keyboard shortcut With no selected text With selected text
CTRL+f Show the Find dialog. Show the Find dialog and show selected text as the text to find.
F3 Search for the next occurrence of a search entered in the Find dialog or marked by CTRL+F3 (see below). The dialog must be closed for this to work.  
SHIFT+F3 Search for the previous occurrence of a search entered in the Find dialog or marked by CTRL+F3 (see below). The dialog must be closed for this to work.  
CTRL+F3 Search and highlight all occurrences of the word that the insertion point is on. The occurrences can be cycled through using F3 or SHIFT+F3. Search and highlight all occurrences of the selected text. The occurrences can be cycled through using F3 or SHIFT+F3.
ALT+SHIFT+h Switch highlight search on or off.  
CTRL+g Show Goto Line dialog.
