Accessible functionality when transitioning from legacy products

PSE has designed gPROMS FormulatedProducts (gFP 1.0) such that your licences for pre-gFP 1.0 products will allow you to:

  • Launch gFP 1.0 (the base product).
  • Open any gFP 1.0 libraries corresponding to libraries you had licences for
  • Use any platform functionality (simulation, parameter estimation, optimisation and custom modelling) for which you already had licenses.

You will find a list of legacy models and libraries as well as the corresponding models and libraries under the gFP 1.0 framework in these sections:

gCRYSTAL model migration

gSOLIDS model migration

gCOAS model migration

gFP 1.0 will contain a number of libraries for which you most likely do not have licences. These fall into two categories:

  • Libraries from other pre-gFP 1.0 products that you did not have licences for. E.g. you may have purchased gCRYSTAL licences but no gSOLIDS or gCOAS licences.
  • Libraries containing models that did not exist in any of the pre-gFP 1.0 products. The key models are:
    • Reactor models (CSTR and PFR) for single and multi phase systems
    • Fluid separation (VL and VLL flash drums)
    • Plug flow crystallizer and wet mill models
    • Continuous blender models
    • High shear wet granulator and twin screw granulator

For the libraries which are new in gFP 1.0, PSE is happy to offer free 3-month evaluation licences to start no later than 1 July 2017.

gFP 1.0 also contains new platform functionality for which you most likely do not have licences for, namely:

  • Global System Analysis (GSA)
  • High Performance Computing (HPC)

For both GSA and HPC, PSE is happy to offer free 3-month evaluation licences to start no later than 1 July 2017.
