Level controller

The level controller model is used to control the level of a unit operation, according to a specified control profile, by adjusting its inlet or outlet flow rates.

Key features:

  • P or PI control can be applied.
  • Level control profile is specified by the user.
  • The level measurement signal can be obtained from a level sensor by connecting their ports on a flowsheet.
