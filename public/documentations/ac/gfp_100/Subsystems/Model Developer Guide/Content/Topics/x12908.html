Explicitly reverting changes

INITIALISATION_PROCEDURE IP_Example1
  START
    aParameter := 0 ;
    aVariable := 0 ;
    REPLACE
      anotherVariable
    WITH
      aThirdVariable := 0 ;
    END
    aSelector := Simple ;
  END

  NEXT
    MOVE_TO
      REVERT aParameter ;
      REVERT aVariable ;
      REVERT anotherVariable ;
      REVERT aSelector ;
    END
  END
END

Here, the NEXT section reverts all actions simultaneously and smoothly.
