Material system information

The methods below provide the material system information, such as phases, species, reactions and solutions. They are used as arguments for calling physical properties.

Phases, components and species

Phases, components and species
Method name Input Type of output Dimension of output Description
AllPhases - Ordered set - All phases in the global system
AllPhaseID() - Integer AllPhases ID number of each phase
GasPhase - Ordered set - All vapour phases
GasPhaseID() - Integer GasPhase ID number of each vapour phase
LiquidPhases - Ordered set - All liquid phases
LiquidPhaseID() - Integer LiquidPhases ID number of each liquid phase
SolidPhases - Ordered set - All solid phases
SolidPhaseID() - Integer SolidPhases ID number of each solid phase
CompositePhases - Ordered set - All composite phases
CompositePhaseID() - Integer CompositePhases ID number of each composite phase
AllSpecies - Ordered set - All species in the global system
AllSpeciesID() - Integer AllSpecies ID number of each species
GasSpecies - Ordered set - All species in vapour phases
LiquidSpecies - Ordered set - All species in liquid phases
SolidSpecies - Ordered set - All species in solid phases
GasSpeciesTaggedName GasSpeciesID(species) Ordered set -

Component name of a vapour species.

Species names are appended with " (vapour)".
LiquidSpeciesTaggedName LiquidSpeciesID(species) Ordered set -

Component name of a liquid species.

Species names are appended with " (liquid)".
SolidSpeciesTaggedName SolidSpeciesID(species) Ordered set -

Component name of a solid species.

Species names are appended with " (solid)".
ComponentsForASpecies AllSpeciesID(species) Ordered set -

Component name of a species.

Species names are appended with available state.
SpeciesInThisPhase AllPhaseID(phase) Ordered set - All species in a phase

 

Crystals and salts

Crystals and salts
Method name Input Type of output Dimension of output Description
CrystalPhases - Ordered set - Solid phases that can dissolve in a liquid phase (soluble phases)
CrystalPhaseID() - Integer CrystalPhases ID number of each soluble phase
DissociatingSolidPhases - Ordered set - Soluble phases that can dissociate into multiple species in a liquid phase
SaltSolidPhases - Ordered set - Soluble phases that can dissociate into any ionic species
CrystalPhasesForThisLiquidPhase LiquidPhaseID(phase) Ordered set - Soluble phases that can dissolve in a liquid phase
LiquidPhasesForThisCrystalPhase CrystalPhasesID(phase) Ordered set - Liquid phases that a soluble phase can dissolve into
SolvateSpeciesForThisCrystalPhase CrystalPhasesID(phase) Ordered set - Solvate species of a soluble phase
CrystallisingSolute CrystalPhasesID(phase) Ordered set - Solute components that a soluble phase can dissociate into
CrystallisingSoluteSpecies CrystalPhasesID(phase) Ordered set - Solute species that a soluble phase can dissociate into
CrystalStoichiometricCoefficients CrystalPhasesID(phase) Integer LiquidSpecies Stoichiometry of a soluble phase dissolving in a liquid phase
ChargedLiquidSpecies - Ordered set - Charged liquid species (ionic species)
LiquidSpeciesCharges() - Integer LiquidSpecies Charge of each liquid species. The number is zero for neutral species.
