Model palette

The Model palette graphically displays the Model Entities contained in open Projects (and Library projects) and provides an alternative view to the Project tree. All Models that have an icon defined are shown on the Model palette1Models will only appear on the palette if the Show in palette option is checked on their Properties tabs.. The Models are grouped by the Projects to which they belong. The Model palette provides a graphical view of models that can be used to build flowsheets. Their context menu allows to directly preview specifications and reports, or to open their interface tab.
