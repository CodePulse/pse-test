Known issues, limitations and tips

Tips

  • Ensure that no foreign process interface (FPI) is active. Foreign Objects can be used.
  • Long initialisation procedures may slow down GSA execution. Saved variable sets should be used instead.

Limitations

  • Only the last point in a simulation is reported. Although steady-state analysis is more appropriate, there is a workaround for dynamic analysis.
  • No sampling restrictions or correlated factors for can be specified for sensitivity analysis. However, there is a workaround.
  • Although tags can be defined for factors and responses, they cannot be used to filter results at this time.

Known issues

  • It may be necessary to click elsewhere on the GSA editor after entering a value to trigger the validation of the text field.
  • Currently the covariance matrices of correlated factors cannot be used to produce the correct factor space ellipsoid. This will be fixed in a future release.
