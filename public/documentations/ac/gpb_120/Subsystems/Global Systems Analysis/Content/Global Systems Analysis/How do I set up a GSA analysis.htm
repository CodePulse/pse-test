Summary of the required steps to set up an analysis

To set up a GSA entity, reproduce the steps below.

  1. Load the Project which contains the model or flowsheet of interest.

  2. Create a new entity by selecting 'New entity...' in the 'Entity' menu, by pressing Control+C or by right-clicking on the Project and selecting 'New entity...'.

    Definition of a new Global system analysis entity

  3. Provide a name and description (optional) and click on OK.

  4. The GSA entity will be displayed in the 'Global System Analysis' folder.

  5. On the General tab, select the Process to use as 'Base activity for analysis' from the options in the drop down menu.

  6. Select the type of analysis to be performed, between sensitivity and uncertainty.

  7. If sensitivity analysis is selected, specify the method to be used for analysis as well.

  8. Select the sample generation method.

    General tab

  9. Note that the General tab will display an error message until one or more probabilistic factors are specified.

  10. On the Factors tab, specify the factors of interest by clicking on the 'Select...' button. If the factor is not selectable in this manner, enter the full variable path in 'Variable in model' field.

  11. Set appropriate lower and upper bounds for each factor by double-clicking on each field and editing the values. Alternatively, select each factor and click on the 'Edit...' button.

  12. Select an appropriate probability distribution and values for the parameters of the distribution, if required.

  13. Edit the name of any factors as desired. Note that a number of suggestions are available in the drop down menu of the Name field of each factor.

  14. Optionally, add one or more tags. Note that each tag must be enclosed in double quotes and that multiple tags can be associated with a factor when separated by a space or a comma. In future versions, tags will be used to filter results.

    Factors tab

  15. If uncertainty analysis is selected, it is also possible to:

    1. Add sampling restrictions by clicking on the 'Add...' button and specify the constraint in the 'Constraint expression' field.

      Sampling restrictions editor

    2. Create groups of correlated factors can by selecting multiple factors while holding down the Control key and clicking on the Group button. The correlation can be added by selecting the factor group and clicking on the 'Edit...' button.

      Factor group editor

      Note that, in the figure above, the correlation was added due to operating practice rather than a physical correlation between factors.

  16. Specify the number of samples. The number of discrete deterministic cases is calculated according to the factors which are gridded or enumerated in nature. In contrast, the number of uncertainty scenarios/probability distribution samples is a userdefined. The total number of realisations depends on these values as well as on the type and method of analysis to perform.

  17. Specify the responses of interest in the Responses tab. Similar to factors, either use the 'Select...' button or enter the full variable path into the 'Variable in model' field.

    Responses tab

  18. Optionally add any results elements to the GSA entity.

  19. Execute the GSA activity.
