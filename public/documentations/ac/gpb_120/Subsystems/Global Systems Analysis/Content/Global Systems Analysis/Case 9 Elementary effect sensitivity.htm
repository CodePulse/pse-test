Case 9: Elementary effects sensitivity analysis

There are two methods available for sensitivity analysis, elementary effects and variance-based. Elementary effects is a simple but effective way of identifying a few important factors from a large number of factors in a complex model. This method is ideal when the number of probabilistic factors is high and/or the model is expensive to compute. This example demonstrates how to set up an elementary effects sensitivity analysis and interpret the results.

Specify the responses in the Responses tab. Similar to factors, either use the 'Select...' button or enter the full variable path directly into the 'Variable in model' column, pressing Control+Space to use autocompletion.

Case 9: Responses tab

The 'GSA language' tab is automatically updated once the General, Factors and Responses tabs are set up. You can then launch execution by right-clicking on the GSA entity and selecting 'Analyse system' or pressing icon in the toolbar. 

Case 9: Executing GSA

The 'Analyse System' dialog will be displayed. Change the Case name and select the desired number of workers for parallel computation. Note that this number should not exceed the total number of logical cores on the machine.

Once execution completes successfully, right-click on the 'System Analysis Group' folder and select 'New result element...' to create result elements. Alternatively, copy and paste result elements from other Cases. Note that the 'Factor Sensitivity Table' is not applicable in the context of uncertainty analysis and will not be populated with results.

Case 9: Creating a new result element

The 'Factor Sensitivity Table' indicates that the effect of inlet molar fraction 1 on yield is much greater than that of any other factors. The total effects of coolant temperature, inlet molar flow rate and inlet molar fraction 1 on temperature are much greater than those of other factors.

Note that the factor order, when sorted by relevance, changes for different responses. For example, while the inlet molar flow rate has very little effect on yield, it is the second most important factor as far as temperature is concerned. In contrast, pressure has no effect on any of the responses and can be ignored in subsequent analyses, provided that the responses of interest are the same. These results are consistent with those obtained using the variance-based method.

Note that elementary effects require far fewer samples to achieve accurate results. Even using a single sample as the number of uncertainty samples (with a total number of realisations of base activity to be calculated of about 60), the most important factor is still correctly returned. In fact, when the number of uncertainty samples is 10, all factors are similarly sorted by relevance as when the number of uncertainty samples is 100.
