Editing using external editor software

Right clicking on the selected Saved Variable Sets file and subsequently selecting Edit using external program... will bring up an application chooser window for the user to select the .exe file of the desired editor. The file can then be modified in the chosen editor and saved again to be used by gPROMS (please note that the user needs to close the editor application in order to return to gPROMS ProcessBuilder).
