Exporting stream tables

The context menu of a stream table offers three options for data transfer with other applications such as spread sheets or word processing. The data transfer uses either the clipboard or intermediate files:

Copy

copies the selected range of cells to the clipboard in table format so that they can be directly pasted into separate cells.

Copy stream table

copies the entire table including the column and row headers in table format to the clipboard.

Export stream table ...

exports the entire table including extra information such as date and table name to a file in CSV format.
