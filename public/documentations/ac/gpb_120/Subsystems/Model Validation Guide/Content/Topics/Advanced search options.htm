Advanced search options

Match any character
? matches one character. * matches zero or more characters.
Typing M*flow will match Mass flow and Molar flow. Typing p??_c will match PID_controller but not P_constroller.
Match the start or the end
Prepend a search term with ^ to match only at the beginning of text. Append $ to match only at the end of text.
Typing ^in will match Inner pipe but not Fin. Typing pump$ will match Pump but not pump_simple
