Source material mass balance

The Source material mass balance model is used to create a starting point for a given process material stream (e.g. when a stream enters the process flowsheet). This model allows for definition of the stream's pressure, composition and flowrate. Optionally a physical property package can be defined which is used to obtain the components in the stream and the molecular weight of each component. If a physical property package is not defined the components in the stream and their associated molecular weight must be specified.

Energy balances are not taken into account for this model. Hence, this model might be useful when a simplified process flowsheet, modelled exclusively by mass balances, is required.
