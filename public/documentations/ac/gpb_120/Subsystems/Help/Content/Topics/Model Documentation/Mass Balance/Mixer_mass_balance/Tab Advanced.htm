Advanced

The Advanced tab is used to specify outlet stream pressure data.

Name Default Unit Description

Pressure specification

Selector
Minimum of inlet pressures
The outlet stream pressure is set equal to the pressure of the inlet stream with the lowest pressure.
Specified pressure
The outlet stream pressure is set by the user.
Pressure of a particular inlet stream
The outlet stream pressure is set equal to the pressure of an inlet stream specified by the user.
Outlet pressure equal to pressure of stream - Index of the inlet stream used to define the outlet pressure.
