Material adapter

The Material adapter model is used to change the physical property package of a material stream. This is achieved by defining a physical property package for the outlet material stream that is different to that of the inlet stream. As the physical property package also defines the components, this means that the components in the outlet stream may differ from those in the inlet stream. In that case, the components that are in the inlet stream and do not feature in the outlet stream will be routed through a side outlet port of the material adapter model. Similarly, components that are in the outlet stream that are not in the inlet stream can either be set to have a zero composition in the outlet stream or can be mixed in from a side inlet.
