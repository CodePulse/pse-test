Numerics

The Numerics tab is used to specify parameters relating to model initialisation and robustness.

Name Default Unit Description
Inner pipe outlet pressure drop Selector
Zero pressure drop
Assume that at the outlet of the inner pipe there is zero-pressure drop.
N.B. : If this option is chosen, tubes cannot be connected directly after each other in dynamic mode without a connector type model (e.g. a valve) in between.
Percentage of total pressure drop
Assume that at the outlet of the inner pipe there is a pressure drop that is a fixed percentage of the overall pressure drop in the tubes.
Outer pipe outlet pressure drop Selector
Zero pressure drop
Assume that at the outlet of the outer pipe there is zero-pressure drop.
N.B. : If this option is chosen, tubes cannot be connected directly after each other in dynamic mode without a connector type model (e.g. a valve) in between.
Percentage of total pressure drop
Assume that at the outlet of the outer pipe there is a pressure drop that is a fixed percentage of the overall pressure drop in the tubes.
Discretisation points distribution Selector
Uniformly distributed
The distribution domain has equally spaced points.
Exponential (more grid points towards start)
The distribution domain has a higher concentration of points towards the start. This option can be used if the profiles of the distributed variables are relatively steep towards the start and flat towards the end. This option may be less stable if the profiles change during a dynamic simulation.
Exponential (more grid points towards end)
The distribution domain has a higher concentration of points towards the end. This option can be used if the profiles of the distributed variables are relatively steep towards the end and flat towards the start. This option may be less stable if the profiles change during a dynamic simulation.
Exponential (more grid points towards both ends)
The distribution domain has a higher concentration of points towards both ends and a lower concentration of points in the middle. This option can be used if the profiles of the distributed variables are relatively steep towards both ends and flat in the middle. This option may be less stable if the profiles change during a dynamic simulation.
Temperature for heat transfer Selector
At cell downstream boundary (BDFM)
Temperature of the downstream boundary of each cell is used. This option is numerically stable and suitable for dynamic operation but can require a large number of grid points.
Linear interpolation between cell boundaries
The temperature used for heat transfer is obtained by linear interpolation of the cell inlet and outlet temperatures. This option is more accurate than using the downstream boundary temperature but may be less stable and result in oscillating temperature profiles.
Geometric interpolation between cell boundaries
The temperature used for heat transfer is obtained by geometrical interpolation of the cell inlet and outlet temperatures. This option is more accurate than using the downstream boundary temperature but may be less stable.
Inner pipe phase Selector
Multiphase
Use this option if multiple phases are expected to be present.
Vapour
Use this option if only vapour is expected to be present in the unit.
N.B. Use of this option might speed up calculations but results might be incorrect if the conditions are such that there are multiple phases or only a liquid phase.
Liquid
Use this option if only liquid is expected to be present in the unit.
N.B. Use of this option might speed up calculations but results may be incorrect if the conditions are such that there are multiple phases or only a vapour phase.
Outer pipe phase Selector
Multiphase
Use this option if multiple phases are expected to be present.
Vapour
Use this option if only vapour is expected to be present in the unit.
N.B. Use of this option might speed up calculations but results might be incorrect if the conditions are such that there are multiple phases or only a liquid phase.
Liquid
Use this option if only liquid is expected to be present in the unit.
N.B. Use of this option might speed up calculations but results may be incorrect if the conditions are such that there are multiple phases or only a vapour phase.
Initialisation procedure Selector
Automatic (Fast)
Automatic (Fast) is a unit initialisation procedure that is optimised for speed. This initialisation procedure may occasionally fail for some problems. If this happens, use the Automatic (Robust) option.
Automatic (Robust)
Automatic (Robust) is a unit initialisation procedure that is optimised for robustness. This initialisation procedure should work in all cases but is generally slower than the Automatic (Fast) option.
Manual
Allows selection of any one of the existing initialisation procedures. This option is recommended for advanced users only.
Off
Do not use an initialisation procedure for this unit. This option can be used when the unit is initialised from a Saved Variable Set (SVS).
Number of discretisation points - Number of discretisation points used in the axial distribution domain of the model.
Inner pipe typical flowrate kg/s Specifies the typical flowrate for this simulation. Under normal circumstances, this flowrate parameter does not need to be specified but for some simulations, with either very small or very large flowrates in (part of) the flowsheet, setting this parameter may improve the accuracy of the results.
Outer pipe typical flowrate kg/s Specifies the typical flowrate for this simulation. Under normal circumstances, this flowrate parameter does not need to be specified but for some simulations, with either very small or very large flowrates in (part of) the flowsheet, setting this parameter may improve the accuracy of the results.
Inner pipe typical energy rate kW Specifies the typical energy rate for this particular simulation. This should be set to a value equal to the order of magnitude of the typical energy rate for this unit in the process being simulated. Under normal circumstances, the typical energy rate parameter does not need to be specified but for some simulations with either very small or large energy rates, setting this parameter might improve the accuracy of the results.
Outer pipe typical energy rate kW Specifies the typical energy rate for this particular simulation. This should be set to a value equal to the order of magnitude of the typical energy rate for this unit in the process being simulated. Under normal circumstances, the typical energy rate parameter does not need to be specified but for some simulations with either very small or large energy rates, setting this parameter might improve the accuracy of the results.
Domain transformation parameter - Specifies the exponential transformation parameter. This parameter is used to calculate the grid points distribution. The higher the value, the higher the deviation from an equally-spaced grid.
