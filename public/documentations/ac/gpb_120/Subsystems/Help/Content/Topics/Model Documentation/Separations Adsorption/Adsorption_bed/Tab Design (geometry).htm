Design (geometry)

The Design (geometry) tab is used to specify basic geometrical design of the bed, such as the bed and layer sizing.

Name Default Unit Description
Bed configuration Selector
Flow in axial direction
Fluid flows in the axial direction of the cylindrical bed.
Flow in radial direction
Fluid flows in the radial direction of the cylindrical bed. This option is required to model radial-flow beds.

Bed size specification

Selector

Determine size from the bed diameter and the bed length.

Determine size from the bed volume and the bed length.
Bed cross-sectional area
Determine size from the bed cross-sectional area and the bed length.

Number of layers

 

Number of distinct layers in the bed model. Each layer can have different types of adsorbent material(s) and loading(s).
