Design (Shell)

The Design (Shell) tab is used to specify data relating to the shape and dimensions of the shell.

Name Default Unit Description
Shell model specification Selector
Bell-Delaware
Selects the use of the Bell-Delaware model. This rigorous model takes into account the different bypass streams that exist in a shell and calculates correction factors for both heat transfer and pressure drop.
Number of baffles specification Selector
Number of baffles
The total number of baffles is specified.
Baffle spacing in central zone
The total number of baffles is calculated using the specified distance between baffles in the central zone, as well as the baffle spacing at shell inlet and outlet, for the given shell length.

Baffle spacing specification

 Selector
Specified baffle spacing at inlet and outlet
Allows the specification of different baffle spacings at shell inlet and outlet.
Equal to specified baffle spacing for central zone
Selects the baffle spacing at both shell inlet and oulet to be equal to baffle spacing in the central zone.

Inner diameter

 m Inner shell diameter.

Baffle cut

 m Sets the baffle cut as a fraction of baffle diameter.

Tube-to-baffle diametrical clearance

 m Difference between the baffle tube hole radius and the tube radius.

Shell-to-baffle diametrical clearance

 m Difference between the shell radius and the baffle radius.
Number of sealing strips -

Number of sealing strips per baffle.

Number of pass divider lanes

 - Number of pass divider lanes. This number depends on the number of tube passes and the arrangement of the tube bundles for each pass.

Pass divider lane width

 m Width of the pass divider lane.
