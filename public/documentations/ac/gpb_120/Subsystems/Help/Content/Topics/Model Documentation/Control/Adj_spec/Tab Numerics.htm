Numerics

The Numerics tab is used to specify parameters related to model robustness.

Name Default Unit Description
Operation mode Selector
Initialisation procedure
An initialisation procedure is included to aid model initialisation.
Active
The target value imposed directly. An initialisation procedure is not used.
Inactive
The adjusted variable value is specified instead of being guessed and therefore the target value is not met. An initialisation procedure is not used.
Initialisation procedure Selector
Automatic (Fast)
Automatic (Fast) is a unit initialisation procedure that is optimised for speed. This initialisation procedure may occasionally fail for some problems. If this happens, use the Automatic (Robust) option.
Automatic (Robust)
Automatic (Robust) is a unit initialisation procedure that is optimised for robustness. This initialisation procedure should work in all cases but is generally slower than the Automatic (Fast) option.
Ramp (Robust)
Ramp (Robust) is a initialisation strategy that ramps the adjusted variable until either the target is reached or the adjusted variable hits its upper or lower limit. To use this produce the relation between the target variable and the adjusted variable and needs specified to be either Direct or Reverse. The advantage of this method over the Automatic (Robust) method is that it may succeed for models where the relation between adjusted and target variable is not present for all initial values of the adjusted variable.
Manual
Allows selection of any one of the existing initialisation procedures. This option is recommended for advanced users only.
Off
Do not use an initialisation procedure for this unit. This option can be used when the unit is initialised from a Saved Variable Set (SVS).
Adjuster action  
Reverse
By selecting this mode, the adjusted variable decreases as the variable that is to be adjusted to match the target increases.
Use this mode when the gain of the process is positive, for example when adjusting the heater outlet temperature using the heat duty. In this case, when the outlet temperature of the heater (the variable that is to be adjusted to match the target in this case) increases, the heat duty of the heater (the adjusted variable in this case) should decrease.
Direct
By selecting this mode, the adjusted variable increases as the variable that is to be adjusted to match the target increases.
Use this mode when the gain of the process is negative, for example when controlling the level in a tank using the stem position of a valve that controls the outflow out of the tank. In this case, when the level fraction in the tank (the variable that is to be adjusted to match the target in this case) increases, the stem position setting of the valve (the adjusted variable in this case) should also increase.
Typical values for target value - Sets the scaling for the target variable's equations.
Typical values for adjusted variable - Sets the scaling for the adjusted variable's equations.
