Advanced

The Advanced tab is used to specify the outlet temperature and/or pressure.

Name Default Unit Description

Temperature specification

 Selector
Same as inlet temperature
The temperature of each of the outlet streams is set equal to the temperature of the inlet stream.
Specify outlet temperatures per outlet
The temperature of each of the outlet streams is specified directly.
Specify temperature differences between each outlet stream and the inlet stream
The difference in temperature between each outlet stream and the inlet stream is specified. A negative temperature difference indicates that the outlet is colder than the inlet.
Adiabatic and equal outlet temperatures
Determines the outlet temperatures assuming an adiabatic system and equal outlet temperatures.
Pressure specification Selector
Same as inlet pressure
The pressures of the outlet streams are set equal to the pressure of the inlet stream.
Specify outlet pressure per outlet
The pressures of the outlet streams are specified directly.
Specify pressure drop per outlet
The differential pressures between the inlet and the outlets are specified.
