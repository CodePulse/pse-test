Isotherm

The Isotherm tab is used to specify the isotherm that predicts the amount of material that can be adsorbed by the adsorbent material in the bed as a function of the fluid composition and conditions.

Name Default Unit Description
Specifications Selector
Specify isotherm in this bed model
Select and specify an isotherm for the bed model.
Connect isotherm model to this bed model
Connect an isotherm model to this bed model. This is useful if a large number of beds are loaded with the same adsorbents.

Isotherm model type

Selector
Langmuir
Assume the equilibrium between the fluid phase and the adsorbed phase can be predicted by a Langmuir type isotherm.
Langmuir-Freundlich
Assume the equilibrium between the fluid phase and the adsorbed phase can be predicted by a Langmuir-Freundlich type isotherm.
Ideal Adsorbed Solution (IAS)
Assume the equilibrium between the fluid phase and the adsorbed phase can be predicted by an Ideal Adsorbed Solution (IAS) type isotherm.
Custom
Use a custom isotherm to predict the equilibrium between the fluid phase and the adsorbed phase. Use the custom model in gML Templates to code the custom isotherm using the gPROMS ProcessBuilder Custom Modelling feature.
