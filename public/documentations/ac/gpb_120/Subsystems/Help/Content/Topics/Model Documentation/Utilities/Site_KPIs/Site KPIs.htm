Site KPIs

The Site KPIs model can be used to represent customised site Key Performance Indicators, such as comparative steam to fuel and electricity to fuel ratios, and total site supply and demand. There are no direct connections to this model: type the names of each KPI to be added and introduce the equations in the 'gPROMS language' tab. Custom model library management must be enabled in order to configure the gPROMS language. The model icon, as it appears on the Palette, is shown below.
