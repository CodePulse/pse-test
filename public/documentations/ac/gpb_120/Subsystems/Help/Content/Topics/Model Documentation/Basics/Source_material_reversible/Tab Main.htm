The Main tab is used to specify the process material stream data.

Name Default Unit Description

Material

-

 Physical property package to be used.
Pressure bar Pressure of the outlet stream.
Temperature K Temperature of the outlet stream.
Vapour fraction kg/kg Vapour fraction of the outlet flow stream.
Enthalpy kJ/kg Specific enthalpy of the outlet flow stream.
Degree of subcooling K

The difference between the saturated liquid temperature (bubble temperature) and the actual temperature of the outlet stream contents of the vessel.
Degree of superheating K

The difference between the actual temperature of the outlet stream contents of the vessel and the saturated vapour temperature (dew temperature).
Mass fraction kg/kg Mass fraction of the outlet stream.
Molar fraction mol/mol Molar fraction of the outlet stream.
Mass flowrate kg/s Total mass flowrate of the outlet stream.
Molar flowrate kmol/h Total molar flowrate of the outlet stream.
Component mass flowrates kg/s Component mass flowrates of the outlet stream.
Component molar flowrates kmol/h Component molar flowrates of the outlet stream.
Standard volumetric flowrate std m3(1 bar, 0 °C)/h Volumetric flowrate of the outlet stream at standard temperature and pressure conditions.
Volumetric flowrate m3/h Volumetric flowrate of the outlet stream.
Reference pressure bar Reference pressure for volumetric calculations (default value is 1 atm).
Reference temperature K Reference temperature for volumetric calculations (default value is 273.15 K).
