Design (valve type)

The Design (valve type) tab is used to specify data regarding the type of valve used.

Name Default Unit Description

Check valve

Selector
No
In this mode, the valve does not behave like a check valve. If reversible flow is predicted (i.e. the pressure at the valve outlet is higher than at the inlet), a non-reversible valve will fail, since back flow is not allowed. For a reversible valve, both flow directions are allowed.
Forward
The valve behaves like a check valve and flow only occurs in the normal direction (i.e. from valve inlet to outlet ports). If back flow conditions occur, the flowrate through the valve is set to zero.
Backward
The valve behaves like a check valve and flow only occurs in the backward direction (i.e. from valve outlet to inlet ports). If the flow is not in this direction, the flowrate through the valve is set to zero. This option is only available for the reversible valve.
Inherent characteristic Selector
Linear
A linear valve characteristic function is used to determine the flow coefficient fraction.
Equal percentage
An equal percentage valve characteristic function is used to determine the flow coefficient fraction.
Quick opening
A quick opening valve characteristic function is used to determine the flow coefficient fraction.
Butterfly
A butterfly valve characteristic function is used to determine the flow coefficient fraction.
Custom curve
A user-specified custom curve is used to determine the flow coefficient fraction.
Leakage fraction - Assigns the valve leakage fraction.
Rangeability factor - Assigns the valve rangeability factor.
Data table - Data table to be used.
Data table column (valve opening) - Sets the column number containing the valve opening data.
Data table column (FL) - Sets the column number containing the FL data.
Data table column (Xt) - Sets the column number containing the Xt data.
