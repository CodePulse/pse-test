Pressure (Outer pipe)

The Pressure (Outer pipe) tab is used to specify data relating to the pressure drop in the inner pipe of the heat exchanger.

Name Default Unit Description
Pressure drop Selector
Off
Selects no pressure drop.
On
Selects calculation or specification of pressure drop.
Pressure drop specification Selector
Specified pressure drop correlation
Pressure drop is calculated using a correlation.
Specified pressure drop
Pressure drop is specified and the corresponding Fanning friction factor is calculated.
Specified outlet pressure or pressure drop
Outlet pressure is specified and the corresponding Fanning friction factor is calculated.

Pressure drop correlation

 Selector
Specified Fanning friction factor
Selects specification of the Fanning friction factor.
Smooth pipe (Blasius correlation)
Selects the use of the smooth pipe equation.
Linear mass flow coefficient
Selects the specification of a linear mass flow coefficient.
Haaland
Selects the use of the Haaland equation.
Colebrook
Selects the use of the Colebrook equation.
Custom
Selects the use of a custom equation.

Use frictional adjustment factor

 Selector
No
No adjustment factor is used.
Yes
Selects the use of an adjustment factor.
Frictional adjustment factor -
Factor that is multiplied by the computed pressure drop to match an observed pressure drop at certain conditions.
Fanning friction factor - Sets the value of the Fanning friction factor.

Linear mass flow coefficient

 kg s-1 Pa-1 Sets the value of the proportionality constant when Linear mass flow coefficient mode is selected.
Wall roughness m Sets the wall roughness.
