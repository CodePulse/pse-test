Configuration

The Configuration tab is used to specify the controller configuration data.

Name Default Unit Description
Match target Selector
Exactly
The setpoint is set equal to the controlled variable value in a steady-state mode.
Using P-control
The manipulated variable is calculated using a P-control type equation with a specified gain and setpoint in a steady-state mode.
Controller class Selector
PI
Controller with proportional and integral modes.
PID
Controller with proportional, integral and derivative modes.
P
Controller with a proportional mode.
PD
Controller with proportional and derivative modes.

Scaling for gain

 Selector
Scaled to limits
By selecting this option, the gain is scaled to the bounds provided by the controller inputs and outputs.
Not scaled
No scaling is applied to the controller gain.

Controller mode

 Selector
Automatic
The manipulated variable is determined according to the PID dynamic equations.
Manual
The manipulated variable is given a user-specified value that can be reassigned during a dynamic simulation.
Cascade
The manipulated variable is determined according to the PID dynamic equations with an external setpoint. This external setpoint is provided by an external model connected to the external_target port.
Setpoint source Selector
Specified here
The value of the setpoint is specified in the dialog.
From external target input
The value of the setpoint is specified from an external source (e.g. calculation from other model).

Controller action

 Selector
Reverse
By selecting this mode, the Manipulated Variable (MV) decreases as the Process Variable (PV) increases.
Use this mode when the gain of the process is positive, for example when controlling the heater outlet temperature using the heat duty. In this case, when the outlet temperature of the heater (the Process Variable (PV) in this case) increases, the heat duty of the heater (the Manipulated Variable (MV) in this case) should decrease.
Direct
By selecting this mode, the Manipulated Variable (MV) increases as the Process Variable (PV) increases.
Use this mode when the gain of the process is negative, for example when controlling the level in a tank using the stem position of a valve that controls the outflow out of the tank. In this case, when the level fraction in the tank (the Process Variable (PV) in this case) increases, the stem position setting of the valve (the Manipulated Variable (MV) in this case) should also increase.
