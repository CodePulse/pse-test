Common options

The menus at the top of the specification dialog are used to select the operation mode.

Name Default Unit Description

Operation mode

Selector
Initialisation Procedure
A specified outlet energy flow initial guess is used to help initialise the unit model.
Closed
The inlet and outlet Recycle breaker energy streams are identical and a unit initialisation procedure is not used. This option may be selected to initialise dynamic flowsheets with saved variable sets.
Open
The inlet and outlet Recycle breaker energy streams are defined independently functioning as two different energy streams, with the outlet stream energy flow being defined in the Main tab. A unit initialisation procedure is not used. This option may be selected to diagnose initialisation issues in a given flowsheet.
