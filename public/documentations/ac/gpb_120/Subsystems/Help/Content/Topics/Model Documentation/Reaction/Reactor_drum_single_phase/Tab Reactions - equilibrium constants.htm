Reactions - equilibrium constants

The Reactions - equilibrium constants tab is used to specify the parameters for the equilibirum constants of the reactions taking place.

Name Default Unit Description
Equilibrium constants calculation Selector
ln(K) = -ΔG/RT
Calculates the equilibrium constants using the change in the Gibbs free energy from the physical properties package.
K = A*exp(-B/RT)
Determines the equilibrium constant from an exponential expression.
ln(K) = A1*ln(T) + A2/T + A3 + A4*T + A5*T^2 + ...
Determines the equilibrium constant from a polynomial expression.
Equilibrium constant pre-exponential factor (A) Units depends on the chosen basis Pre-exponential factor used in the exponential expression.
Equilibrium constant energy of reaction (B) J/mol Energy of reaction used in the exponential expression.
Number of parameters -
Number of parameters to use for the polynomial expression.
Parameter values Units depends on the chosen basis Parameters used in the polynomial expression.
