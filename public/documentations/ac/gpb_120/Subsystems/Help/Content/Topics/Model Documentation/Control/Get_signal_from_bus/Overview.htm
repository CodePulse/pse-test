Get signal from bus

The Get signal from bus model is used to select a variable from a unit equipment model or a stream analyser and obtain the value as a signal variable. It can be used to interface from gMLBus to gMLSignal connections. This model needs to be used when the variable obtained from the unit equipment model or stream analyser is to be used as an input for a calculation. When the variable in the unit equipment model is to be given the value of the signal variable which is the output of a calculation, the model Set signal to bus ('Set signal to bus') needs to be used instead.
