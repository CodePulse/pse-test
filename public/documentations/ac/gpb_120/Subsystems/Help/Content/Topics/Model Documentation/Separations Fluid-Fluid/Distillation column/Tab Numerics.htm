Numerics

The Numerics tab is used to specify parameters relative to the model initialisation and robustness. Under normal circumstances, there is nothing to change in this tab.

Name Default Unit Description
Maximal number of stages for optimisation Selector
Equal to number of stages
Sets the upper bound for the number of stages during an optimisation to the value specified in the Column tab.
Specified
Uses the value specified in the dialog for the upper bound for the number of stages during an optimisation.

Initialisation procedure

 Selector
Iterative method with simplified thermodynamics
This is an iterative method based on the bubble/dew point algorithm.
Newton's method
This method determines an initial composition and temperature profile in the column from a flash of the feed and then reverts to solving the MESH equations.
Dynamic
This method is the default when the column is used in dynamic mode with non-steady state initial conditions.
Off (Requires Saved Variable Set)
No initialisation procedure is employed. The column model requires a Saved Variable Set to converge successfully.
Mass balance basis for initialisation procedure Selector
Liquid
By selecting this option, the mass balance in the iterative initialisation method will be written in terms of the liquid phase. This option is recommended when non-volatile components are considered.
Vapour
By selecting this option, the mass balance in the iterative initialisation method will be written in terms of the vapour phase. This option is recommended when non-condensible components are considered.
Initialisation with ramped side draw flowrates Selector
No
The side draw flowrates or fractions of the total flow will remain constant during the initialisation procedure, according to the value specified in the Side operations tab.
Yes
By selecting this option, the side draw flowrates or side draw fractions of the total flow will be iteratively ramped up during the initialisation procedure. This feature may improve model robustness by avoiding mass balance inequalities (e.g., sum of the outlet mass flowrates larger than the sum of the inlet mass flowrates) during the initialisation stage.
Side draw ramp initialisation specification Selector
Step size
The step size increase in the side draw flowrate scaling factor of each iterative cycle in the initialisation procedure.
Number of iterations
The number of iterative changes in the side draw flowrate scaling factor during the initialisation procedure.
Switch from initial guesses to main specification Selector
Fast
Applies to the last initialisation step of the column, when switching from the specifications in the Initial guesses tab to the ones in the Operation tab (main specifications). Fast means that the switch is done by changing the specifications and reinitialising directly.
Robust
Applies to the last initialisation step of the column, when switching from the specifications in the Initial guesses tab to the ones in the Operation tab (main specifications). Robust means that the switch is done following a linear path from the initial specifications to the main specifications.
Typical energy rate kJ/s Specifies the typical energy rate for this particular simulation. This should be set to a value equal to the order of magnitude of the typical energy rate for this unit in the process being simulated. Under normal circumstances, the typical energy rate parameter does not need to be specified but for some simulations with either very small or large energy rates, setting this parameter might improve the accuracy of the results.
Typical flowrate kg/s Specifies the typical flowrate for this simulation. Under normal circumstances, this flowrate parameter does not need to be specified but for some simulations, with either very small or very large flowrates in (part of) the flowsheet, setting this parameter may improve the accuracy of the results.
Initialisation damping factor - Damping factor for the iterative initialisation method.
Maximum number of iterations - Maximum number of iterations to use while initialising the column.
Convergence criterion - Convergence criterion used for the iterative initialisation method.
Initial side draw flowrate scaling factor -

If the side draw flowrates are given, this value refers to the initial side draw flowrate divided by the specified side flowrate. If the side draw fractions of the total flow are specified by the user then the scaling factor is the side draw initial fraction of the total flow divided by the user-supplied side draw fraction.
Step size for side draw initialisation - The absolute step size increase in the side draw flowrate scaling factor during each iterative cycle in the initialisation procedure.
Number of iterations for side draw initialisation - The number of iterative cycles which lead to a change in the side draw flowrate scaling factor during the initialisation procedure.
