Manufacturing

Example flowsheets related to manufacturing processes can be found within this folder. There are three main sections considering synthesis and solvent recovery, solids processing and crystallization.

Crystallization, wet milling and filtration

This folder contains the following examples for crystallization, wet milling and filtration for both batch and continuous processes. Further information can be found on the flowsheet:

  • gCRYSTAL API batch cooling - optimisation
  • gCRYSTAL Batch and continuous crystallization
  • gCRYSTAL Batch Crystallization - Parameter Estimation
  • gCRYSTAL Batch reaction and crystallization
  • gCRYSTAL Plug flow crystallization
  • gCRYSTAL Polymorphism L-glutamic acid
  • gCRYSTAL Wet milling

Synthesis and solvent recovery

There are two example projects demonstrating the reactors; one demonstrates in a single phase scenario, the other in a two phase scenario. Both batch and continuous processes can be found in the following projects:

  • gFORMULATE Reactors - Single phase
  • gFORMULATE Reactors - Two phase

Solids processing

Solids processing is a diverse area, the model library is comprised of unit operations to support batch and continuous processing as well as dry, wet and direct routes. The following examples contain further information on the flowsheets:

  • gSOLIDS Twin screw granulation
  • gFORMULATE Agglomerator
  • gSOLIDS Closed loop spray drying
  • gSOLIDS Continuous blending
  • gSOLIDS Dry milling
  • gSOLIDS High shear granulation
  • gSOLIDS Roller compactor
  • gSOLIDS Spray drying tech transfer
  • gSOLIDS Spray drying
  • gSOLIDS Tablet compaction with lubrication
  • gSOLIDS Tablet press
