Features still to be ported

A vast majority of legacy models have been ported to gPROMS FormulatedProducts. However, there are some features that have not yet been transferred:

  • Solids processing
    • gSOLIDS conveying library
    • Fluid bed granulator
  • Crystallization, wet milling and filtration
    • Filtration models
    • Classification in the crystallizers
    • Evaporation in the crystallizers
    • Gahn & Mersmann attrition model
    • Multizonal models for spray drying and crystallization
  • Product performance
    • Food dose
    • In vitro infusion
    • Post prandial dynamic physiology (e.g. In vivo feeding paradigm option)
    • Gibbs-Thomson equation for particle size dependent solubility
    • In vitro vessel media dialog functionality to specify the total buffer concentration
    • Halide ion option in the Thelen et al. permeation model
    • Permeability analyser
  • Help documentation does not include information on dialog specifications

Some models have been discontinued and will be replaced by new functionality:

  • Product performance
    • Solubility comparator replaced by upcoming external model validation capabilities
    • PK comparator replaced by upcoming external model validation capabilities
