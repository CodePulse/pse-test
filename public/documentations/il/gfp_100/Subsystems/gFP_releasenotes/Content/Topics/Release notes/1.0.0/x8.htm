Known issues

gPROMS FormulatedProducts has undergone extensive testing and as such some issues have been identified. The list below gives the known issues:

  • Global specifications in gFP gSOLIDS, gFP gCRYSTAL and gFP gCOAS libraries
    • Global specifications model for gSOLIDS, gCRYSTAL and gCOAS has been included to allow users to transfer their previous specifications without requiring to import into a database
    • This feature has several limitations including:
      • Only one flowsheet maybe included in a project using this feature
      • gCRYSTAL only: custom options for physical properties cannot be used, van't Hoff equation model not ready for use, "Equilibrium vapour composition" is not currently available

      • Users may need to close and reopen the project in order for their specifications to be recorded

  • gFP gCOAS libraries
    • Selecting inactive growth in the in vitro and in vivo models also switches off dissolution.

If any further issues are identified that are not in the list, please contact support.formulatedproducts@psenterprise.com.
