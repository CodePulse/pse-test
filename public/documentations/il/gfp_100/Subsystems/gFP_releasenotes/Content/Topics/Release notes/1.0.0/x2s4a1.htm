Opening legacy flowsheets with legacy libraries

The libraries of gCRYSTAL 4.2, gSOLIDS 4.1 and gCOAS 1.3 are included in the installer of gPROMS FormulatedProducts 1.0.

To open your legacy flowsheet with legacy libraries:

  1. Close all libraries currently open
  2. Click the "Open libraries" icon or follow "File" -> "Open/close libraries..."
  3. Select the library of interest e.g. "gSOLIDS 4.1 - pre-gFORMULATE"
  4. Open the legacy flowsheet
  5. Use as normal
    1. Note there may be warnings due to new validation rules for the specifications - these can be ignored
  6. Optional: Utilise the new Global System Analysis feature - right-click the project folder and create a new "Global System Analysis" entity
