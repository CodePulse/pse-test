Selecting lower and upper bounds and distribution

The lower and upper bounds of factors should be inferred from the region of interest and must be within the bounds of the mapped variable. Any bound violations are detected and warnings displayed.

Factors can be disabled via a checkbox, which is enabled by default. Disabled factors remain constant at user-provided values in the Process during the execution of a GSA activity, and are ignored in the execution results. However, these values must lie within the bounds of the factor.

The distribution dictates which factor values are used in the analysis. For deterministic distributions, factor values are either user-specified values or grids. For probabilistic distributions, each factor value is allocated a probability and samples are taken from this distribution based on the selected sampling method.

GSA supports the variability/probability distributions listed and shown in the figure below (note that different factors can have different distributions):

  • Deterministic
    • Enumerated set: This allows the identification of factor values with a scenario to facilitate the analysis of the responses. For example, consider equipment sizes. With an enumerated set, it is easy to filter the analysis results by individual equipment sizes. Note that it is only possible to analyse the results for one scenario at a time.
    • Gridded set: This permits the generation of an equidistant grid of factor values, with the possibility of including or excluding the lower and upper bounds of the domain. This is useful when a balanced overview of the factor space is required.
  • Probabilistic
    • Uniform distribution: In this case, the probability of the factor value is the same for every value in the specified domain.
    • Discrete distribution: This distribution allows the identification of valid values and their associated probability. This is particularly useful when a factor is limited to discrete values, for example, catalyst pellet sizes.
    • Normal distribution: The probability of the factor follows a normal distribution with user-specified mean and variance.

Factor distribution options

The selection of an appropriate distribution is case-specific and should be based on knowledge of the system. For example, to conduct a scenario analysis, only select deterministic factors. However, if uncertainty in estimated parameter values is to be considered, then a parameter estimation will provide information on the distribution parameters (mean and covariance).

The distribution can be determined in three ways (Tony Oâ€™Hagan, 1998):

  • Directly from data, if available
  • Through statistical analysis, for example, from a covariance matrix supplied by parameter estimation results
  • Expert elicitation.

If none of the above are available or if an overview of the factor space is required, either gridded or uniform distributions can be used to ensure an even coverage of the factor space. However, note that the choice of distribution will influence the results, and it is recommended that an attempt to determine the correct distribution is made beforehand.
