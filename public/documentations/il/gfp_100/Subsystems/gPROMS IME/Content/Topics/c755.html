Constructing flowsheet Models

gPROMS FormulatedProducts allows users to build composite Models graphically. This functionality is designed to enable one quickly to construct process flowsheets that are suitable for use in any Model based activity (e.g. steady-state and dynamic simulation, parameter estimation and optimisation studies). Recall the standard gPROMS language and Properties tabs that are used to describe Model Entities: the Topology tab is used to put flowsheets together graphically and to specify the information needed for successful Model based activity.
