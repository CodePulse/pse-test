Read-only Projects

In order to protect against inadvertent modifications being made to a particular Project or Library project, it is recommended that the corresponding gPJ file be declared as read-only using standard operating system facilities prior to loading the project into gPROMS FormulatedProducts. In such cases, a lock symbol will be displayed against both the Project and any Entity that it contains in the gPROMS FormulatedProducts's navigation tree. Any operation that may result in a modification of this Project is automatically disabled.
