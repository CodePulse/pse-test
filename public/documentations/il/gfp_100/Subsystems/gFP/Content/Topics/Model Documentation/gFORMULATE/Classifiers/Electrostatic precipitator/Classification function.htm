Classification function

The classification function is the ratio of mass flow rate between the oversize stream and the inlet stream (for a given particle size). The classification function is as described below.

Note: In the equations below, parameters to be specified or estimated are printed in blue.

The migration velocity is in turn defined as follows. For particles greater than 1 μm in size, the migration velocity is a function of particle size. For particles less than 1 μm in size, the migration velocity is constant.
