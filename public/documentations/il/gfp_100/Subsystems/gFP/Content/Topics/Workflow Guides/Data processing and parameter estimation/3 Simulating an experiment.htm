Simulating an experiment

Note

Always do this before including an experiment in parameter estimation to ensure correct configuration of your flowsheet.

To do this, follow the steps below:

  • Right click on the experiment entity.
  • Click on simulate. 
  • gPROMS FormulatedProducts will prompt you for the name of a new Process entity.
  • Accept the default name provided.
  • A new process will be created within the same project as the original Process
  • If no problems are reported, this process will automatically be run as a Simulation activity. 
  • Once the simulation has completed and you have reviewed the results (in particular the implementation of the controls), you can either delete the newly created Process or keep it.

 

Follow the above steps for all the experiments you enter in the performed experiment entity before carrying out the parameter estimation.
