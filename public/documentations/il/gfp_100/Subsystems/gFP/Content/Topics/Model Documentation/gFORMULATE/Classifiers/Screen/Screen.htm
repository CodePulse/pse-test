Screen

This document describes the steady state screen model. This model provides multiple approaches for predicting the performance of this separation unit. In this unit operation, solid particles are made to flow over a sieve of a certain aperture size. In theory, all particles with sizes greater than the aperture remain on the screen and go into the oversize stream while all particles with sizes smaller than the aperture size fall through the screen. This is referred to as ideal screening. However, in practice some of the smaller particles may stay in the overflow stream leading to non-ideal separation of particles. This model provides the user with the option to assume either ideal or non-ideal screening.

The screen model is used to model a screen, where solid particles flow over a sieve and are separated as particles smaller than the aperture size fall through.

Key features:

  • No accumulation of material within unit.
  • Ideal or non-ideal separation of material can be assumed.
  • For non-ideal separation, two options are provided. In the first option, the extent of non-ideality is defined by the separation range parameter specified by the user. For example, if the user specifies a separation range of 20 micron, then 90% of particles of size = aperture size + 20 micron go into the oversize stream with the remaining particles going into the undersize stream. In the second option, the user must specify the number of screens, the residence time on each screen and the screening rate constant. This information is then used to calculate the screening efficiency curve.
