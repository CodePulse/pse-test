Mass conservation check

The mass conservation check model can be used to check that mass is conserved in a flowsheet. It is particularly useful for testing whether custom user models successfully conserve mass.

Key features:

  • Connects to all sources, sinks, and unit operations that have hold up.
  • Evaluates whether the change in the total hold up is equal to difference between the mass created through sources and the mass consumed by sinks.
