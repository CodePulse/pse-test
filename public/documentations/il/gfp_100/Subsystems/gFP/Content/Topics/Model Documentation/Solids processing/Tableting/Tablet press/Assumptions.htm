Assumptions

The model is developed under the following assumptions:

  • Tablets are assumed to be cylindrical with the compaction force applied perpendicular to the circular cross section of the tablet.
  • At any given point in time, all tablets are assumed to have the same size and content. Variability in tablet uniformity is not considered.
  • The tensile strength of the tablets is assumed to be a function of the bulk chemical composition of the tablets. The effect of the size distribution of the constituent particles is not considered. As a result, two tablets with the same bulk chemical composition but with particles of different sizes will be assumed to have the same tensile strength.
  • The maximum density of the tablets (that is when the porosity is zero) corresponds to the skeletal (crystalline) density of the constituent material.
  • The compaction process is assumed to remove both the void between particles (inter-particle void) and the internal void within particles (intra-particle void). As the compaction pressure increases, the inter-particle void is affected first. When the inter-particle void is reduced to zero, the intra-particle void begins to be compacted.
  • If there is a granule phase in the tablet, the intra-particle void within the granule phase consists of two parts: inter-particle void between granule primary particles and intra-particle void within the primary particles. The inter-particle void is compacted first, and then the intra-particle void decreases.
