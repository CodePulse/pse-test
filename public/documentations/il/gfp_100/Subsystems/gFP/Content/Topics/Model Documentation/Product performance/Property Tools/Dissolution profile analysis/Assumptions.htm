Assumptions

The dissolution profile analysis makes the following assumptions:

  1. Experimental dissolution data should always increase over time. If it decreases at any point, dissolution will be kept constant until the data reaches the same dissolution value where it started decreasing

User data is interpolated with splines, which are known to have a few drawbacks, namely:

  1. They only apply to the range of values that are being interpolated
  2. The function is forced to fit the data so undesired points are accounted for
  3. Data needs to be fairly smooth (i.e., there are no sharp discontinuities)

Users should be warned that calculating splines for large data sets will result in slower simulations due to the additional computational effort required (in comparison with smaller data sets, or with other dissolution profile models).
