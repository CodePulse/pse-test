Classification functions

The classification function is the ratio of mass flow rate between the oversize stream and the inlet stream (for a given particle size). A few options are available for the classification function as described below.

Note: In the equations below, parameters to be specified or estimated are printed in blue.

Ideal separation

In this approach, all particles of size greater than the aperture size are assumed to go into the oversize stream while all particles of size lower than the aperture size are assumed to go into the undersize stream.

Separation curve

In this approach, all particles greater than the aperture size are assumed to go into the oversize stream. However, some particles smaller than the aperture size are assumed to stay in the oversize stream. In particular, this applies to particles slightly smaller than the aperture size. For particles much smaller than the aperture size, separation is assumed to be 100% efficient. This behaviour is captured by defining a range of particle sizes where the separation is not 100% efficient and by defining a sigmoidal classification function over this range.
