Developing models with gML Utilities

This is a brief overview of creating flowsheets based on the gML Utilities model libraries. These libraries can be combined with other gPROMS libraries and custom models. This Workflow Guide is supplied to assist with initial familiarisation with this PSE product and should not be regarded as a full description of the capabilities available in gPROMS. A more detailed explanation of the gPROMS platform can be found in the gPROMS ProcessBuilder Guide.

This guide is split into the following sections:

Creating flowsheets

Configuring custom equations

Executing simulations
