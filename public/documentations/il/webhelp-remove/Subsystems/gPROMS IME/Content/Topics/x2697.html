Export to gPROMS ModelBuilder v3.5 Project

gPROMS ModelBuilder v4.0 uses a different file format for storing its projects to that used by versions of gPROMS ModelBuilder prior to v3.5 (inclusive). In the interests of forward compatibility, gPROMS ModelBuilder v4.0 projects may be exported in v3.5 format using the Export ... tool in the Tools menu. Note that not all newer gPROMS concepts are recognised by those earlier versions of gPROMS ModelBuilder and that the exported project will load but may require manual work to make it fully functional.
