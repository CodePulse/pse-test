Navigation shortcuts - Jump list shortcuts

Jump lists are implicitly created when editing the gPROMS language of a gPROMS entity or a text file. If a file is edited in a sequence of locations, then each location creates an entry in the jump list. For instance, when editing line 15, then line 6, then line 31 and then line 1, four entries will have been created and the keyboard shortcuts below allow the cycling through these locations. At present, only backward cycling is enabled; all other shortcuts in the table below are disabled.

Jump list shortcuts
Keyboard shortcut Action
ALT+k Go to previous entry in the jump list.
ALT+l Go to next entry in the jump list. Currently disabled.
SHIFT+ALT+k Go to the previous jump list entry not in the same file. Currently disabled.
SHIFT+ALT+l Go to the next jump list entry not in the same file. Currently disabled.
