Miscellaneous Utilities

A range of other features increase the functionality of gPROMS ModelBuilder. These features include standard utilities such as printing capabilities and window layouts. Also, gPROMS ModelBuilder has a number of shortcut keys, many of which are the same as standard shortcut keys plus some gPROMS ModelBuilder specific ones.
