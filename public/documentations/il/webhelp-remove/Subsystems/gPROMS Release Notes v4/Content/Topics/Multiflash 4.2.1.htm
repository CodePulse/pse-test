Multiflash

Multiflash has been updated and improved in the following areas:

Update to Multiflash v4.3.54

The engine and GUI of Multiflash for gPROMS have been updated to v4.3.54. The new version resolves issues for chemical equilibrium calculations where trace components had not been considered properly which had led to fatal errors.

Additional physical property methods

The following methods have been added to the Multiflash Foreign Object for improved interchangeability with the CAPE-OPEN Foreign Object.

TVFFlash

Equilibrium flash given temperature and vapour fraction. This flash is an alias for TVFMassFlash or TVFMolarFlash depending if mass or moles are the basis of the calculation.
