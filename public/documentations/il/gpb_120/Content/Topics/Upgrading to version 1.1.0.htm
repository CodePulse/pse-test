Upgrading to version 1.2.0

Process Systems Enterprise Limited will update client flowsheets created using version 1.0.0 or version 1.1.0, if required.

The known issues associated with updating existing flowsheets to version 1.2.0 are listed below.

  • The outlet temperature for the Compressor centrifugal model might be slightly different if the mechanical efficiency was not set to 100%.
  • Specifications for columns with side-draws might need to be updated.
  • Pressure drops in heaters, coolers and heat exchangers need to be turned on if they were non-zero.
  • Warnings have been added to the libraries, which may cause some units to turn red in the Execution output during simulation. If this happens, it is important to check the Warnings report for that unit.
