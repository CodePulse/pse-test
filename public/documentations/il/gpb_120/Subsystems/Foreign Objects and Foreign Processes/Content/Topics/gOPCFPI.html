Introduction

The OPC Foreign Process Interface gOPCFPI is designed for the easy configuration and use of bidirectional connections for data exchange with OPC servers. This data exchange can be used for both online and offline applications.

Its key features are listed below.

  • Implementation of the standard FPI elementary tasks:
    • Get
    • Send
    • Pause
  • 'Get' and 'Send' support exchanging both scalars and arrays of values
  • Built-in access to an OPC historian
  • Automatic synchronisation with real time for each 'Get' and 'Send' elementary task execution
  • Easy adaptation to systems with their own time scale via a configurable real-time factor, that is either fixed or (dynamically) adjusted by the OPC server
  • Automatic establishment and termination of connection with OPC server
  • All specifications related to the OPC communication are held in a clear-text configuration file and require no programming effort.

The following FPI functions are not supported by gOPCFPI:

  • SendMathinfo
  • Linearise.
