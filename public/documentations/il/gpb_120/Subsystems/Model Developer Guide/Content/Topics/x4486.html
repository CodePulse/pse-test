3D Plot Orientation

It is often necessary to adjust the orientation of a 3D plot to show the shape of the surface more clearly. This can be achieved using the <Rotation/> tag within the <Plot3D> construct. This allows the user to specify the orientation of the plot by using standard rotations about each of the cartesian-coordinate axes. The basic syntax is as follows (using the example presented before).

<Plot3D version="1">
 <Surface id="Temperature(TIME=0,#x,#y)"/>
 <Rotation x="0" y="0" z="0"/>
</Plot3D>

Each attribute defines the rotation in degrees about its corresponding axis. The values may be positive or negative. The values above (i.e. no rotation) produce a coordinate system with the x and z axes in the plane of the screen/page, with the x axis horizonal. By defintion, the y axis therefore points into the screen/page. This is not a particularly useful view, so the default view is to rotate the coordinates 45 degrees about both the x and z axes. In other words, omitting the <Rotation/> tag altogether is the same as specifying:

<Plot3D version="1">
 <Surface id="Temperature(TIME=0,#x,#y)"/>
 <Rotation x="45" y="0" z="45"/>
</Plot3D>

Of course, any other values may be specified to suit the Variable being plotted.
