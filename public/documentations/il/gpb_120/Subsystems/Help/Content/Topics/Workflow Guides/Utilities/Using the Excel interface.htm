Using the Excel interface to optimise multiple scenarios

This Workflow Guide describes the setup for executing multiple optimisations from an external interface (currently MS Excel). This is done using gO: Run, the gPROMS executable solution engine which can be used to perform simulations and optimisations from the command line.

gML Utilities currently provides an Excel template for configuring and running multiple optimisations. This interface is described here. The release version will include a wizard for preparing the gPROMS project for remote execution and pre-populating the Excel template.

An example is provided in the 'Examples' subfolder of the installation directory.
