Stream analyzer

The Stream analyzer model can be used to report a range of material stream properties or compositions and properties of individual phases present in the material stream. Based on the stream mass fractions and conditions, this model can be used to determine molar fractions, or to invoke the physical property package to determine properties such as vapour fraction, density, heat capacity, viscosity and thermal conductivity.

This model can also be used to pass this type of information to a gMLBus port for use in trading-off specifications, additional calculations, etc.

N.B. inserting this model in the flowsheet does not affect the results of the simulation.
