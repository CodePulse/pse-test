Standard coil geometries

The standard geometries for the different radiant coils available are presented below. The geometry values can be changed in the model specification dialog of the Furnace model. When the desired coil geometry is not available, a custom coil model with the desired coil geometry can be created and integrated to the Furnace model. For creating custom coil check 'Creating custom coil models'.
