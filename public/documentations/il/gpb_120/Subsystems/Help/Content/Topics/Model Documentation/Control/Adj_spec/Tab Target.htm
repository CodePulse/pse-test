Target

The Target tab is used to specify the target variable types/values.

Name Default Unit Description

Target variable type

 Selector
Scalar
The specified variable is a scalar.
Array
The specified variable is an array.
Set
The specified variable is a set.

Value for target variable

 Selector
Specified here
The target variable value is specified internally (in the Adj spec model).
From external target input
An external model is used to assign the target variable value. This is done by connecting an external model with the desired target variable value using the external_target port and simultaneously connecting the target port to the model with the required target variable.

Target variable

-

Selects the target variable from a drop-down menu with options given by the model connected to the target port.

Target value

 The units of the variables depend on the type of variable chosen (see 'Units of measurement in gMLBus ports'). Specify the value of the chosen variable.
Target variable array index - Sets the index of the chosen element in an array variable.
Target variable element index - Selects the chosen element in a set variable.
