Pseudo-components

The Pseudo-components tab is used define pseudo-components and determine their composition in terms of the original components.

Name Default Unit Description
Pseudo-components in outlet  
Select which outlet components are considered to be pseudo-components from the list of all outlet components. Each of these outlet components consists of multiple inlet components.
Pseudo-components in inlet   Select which inlet components are considered to be pseudo-components from the list of all inlet components. Each of these inlet components consists of multiple outlet components.
Inlet components that make up each outlet pseudo-component   For each outlet psuedo-component, select which inlet components are grouped in this pseudo-component.
Inlet pseudo-component composition in terms of outlet components   For each inlet psuedo-component, specify what its composition is in terms of the outlet components.
