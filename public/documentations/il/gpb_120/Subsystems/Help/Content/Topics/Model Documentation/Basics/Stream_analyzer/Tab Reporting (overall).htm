Reporting (overall)

The Reporting (overall) tab is used to select which overall results are reported.

Name Default Unit Description

Vapour fraction

 Selector
Off
Turns off the option to report the stream vapour fraction.
On
Turns on the option to report the stream vapour fraction.
Flash analysis Selector
Off
Turns off the option to report stream flash analysis data.
On
Turns on the option to report stream flash analysis data.
Volumetric flowrate Selector
Off
Turns off the option to report the stream volumetric flowrate.
On
Turns on the option to report the stream volumetric flowrate.
Density Selector
Off
Turns off the option to report the stream overall density.
On
Turns on the option to report the stream overall density.
Entropy Selector
Off
Turns off the option to report the stream overall mass specific entropy.
On
Turns on the option to report the stream overall mass specific entropy.
Bubble temperature Selector
Off
Turns off the option to report the stream bubble temperature.
On
Turns on the option to report the stream bubble temperature.
Dew temperature Selector
Off
Turns off the option to report the stream dew temperature.
On
Turns on the option to report the stream dew temperature.
Vapour quality Selector
Off
Turns off the option to report the vapour quality.
On
Turns on the option to report the vapour quality. Vapour quality on an enthalpy basis can go over 1 for superheated mixture and below 0 for subcooled mixtures.
Conditions (P, T) for volumetric flowrate calculation Selector
Conditions of mixture
Uses the pressure and the temperature of the mixture.
Standard conditions
Uses the references pressure and temperature provided in this dialog.
Standard pressure bar Reference pressure for volumetric calculations (default value is 1 atm).
Standard temperature K Reference temperature for volumetric calculations (default value is 273.15 K).
