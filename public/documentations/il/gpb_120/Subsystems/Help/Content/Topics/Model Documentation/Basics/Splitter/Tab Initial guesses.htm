Initial guesses

The Initial guesses tab is used to provide initial guesses for some variables in order to improve the speed and robustness of initialisation.

Name Default Unit Description

Specification type

Selector
Split fraction
Determines the initial mass flowrate for each outlet as a fraction of the total inlet flow. Using this option might be more appropriate if the inlet flowrate is subject to change.
Flowrate
Determines the flowrate for each outlet by specifying it directly.
Initial flowrate Selector
Equal to inlet flowrate
Supplies a guess for the initial flowrate equal to the inlet flowrate.
Specified
Use the value specified in this dialog for the initial flowrate.
Split fractions - The split fraction is a value between 0 and 1 which determines the proportion of the total inlet flow that will initially be split into each outlet.
Mass flowrate kg/s All but one outlet mass flowrates are provided as initial guesses.
Molar flowrate kmol/h All but one outlet molar flowrates are provided as initial guesses.
