Reactions - cracking

The Reactions - cracking tab is used to specify data pertaining to the cracking reactions. Details of cracking kinetic models and parameters can be found in 'Cracking kinetic models and parameters'.

Name Default Unit Description
Cracking Selector
On - Specified here
This option enables cracking in the Cracking tube model and the cracking specifications are set in the unit's dialog.
On - Specified at higher level
This option enables cracking in the Cracking tube model, however, the cracking specifications must be set in the unit that uses the Cracking tube model.
Off
If this option is selected then no cracking occurs.
Kinetic model source Selector
Default
Allows to choose one of the kinetic models available as part gML Olefins libraries.
External kinetics package
Allows to use cracking kinetics from an external package. Please contact PSE to link external kinetic packages to furnace models in gML Olefins.
Assumption for radical species Selector
None
Concentration of radicals calculated based on mass balance of radicals.
Quasi steady-state (QSSA)
Concentration of radicals is calculated using quasi steady state assumption.

Reaction mechanism involving both molecular and radical components.

Physical properties Selector
Using molecular properties only
Only the molecular components will be used for the calculation of physical properties. Physical properties will be calculated using Multiflash.
Using custom properties
This option enables the use of custom physical properties for the calculation of enthalpy of formation, entropy of formation and specific heat capacity. The values of physical properties (Enthalpy of formation and entropy of formation) and the coefficients needed to calculate specific heat capacity of each component (a0, a1, a2, b) needs to specified in a .txt file. The coefficients for calculating gPROMS ProcessBuilder accesses these properties using ReadDataFO (ReadData Foreign Object). The specific heat for each component is calculated as:
Cp = a0 T^3 + a1 T^2 + a2 T + b
 
Reactions for rate constant tuning -

Set of reactions selected for rate constant tuning.
Reactions for activation energy tuning - Set of reactions selected for activation energy tuning. This is applicable for molecular type kinetic mechanisms.
Rate constant multipliers -

Rate constant multipliers for the reactions selected for tuning. This is applicable for molecular type kinetic mechanisms.

Reaction rate constant at 1073 K = Rate constant multiplier x Default rate constant at 1073 K
Activation energy adjustment J/mol

Adjustment to the default activation energies for the reactions selected for activation energy tuning.

Activation energy = Default activation energy + Activation energy adjustment
Custom model - Drop-down menu containing all the user-defined cracking kinetic models.
Pathname for the LSKM.FO text files - Input files for the LSKM (large-scale kinetics mechanisms) foreign object, which is used for the implementation radical type kinetic mechanisms. These comprise the SpeciesFile and the ReactionFile.
Pathname for the ReadData.FO text files - Input file containing the enthalpy of formation, entropy of formation and heat capacity parameters.
