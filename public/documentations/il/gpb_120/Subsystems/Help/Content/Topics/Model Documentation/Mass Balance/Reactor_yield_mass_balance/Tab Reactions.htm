Reactions

The Reactions tab is used to specify reaction data.

Name Default Unit Description

Reactions

Selector
Specified here
The reaction data is specified internally (tab Reactions).
From kinetic connection
The reaction data is given externally by the model connected to the gMLKinetics port.

Conversion specification mode

 Selector
Parallel conversions
All yield reactions occur independently from one another.
Serial conversions
Yield reactions occur in a user-defined ordered sequence.

Yield specification basis

 Selector
Mass
The yield per component is defined in a mass basis.
Molar
The yield per component is defined in a molar basis.
Components for which yield is defined - Selects the components for which the yield is defined.
Yield per component - Defines the yield for each product component, i.e., the amount of each product component generated per amount of component (for which the yield is defined).
Yield percentage % Assigns the yield percentage for all the reactant components for which the yield is defined. The yield percentage is defined as the amount of component that reacts per unit of time divided by the inlet component flowrate.
