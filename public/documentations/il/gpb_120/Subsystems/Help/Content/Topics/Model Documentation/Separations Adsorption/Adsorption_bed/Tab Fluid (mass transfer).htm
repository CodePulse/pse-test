Fluid (mass transfer)

The Fluid (mass transfer) tab is used to specify the mass transfer properties for the fluid phase.

Name Default Unit Description
Mass transfer from fluid to adsorbent(s) Selector
LDF (Linear Driving Force)
Mass transfer from the fluid to the adsorbents is governed by a linear driving force model.
Custom
Custom mass transfer driving force relation is provided. Use the custom model in gML Templates to code the custom correlation using the gPROMS ProcessBuilder Custom Modelling feature.
Mass transfer coefficient calculation Selector
Constant
Mass transfer coefficient is constant and does not depend on fluid properties and flowrate.
Bulk
Mass transfer limitation is assumed to be in the bulk fluid phase.
Knudsen
Mass transfer is assumed to be limited by Knudsen diffusion in the adsorbent particles.
Custom
Custom mass transfer coefficient correlation. Use the custom model in gML Templates to code the custom correlation using the gPROMS ProcessBuilder Custom Modelling feature.

Mass transfer driving force basis

Selector
Solid concentration
Driving force for mass transfer from the fluid phase to the solid phase calculated based on the difference between the actual solid concentration and the solid concentration in equilibrium with the fluid partial pressure as predicted by the isotherm model.
Bulk phase molar concentration
Driving force for mass transfer from the fluid phase to the solid phase calculated based on the difference between the actual bulk phase molar concentration and the molar concentration in equilibrium with the solid phase concentration as predicted by the isotherm model.
Bulk phase partial pressure
Driving force for mass transfer from the fluid phase to the solid phase calculated based on the difference between the actual partial pressure and the partial pressure in equilibrium with the solid phase concentration as predicted by the isotherm model.

Dispersion coefficient

 Selector
Specified
Directly specify an overall dispersion coefficient.
Calculated (Wakao, T.Funazkri, 1978)
Calculate from the component molecular diffusivities.
Custom
Custom dispersion coefficient correlation. Use the custom model in gML Templates to code the custom correlation using the gPROMS ProcessBuilder Custom Modelling feature.
No dispersion
Assume there is no dispersion in the flow direction.
