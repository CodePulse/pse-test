gMLMaterial to PMLMaterial

The gMLMaterial to PMLMaterial model can be used to convert from the streams used in the gML libraries to the streams used in the PML libraries. This allows the construction of flowsheets involving sections with models from both libraries. This can be useful for example to interface custom models developed on the basis of the PML library stream to models from the gML library.

The gML and the PML libraries differ in the aspect of how the enthalpy reference state is defined. For the gML libraries this is assumed to be on an element basis, for the PML this is assumed to be on a component basis. To ensure that this this is handled correctly, the converter requires the definition of a physical property package to be used for the downstream section of the flowsheet which uses the PML convention. This physical property package should return enthalpies on a compound basis.
