The Interface tab

Once a model of a utility system is built, configured and tested, it can also be deployed within the gPROMS Utilities Planner and gPROMS Utilities Advisor applications. These applications include interactivity with existing flowsheets created using gML Utilities, such as:

  • Setting model values based on reading planning forecast information or live site measurements ("Variable" settings)
  • Optimising utility management based on reading availability information and/or current operating conditions ("Managed" settings)
  • Applying and adjusting contractual costs based on reading contract information ("Contract" settings)
  • Reconciling steam flows at minimum RMS error

For each model in the gML Utilities library, options for Embedded Applications may be found in the 'Interface' tab. Any model variables which may be "Variable" or "Managed" will appear with their default setting in a drop-down box which can be changed on a unit by unit basis in order to further customise the application.
