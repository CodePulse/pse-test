Jacket configuration

The Jacket configuration tab is used to specify jacket design and operation parameters.

Name Default Unit Description
Flow direction Selector
Co-current
Co-current flow direction of the cooling fluid stream with the associated bed section's flow direction.
Counter-current
Counter-current flow direction of the cooling fluid stream with the associated bed section's flow direction.
Jacket diameter m
Diameter of the cooling jacket.
H.T.C. adjustment factor - Adjustable factor (a multiplier) used in the heat transfer coefficient correlation.
