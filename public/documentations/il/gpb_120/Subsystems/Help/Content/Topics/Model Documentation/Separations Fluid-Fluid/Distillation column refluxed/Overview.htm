Distillation column refluxed

This model describes a two-phase (vapour-liquid) refluxed column. The column is divided into multiple stages. For each stage a mass and energy balance is determined and by default it is assumed that vapour-liquid equilibrium is achieved at each stage.

In the condenser it is also assumed that vapour-liquid equilibrium is achieved. Columns with total and partial condensers can be modelled. The column model can also represent columns with liquid and vapour side-draws.

Non-ideal trays can be modelled using the Murphree efficiency.

This model is a distillation column with no reboiler. The options of this model are similar to those of the distillation column but the user needs to provide only one or two specifications in the Operation tab of the model depending on whether a total or partial condenser is used. For a column with a total condenser the single specification determines the ratio between the reflux (R) and distillate (D) flowrate. For a column with a partial condenser additionally a specification can be made that determines the cooling rate in the condenser.
