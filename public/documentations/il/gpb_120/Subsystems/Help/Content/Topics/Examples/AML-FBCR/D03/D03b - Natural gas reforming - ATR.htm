D03b - Natural gas reforming - ATR

In this example, an autothermal reactor is modelled considering both the combustion chamber and catalytic bed. The combustion chamber of the reformer (which includes the burner and the homogeneous reaction zone) is modelled using a conversion reactor from the gML Reaction library, while the catalystic fixed-bed section is modelled in detail with the AML:FBCR models.

The two flowsheet models included represent an ATR reactor in adiabatic operation or considering some heat losses to the environment.

The process (D03b_ATR_reactor_adiabatic_1D_Init) can be used to initialise the model from poor initial guesses - the existing Saved Variable Set will be updated if the option to auto-update is selected upon starting a simulation. The remainder processes use the saved variable set to initialise the simulation.
