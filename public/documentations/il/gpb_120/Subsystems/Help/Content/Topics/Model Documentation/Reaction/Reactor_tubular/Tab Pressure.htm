Pressure

The Pressure tab is used to specify data relative to the pressure drop in the reactor.

Name Default Unit Description
Pressure drop Selector
On
Selects calculation of pressure drop.
Off
Selects no pressure drop.

Pressure drop correlation

 Selector
Specified fanning factor
Selects specification of the fanning friction factor.
Smooth pipe
Selects the use of the smooth pipe equation.
Linear mass flow coefficient
Selects the specification of a linear mass flow coefficient.
Haaland
Selects the use of the Haaland equation.
Ergun equation for porous beds
Selects the use of the Ergun equation.
Custom
Selects the use of custom equation.

Include effect of gravity

 Selector
No
Neglects gravity effect due to inclination in the momentum equation.
Yes
The effect of the inclination is included in the momentum equation.
Pressure drop adjustable factor - Factor that can be adjusted to match an observed pressure drop at certain conditions.
Fanning friction factor - Sets the value of the Fanning friction factor.

Linear mass flow coefficient

 kg s-1 Pa-1 Sets the value of the proportionality constant when Linear mass flow coefficient mode is selected.
Angle of inclination rad Angle of inclination of the tubes (in case of tube-side reactions) or of the shell (in case of shell-side reactions).
