Reactions - coking

The Reactions - coking tab is used to specify data pertaining the coking reactions.

Name Default Unit Description
Coking in coils Selector
On
Coke deposition in radiant coil is taken into account.
Off
Coke deposition in radiant coil is not taken into account.
Coking in TLE Selector
Off
Coke deposition in TLE is not taken into account
On
Coke deposition in TLE is taken into account.
Kinetic model source Selector
Default
Uses a coking kinetic model from gPROMS.
External kinetics package
Allows to use coking kinetics from an external package. Please contact PSE to link external kinetic packages to furnace models in gML Olefins.

Coking model

 Selector
Default
Uses default coking kinetic model from gML Olefins.
Custom
Allows to use a user-defined custom kinetic model for calculating coking rates.
Dynamics Selector
On (Dynamic initial conditions)
Off
Coke density kg/m3

Density of the coke layer.
Coking agents - Species responsible for coking.
Rate constant at 1073 K (g m-2s-1) / (mol m-3) Rate constant at 1073 K for the coking reactions involving the selected coking agents.
Activation energy J/mol Activation energy for the coking reactions involving the selected coking agents.
Rate constant multiplier -

Rate constant multipliers for the coking reactions.

Reaction rate constant at 1073 K = Rate constant multiplier x Default rate constant at 1073 K
Coke layer thickness m Thickness of the coke layer.
Initial coke layer thickness m Initial thickness of the coke layer.
Custom model - Drop-down menu containing all the user-defined coking kinetic models.
