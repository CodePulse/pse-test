Templates Kinetics

This library contains templates the user can start from to write custom kinetic models. These models are then used by the reactor models to calculate the reaction rates.

It is important to choose the correct model when starting to customise the kinetic model. There are templates for both 1D pellet and 0D (or lumped) pellet model and each of those in a 1D bed or 2D bed. A particular template will apply to a particular reactor and pellet model:

  • 1D bed, 0D pellet: kinetics_in_0D_layer_in_1D_bed_gML
  • 1D bed, 1D pellet: kinetics_in_1D_layer_in_1D_bed_gML
  • 2D bed, 0D pellet: kinetics_in_0D_layer_in_2D_bed_gML
  • 2D bed, 1D pellet: kinetics_in_1D_layer_in_2D_bed_gML
