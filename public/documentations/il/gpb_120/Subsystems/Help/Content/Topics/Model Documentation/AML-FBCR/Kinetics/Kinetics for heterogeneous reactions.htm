Heterogeneous reactions

All AML:FBCR reactor models use user specified kinetic models for heterogeneous reactions. Kinetic models must be used to provide reaction stoichiometry and rate equations. The provided templates can be copied and customised in a local project - the following tables show the parameters and variables which are required in the kinetic models.

The following parameters need to be supplied in all kinetics models:

The user is free to define additional parameters and variables, as long as these are assigned and/or equations are provided to calculate them. Typical examples are variables for calculation of reaction rate constants, Arrhenius factors and reaction orders.

Choosing the correct kinetic template is important. Different reactor models will use different kinetic models. The correct model will depend on the level of detail of the fixed-bed model (1D or 2D) and on the option for the pellet model (lumped or 1D).
