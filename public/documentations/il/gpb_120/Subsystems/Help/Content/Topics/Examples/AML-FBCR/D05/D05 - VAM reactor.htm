D05 - VAM reactor

In this example, the vinyl acetate monomer production process is modelled using an axial flow tubular reactor with external cooling.

The catalyst section model used represents the catalyst bed with a 2D bed model and a 1D distributed pellet model. The coolant is modelled either with a simple fixed coolant model or with a boiling water cooling section. Two different flow configurations are considered: co-current and counter-current. Deactivation kinetics are also considered in both flowsheets with the boiling cooling water section.

The processes (D05_VAM_reactor_co_current_Initialisation, D05_VAM_reactor_cr_current_Initialisation and D05_VAM_reactor_fixed_coolant_Initialisation) can be used to initialise the models from poor initial guesses - the existing Saved Variable Sets will be updated if the option to auto-update is selected upon starting a simulation. The remainder processes use the saved variable sets to initialise the simulation.

This project also includes the models with the kinetics for the reactor model configuration considered: 1D distributed pellet model in 2D bed model. It also includes the physical property package configuration file for the main process stream.

The flowsheets with boiling water cooling sections use a gML adjust block in order to set the utility stream pressure at the outlet, at the same time ensuring a flow-driven initialisation.
