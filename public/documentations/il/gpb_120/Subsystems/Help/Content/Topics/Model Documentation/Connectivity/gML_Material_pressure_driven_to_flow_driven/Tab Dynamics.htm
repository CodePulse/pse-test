Dynamics

In the gML Material pressure-driven to flow-driven model the outlet material stream operates in a flow-driven mode, being identical to the inlet material stream. The inlet stream works as a sink, defining the upstream operating mode. This operating mode is specified by the user in the Dynamics tab. Note that although the main functionality of the model is to operate the upstream units in a pressure-driven mode, a flow-driven inlet may also be selected if required.

Name Default Unit Description

Mode

 Selector
Pressure-driven (flow-driven initialisation)
A pressure-driven inlet sink is created. The assigned inlet pressure is equal to the outlet pressure and affects the pressure or the flowrate of the upstream units. An initialisation procedure is used to aid flowsheet initialisation.
Pressure-driven label (flow-driven initialisation, apply results to pressure-driven case)
Selects a pressure-driven inlet sink with no assigned inlet pressure. An initialisation procedure is used to aid flowsheet initialisation.
Pressure-driven
Sets a pressure-driven inlet with an assigned inlet pressure which affects the pressure or the flowrates of the upstream units. This mode can be used when no initialisation in flow-driven mode is required, for example when the initial state is restored from a Saved Variable Set (SVS).
Flow-driven
Selects a flow-driven inlet which defines an upstream flow-driven section.
