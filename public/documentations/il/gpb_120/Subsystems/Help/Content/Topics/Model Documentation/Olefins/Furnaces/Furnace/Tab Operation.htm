Operation

The Operation tab is used to select and assign values to the main operation variables in the furnace.

Name Default Unit Description

Use input data file

 Selector
Off
All the operating variables are specified in the dialog box of the furnace.
On
Allows for some or all of the operating variables of the furnace to be specified via a data file.
CIT Selector
Specified here
The coil inlet temperature (CIT) is specified directly in the furnace dialog box.
From data file
The CIT is specified in a data file.
Pressure specification Selector
CIP
The coil inlet pressure (CIP) is specified in the model dialog box. The coil outlet pressure is then calculated using the pressure drop correlation and parameters ('Pressure').
COP
The coil outlet pressure (COP) is specified in the model dialog box. The coil inlet pressure is then calculated using the pressure drop correlation and parameters ('Pressure').
From data file
The pressure specification is obtained from a data file ('Input data file').
Severity specification Selector
COT
The coil outlet temperature (COT) is specified in the model specification dialog box and used as a measure of severity of cracking.
Conversion
The outlet conversion of a given component in the feed is specified in the model specification dialog box and used as a measure of severity of cracking.
From data file
The severity specification is obtained from a data file ('Input data file').
CIT °C Coil inlet temperature.
CIP barg Coil inlet pressure.
COP barg Coil outlet pressure.
COT °C Coil outlet temperature.
Component - Defines the component for conversion specification.
Conversion % Conversion at furnace outlet.
