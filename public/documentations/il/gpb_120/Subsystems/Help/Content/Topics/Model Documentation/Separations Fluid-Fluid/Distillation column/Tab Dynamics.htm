Dynamics

The Dynamics tab is used to turn dynamic mode on and off, as well as and specify the initial condition. This tab is only activated when the Pressure profile specification: Calculated from hydraulics in the Pressure tab is selected.

Name Default Unit Description
Mode Selector
Flow-driven
Simulation mode where the flowrates are specified at the sources or in splitter units.
This mode can be used to perform steady-state material and energy balances. It is not necessary to define flow-pressure relationships for flow-transportation equipment.
Pressure-driven (flow-driven initialisation)
Simulation mode where the user specifies pressures at the sources and the sinks and where flowrates are calculated based on these pressures and on the flow-pressure relationship in the unit operations. To facilitate initialisation of a steady-state process in pressure-driven mode, a flow-driven simulation is performed prior to the pressure driven one and the results from this simulation are used as an initial guess for the pressure driven simulation.
This mode requires the definition of flow-pressure relationships for all flow-transportation equipment (e.g. valves, pipes, compressors, pumps and turbines should be set to Performance Mode). It also requires the specification of initial guesses for the flowrates of all sources and for split fractions of all splitters. These will be used for the flow-driven initialisation.
Pressure-driven
Simulation mode where the user specifies pressures at the sources and the sinks and where flowrates are calculated based on these pressures and on the flow-pressure relationship in the unit operations. This mode can be used when no initialisation in flow-driven mode is required (e.g. when the flowsheet is not initialised at steady-state or when the initial state is restored from a snapshot created using a previous simulation).
This mode requires the definition of flow-pressure relationships for all flow-transportation equipment (e.g. valves, pipes, compressors, pumps and turbines should be set to Performance Mode).
Dynamics Selector
Off
Simulates the unit in steady-state mode. If this option is selected, the unit will always be in steady-state mode.
On
Simulates the unit in dynamic mode. If this option is selected, changes in time for mass, momentum and energy holdups will be automatically calculated.

Initial conditions specifications

 Selector
Steady-state
The model is at steady-state initially.
Dynamic

Determines the initial material holdup in the vessel from the composition and an intensive variable (pressure, temperature or enthalpy).
Initial sump holdup Selector

Determines the initial material holdup in the vessel from the composition and the volume.

Pressure varies linearly between the top and the bottom of the column. Top and bottom pressures in the column are specified.

Pressure varies linearly between the top and the bottom of the column. Constant pressure drop per stage and one extra pressure on the column are specified.

Sump volume specification Selector
Liquid level fraction
When selecting this option, the liquid level fraction, initially in the vessel, will be specified. The liquid level fraction is the ratio of the liquid height measured relative to the bottom of the vessel to the vessel height.
Liquid volume fraction
When selecting this option, the liquid volume fraction, initially in the vessel, will be specified. The liquid volume fraction is the ratio of the liquid volume to the vessel characteristic volume.
Liquid volume
When selecting this option, the liquid volume, initially in the vessel, will be specified.

Saturated liquid (Vapour fraction = 0)
When selecting this option, the vessel will initially consist of a saturated liquid (i.e. the vessel temperature is the bubble temperature of the mixture).
Subcooled liquid
When selecting this option, the vessel will initially consist of a subcooled liquid (i.e. outlet temperature is calculated using the provided degree of subcooling relative to the saturated liquid).
