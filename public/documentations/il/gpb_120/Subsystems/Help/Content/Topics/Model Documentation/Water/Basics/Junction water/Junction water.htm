Junction water

The Junction water model mixes a given number of inlet material streams and then splits the resulting flow into a number of outlet streams. To split the flow, the flowrate of each outlet process stream can be specified or a split fraction can be given, which determines how much of the combined flow from the inlets goes to each outlet.
