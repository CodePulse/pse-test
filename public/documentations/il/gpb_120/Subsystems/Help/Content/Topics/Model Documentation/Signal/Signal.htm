Signal

This library contains a number of models that have signals as inputs and/or outputs.

  • Elementary mathematical operations applied to an input signal
  • Convolutions of the input signal with linear systems
  • Source units that generate a specific signal
  • Sink units that either report the input signal, in various forms, or enforce a constraint on the input signal

The following table gives an overview of the models included.
