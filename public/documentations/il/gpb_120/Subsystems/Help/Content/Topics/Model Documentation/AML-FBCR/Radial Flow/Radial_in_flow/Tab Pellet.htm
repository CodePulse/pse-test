Pellet

The pellet model, geometry and properties are specified in the Pellet tab.

Name Default Unit Description
Catalyst pellet type Selector
Whole pellet
Solid pellet is made of 100% active material.
Coated pellet
Solid pellet is made of support material covered by active material; egg-shell pellet; inert core pellet.
Catalyst pellet shape Selector
Sphere
Cylinder
Hollow cylinder
Ring shaped pellet.
Multi-hollow cylinder
A hollow cylinder with more than one hole.
Non-ideal shape
Any shape different from the above options.
Catalyst model Selector
Distributed
Internal diffusion in the catalyst pellet is modelled with a distributed (1D) model.
Lumped
Internal diffusion in the catalyst pellet is neglected.
Catalyst 1D model Selector
Sphere
The catalyst pellet is represented by a sphere model. The model is distributed (1D) in the radial direction.
Infinite cylinder
The catalyst pellet is represented by an infinite cylinder model. The model is distributed (1D) in the radial direction.
Infinite hollow cylinder
The catalyst pellet is represented by an infinite hollow cylinder model. The model is distributed (1D) in the radial direction, between the inner and outer surfaces.
Sphere with inert core
The catalyst pellet is represented by a sphere model. Only the active layer is distributed (1D) in the radial direction. The inert core is represented by a lumped model.
Infinite cylinder with inert core
The catalyst pellet is represented by an infinite cylinder model. Only the active layer is distributed (1D) in the radial direction. The inert core is represented by a lumped infinite cylinder model.
Non-ideal shape properties Selector
Equivalent diameters
Properties are provided via the equivalent diameters; only two specifications are allowed.
Area and volume
Properties are provided via area and volume; only two specifications are allowed.
