Design (Heat transfer)

The Design (Heat transfer) tab is used to specify which channels in the multistream heat exchanger exchange heat with each other.

Name Default Unit Description
Assume heat is exchanged via a common wall between all channels -
Yes
Assume the temperature of the walls between all channels is the same.
No
Assume each wall between a pair of channels has a different temperature.

Area for heat exchange per channel

m2

 Area for heat exchange per channel used when it is assumed that all channels exchange heat with a wall that is at the same temperature at each axial position.
Number of connections - Specify the number of connections between any number of pairs of channels that exchange heat with each other.
Heat exchanger configuration - For each connection, specify the pairs of channels that exchange heat.
Area for heat exchange m2 For each connection, specify the area for heat exchange.
