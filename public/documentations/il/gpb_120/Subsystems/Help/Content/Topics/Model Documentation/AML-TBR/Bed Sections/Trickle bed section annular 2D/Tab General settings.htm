General settings

The General settings settings tab is used to specify parameters related to the reactor design and bed properties.

Name Default Unit Description
Flow direction Selector
Set the flow direction with respect to the inner-most tube. The options are:
Co-current
Co-current flow configuration.
Counter-current
Counter-current flow configuration.
Bed porosity calculation Selector
Specified
Bed porosity is directly specified by the user.
Mass of catalyst
Bed porosity is calculated from a mass of catalyst specification.
Haughey and Beveridge
Bed porosity is predicted from a predefined correlation.
Custom
Bed porosity is predicted from a user-defined correlation.
Bed geometry
Bed length m
Length of the bed section.
Bed radius m

Radius of the tube (excluding the wall = ID/2).
Bed porosity
Bed porosity m3/m3

Void fraction of the bed. Defined as the ratio of the free volume between pellets and the total bed volume.
Parameters - Adjustable parameters. Defined as a multiplier for predefined correlations.
Custom correlation Selector
Predefined options
Choose one of the predefined options.
Custom options
This list can be extended in the relevant custom model.
Number of parameters - Defines the number of adjustable parameters to be used in the custom model.
