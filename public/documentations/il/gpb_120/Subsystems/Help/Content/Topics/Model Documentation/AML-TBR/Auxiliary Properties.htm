Auxiliary Properties

This library project includes models that calculate:

  • gas and liquid phase thermodynamic and transport properties
  • vapour-liquid equilibrium

This library is provided in open form so that the user can create a copy with custom modifications as required, for example:

  • to provide Henry coefficients

    • vapour_liquid_interface_fugacity_in_1D_bed_gML

    • vapour_liquid_interface_fugacity_in_2D_bed_gML

  • to provide additional variables for custom correlations models from the 'Auxiliary Correlations' library

    • prop_bed_in_1D_bed_gML

    • prop_bed_in_2D_bed_gML
