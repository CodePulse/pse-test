Design (valve sizing)

The Design (valve sizing) tab is used to specify valve sizing data. The flow coefficient fraction can be calculated for a given design specification.

Name Default Unit Description
Phase Selector
Liquid
Liquid fluid specification.
Vapour
Vapour fluid specification.
Sizing variable Selector
Valve coefficient
The valve coefficient is determined for a given stem position.
Stem position
The stem position is specified for a given valve coefficient.
Flow relation Selector
Fisher equation
The flow-pressure drop relation is the Fisher equation.
Fisher universal gas sizing equation
The flow-pressure drop relation is the Fisher universal gas sizing equation.
ANSI/ISA S75.01 standard
The flow-pressure-drop relation is based on the ANSI/ISA S75.01 standard.
Include effect of choking Selector
No
Choking effect is not taken into account when valve is sized for liquid phase.
Yes
Choking effect is taken into account when valve is sized for liquid phase.
