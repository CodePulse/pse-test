Catalyst CSTR

This is a two-phase steady-state Continuously Stirred Tank Reactor(CSTR) model with catalyst pellets. The catalyst pellet model can be distributed (1D) or lumped. The pellets are assumed to be completely wet and the reaction happens in liquid phase.

The model is suitable for modelling experimental setups.

For more information on the empirical correlations, see 'Auxiliary Correlations'.
