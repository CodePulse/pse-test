Pressure

The Pressure tab is used to specify the pressure drops or the pressure-flow relations for each channel.

Name Default Unit Description
Pressure drop Selector
Off
No pressure drop from inlet to outlet.
On
Pressure drop from inlet to outlet calculated or specified.

Pressure drop specification

Selector
Specified a pressure drop correlation
Specify a pressure drop correlation that relates the outlet pressure to the flowrate.
Specified pressure drop
Specify a fixed pressure drop. This pressure drop is independent of the flowrate.
Specified outlet pressure
Specify an outlet pressure. The outlet pressure is independent of the flowrate.

Pressure drop correlation

Selector
Specified Fanning friction factor
Specify a Fanning friction factor for each channel.
Linear mass flow coefficient
Specify a linear mass flow coefficient for each channel.
Frictional adjustment factor - Factor that can be used to adjust predictions from pressure drop correlations to match observed results.
Fanning friction factor - Fanning friction factor for each channel.
Linear mass flow coefficient kg s-1Pa-1 Linear mass flow coefficient for each channel.
