Specifications

The variable data are specified in the Get signal from bus dialog.

Name Default Unit Description
Variable dimension Selector
Scalar
The obtained variable is a scalar.
Array
The obtained variable is an array.
Set
The obtained variable is a set.
Type of SET variable Selector
Specified
One scalar variable is obtained from the connected unit. This variable is chosen from the Signal types drop down selector.
All variables in connected unit
Obtains all the available scalar variables in the connected unit.
Specified element in array
One element from an array variable is obtained from the connected unit. The array variable is chosen from the Signal types specification drop down selector.
All elements in array
All the elements from an array variable are obtained from the connected unit. The array variable is chosen from the Signal types specification drop down selector.
Specified element in set
One element from a set variable is obtained from the connected unit. The ordered set variable is chosen from the Signal types specification drop down selector.
All elements in set
All the elements from a set variable are obtained from the connected unit. The ordered set variable is chosen from the Signal types specification drop down selector.
Signal types - Selects the variable from a drop-down menu with options given by the model connected to the bus port.
Index - Sets the index of the chosen element in an array variable.
Elements - Selects the chosen element in a set variable.
