Input data file - Data file for the operating variables.
Starting time in data file - The time entry in the data file, which corresponds to model inputs for simulation at time = 0. It has the same unit as the unit of time specified for the data file.
Data smoothing constant - Constant used for the smoothing of noisy plant data.
Column number - Number of the column containing each operating variable.
Component - Defines the component for conversion specification.
