Fluid properties

The AML:FBCR Properties library contains two types of fluid properties:

  1. properties of the fluid in 1D bed, 2D bed and 2D radial-flow bed models:

    • fluid_properties_1D

    • fluid_properties_2D

    • fluid_properties_2D_in_radial_flow_bed

  2. properties of the fluid in the 1D and lumped pellet model in various bed models:

    • fluid_Properties_in_0D_pellet_in_1D_bed

    • fluid_Properties_in_0D_pellet_in_2D_bed

    • fluid_properties_in_0D_pellet_in_radial_flow_bed

    • fluid_properties_in_1D_pellet_in_1D_bed

    • fluid_properties_in_1D_pellet_in_2D_bed

    • fluid_properties_in_1D_pellet_in_radial_flow_bed

The following bed fluid properties are calculated in the models using the corresponding standard gPROMS physical property functions:

  • Mass density

  • Mass specific enthalpy (mixture)

  • Mass specific heat capacity

  • Thermal conductivity

  • Dynamic viscosity (2D bed models only)

The pellet fluid properties include calculation of molar concentration. For gas systems the ideal gas behaviour is assumed as default or, alternatively, the vapour enthalpy function is being used.

  • Molar concentration (pellets only)

The partial enthalpy for all components of the mixture is calculated by assuming either ideal mixture or real mixture behaviour:

  • ideal mixture (default) – partial enthalpy is obtained by calling the enthalpy function at given pressure and temperature for each of the components (i.e., using pure component properties). This approach is sufficient in most of the gas systems.

  • real mixture – partial enthalpy is obtained by perturbation of the mixture enthalpy. This approach is suitable for cases where the state of the pure components at the reaction conditions is different from the state of the mixture (e.g., methanol productions).

The user can specify the partial enthalpy method via the model specifications (dialog box) of the corresponding bed model – see the 'Methods' specification tab in any of the reactor section models.
