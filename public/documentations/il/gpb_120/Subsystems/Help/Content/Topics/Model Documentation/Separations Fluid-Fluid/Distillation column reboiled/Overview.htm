Distillation column reboiled

This model describes a two-phase (vapour-liquid) reboiled column. The column is divided into multiple stages. For each stage a mass and energy balance are determined and by default it is assumed that vapour-liquid equilibrium is achieved at each stage.

In the reboiler, it is also assumed that vapour-liquid equilibrium is achieved. The reboiler can either be a kettle or thermosyphon type reboiler. The column model can also represent columns with liquid and vapour side-draws.

Non-ideal trays can be modelled using the Murphree efficiency.

This model is a distillation column with no condenser. The options of this model are similar to those of the distillation column but the user needs to provide only one specification in the Operation tab of the model which is related to the energy supplied to the reboiler.
