Model analysis

GSA can be used to support modelling activities in many areas, for example:

  • Model analysis
    • What range of temperature and composition can be expected to be observed for a range of input values?
    • How robust is the solution of the model over a wide range of conditions? A simple case study can be found here.
  • Parameter estimation
    • Which parameters are the most influential and which should be estimated? This is particularly useful to speed up a parameter estimation by only focussing on relevant parameters. A simple case study can be found here.
    • What is the effect of uncertainty on the KPIs? A simple case study can be found here.
  • Optimisation
    • What decision variables should be chosen? This is particularly relevant when there is a large number of decision variables available, or if an optimisation needs to be speeded up. A simple case study can be found here.
    • What is the effect of uncertainty in the decision variables on the objective, constraints and/or KPIs? A simple case study can be found here.
  • Model uncertainty
    • How does model uncertainty affect the KPIs? For example, it may not be clear what kinetic mechanism, agglomeration kernel or heat transfer correlation is most appropriate when it is not possible to distinguish between the options based on the data available. However, this uncertainty can have a significant effect on the KPIs. A simple case study can be found here.
