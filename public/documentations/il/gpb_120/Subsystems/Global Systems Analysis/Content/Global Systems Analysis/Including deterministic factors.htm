Including deterministic factors in sensitivity analysis

Although sensitivity methods require probabilistic factors, it is also possible to include deterministic ones. Sensitivity analysis is then performed as per the deterministic scenario.

A scenario represents a single element from the set of all possible combinations of deterministic values. In other words, sensitivity analysis calculates as many sensitivity measures as there are combined deterministic possibilities, and then combines the results for each of them.

If the number of deterministic combinations is Nd, then the total number of samples required to perform variance-base sensitivity analysis for all scenarios is Nd*N*(k+2), where k is the number of probabilistic factors. Given that N must be high, in order to achieve a convergence of the indices, few scenarios should be used.
