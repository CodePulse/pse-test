Factor sensitivity table

The factor sensitivity table displays different measures for each sensitivity analysis method, which are interpreted very differently:

  • Elementary effects
    • The mean is indicative of the importance of the factor in the variance of a response, and higher values denote higher impact. Although the largest values are representative, the impact order of smaller values, particularly when close to each other, is not reliable.
    • The standard deviation (in relation to the mean) is an indication the extent to which the response is influenced by other factors. Large standard deviations suggest that other factors also have significant impact. Note that the elementary effects measures do not reflect mathematical sensitivities.
  • Variance-based
    • The first order effect index (Si) represents the main effect contribution of each factor to the variance of a response. It is also known as 'importance measure'. An important factor is identified by a high value of Si.
      • If Si = 0, the factor has no direct influence on the response. However, it might still be of importance through its interactions with other factors. Therefore, factors with small Si values are not necessarily non-influential, and their total effect index (STi) should be inspected. As a significant difference between Si and STi indicates an important interaction involving that factor.
      • The sum of all Si values is always lower than or equal to one. When equal to one, there are no interactions between the factors, which implies that the model is additive.
      • Negative values should not be trusted. In some cases, negative values can be avoided by increasing the number of samples. However, the total effect index can still be trusted, as it is more robust.
    • The total effect index (STi) accounts for the total contribution of the factor to the response variance, that is, its individual contribution (first-order effect) in addition to all higher order effects due to its interactions with other factors.
      • STi should be higher or equal to Si. When equal, the factor has no interactions with the other factors and the influence of the factor is strictly linear. The closer the Si/STi ration is to one, the more linear the influence.
      • If STi = 0, the factor has no influence on the response and can be fixed at any value within its range of uncertainty.
      • The sum of STi is always higher than or equal to one. When equal to one, then there are no interactions between the factors.

To create a factor sensitivity table within a GSA Case, reproduce the steps below.

  1. Right-click either on the top level grain of the Case or on the 'System Analysis Group' folder.
  2. Select 'New result element...'.
  3. Select 'Factor Sensitivity Table' in the drop down menu, provide a name and click on OK.
  4. In the Attributes panel, on the right-hand side, click on the ellipsis button to select which probabilistic factors and responses to add to the table.
  5. When applicable (that is, for factors with gridded set or enumerated set distributions), change the scenario in the Scenarios table.
