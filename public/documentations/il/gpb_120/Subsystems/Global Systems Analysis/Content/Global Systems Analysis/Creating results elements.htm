Creating results elements and analysing results

A Case is created at the launch of a GSA activity and is populated with the contents, below, at the end of execution.

  • Results folder
    • Output files related to foreign objects, if applicable
    • The GSA report file (with extension .gsareport), which contains a brief overview of the activity
    • The GSA results file (with extension .nc), which contains all results in HDF5 format
    • The GSA resume file (with extension .nc), when execution is stopped. This is used to resume execution.
  • 'System Analysis Group' folder

This folder is populated with copies of all result elements present in the Project as children of the GSA entity, if any such result elements exist. Otherwise, it is displayed empty and can be populated by creating result elements within or copying and pasting existing ones from other Projects or Cases.
