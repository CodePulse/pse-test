System analysis

Once the system is set up and the model or flowsheet can be successfully simulated, GSA can be used to conduct various types of analysis and find solutions for a number of problems, for example:

  • Scenario or parametric analysis
    • How is system performance affected when process inputs are changed?
    • What is the difference between the various design options available?
    • What kind of conditions can be observed in the system?
  • Uncertainty analysis
    • How does uncertainty in the process inputs affect the critical quality attributes (CQAs) of the product?
    • How likely will the operation be profitable given market uncertainty in cost and pricing?
    • What kind of variation can be expected in the production rate, given uncertainties in the system?
  • Sensitivity analysis
    • Where is most of the variability in the product quality/profitability coming from?
    • What uncertainty factors should be considered, to reduce variability?
    • How sensitive are the KPIs to variations in operational decisions?
