Notes:

  1. For 64-bit Linux, PSE recommends and supports RHEL 6. A version 2.6.32 kernel or later on x86_64 is required.
  2. 32-bit gPROMS ModelBuilder is supported on
    1. 32- and 64-bit Windows 7 Professional and Ultimate
    2. 32- and 64-bit Windows 8.1 Professional/Enterprise.
  3. 64-bit gPROMS ModelBuilder is supported on
    1. 64-bit Windows 7 Professional and Ultimate
    2. 64-bit Windows 8 Professional and Enterprise.
  4. When 32-bit gPROMS Objects are deployed under 64-bit Windows, software interfaced to them (e.g. Fluent or MATLAB) must also be installed in 32-bit mode.
  5. gO:CFD is also available for use in parallel mode with Fluent v14.5.0. This requires a separate license.
  6. Multiflash requires the installation of Microsoft .NET v3.5. On Windows 7 and later, this is typically already installed whereas on Windows XP it may be missing.
  7. Multiflash Microsoft Excel add-in requires 32-bit Microsoft Excel® on both 32-bit and 64-bit installations.
  8. The Hybrid Multizonal gPROMS/CFD modelling software is also available for use in parallel mode with Fluent v14.5.0.
  9. On Linux, the physical properties engine is available without the Multiflash GUI.
