Syntax highlighting

gPROMS ModelBuilder automatically employs syntax-sensitive highlighting of the gPROMS language to support the creation and modification of each Entity.

A default set of colours is assigned for different types of gPROMS constructs. The user can overwrite these default settings, or indeed switch syntax highlighting off, by going to the gPROMS ModelBuilder Preferences dialog and highlighting the Entity editor category.
