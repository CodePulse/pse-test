PML flash separation Unit specifications

Feed (Source)

In the source we specify the feed properties (mass fractions of all components, temperature and pressure) as well as the physical properties. It is assumed that one can use ideal physical properties, and so the following specification is made in the physical properties cell of the Source Specification dialog: 

IPPFO::<METHANE, HYDROGEN, METHANOL, ETHANOL,N_ BUTANOL>

The dialog box associated with the Source model is shown below.

gPROMS will automatically propagate all relevant property parameters associated with the Source along the flow path.

Gas_valve (Valve_gas)

The gas valve is configured to be a linear valve, with allowance for reversible flow. The leakage fraction, time constant, flow coefficient and recovery factor are specified as shown in the figure below. Note that the stem position sub-group is set to "Controlled", since the position will be controlled by pressure controller on the Flash drum.

Steady-state initial conditions are chosen for this dynamic valve model.

Press_control (PID_controller)

A proportional controller is used to control the pressure in the flash drum. The controller is reverse acting and automatic. The specifications are shown in the figure below. No initial conditions are needed since this is a proportional controller (no integral or derivative term).

Liquid_valve (Valve_liquid)

The liquid valve is configured to be a linear valve, with irreversible flow. The leakage fraction, time constant and flow coefficient are specified as shown in the figure below. Note that the stem position sub-group is set to "Controlled", since the position will be controlled by liquid level cascade controller on the Flash drum.

Steady-state initial conditions are chosen for this dynamic valve model.

Fl_bottom (Flow_neasurement)

A flow measurement model is added at the base of the Flash drum to record the flow going through the liquid control valve. No specifications are required for this model.

Level_control (PID_controller)

A proportional integral controller is used to control the level in the flash drum. The controller is reverse acting and automatic. The specifications are shown in the figure below.

The initial conditions are that the integral error is set to zero.

Flow_control (PID_controller)

A proportional integral cascade controller is used to adjust the liquid valve stem position to control the level in the flash drum. The connectivity is such that the flow measurement comes from the Flow measurement model and the set-point comes form the Level Controller. The control action (output) is the valve stem position.

The controller is configured to be direct acting and cascade. The specifications are shown in the figure below.

The integral error is initially set to zero.
